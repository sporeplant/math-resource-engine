# 图片资源验证器

验证所有输出文件中的图片引用是否符合 `主控/图片引用协议.md` 和 `技能/图片资源技能/技能.md`。

---

## A 组：路径合规（硬性条件）

- [ ] 课件图片引用使用 `![图注](./images/文件名)`。
- [ ] 图片图注非空。
- [ ] 禁止出现跨目录图片路径。
- [ ] 禁止出现旧资源目录图片路径。
- [ ] 禁止出现 `C:/`、`D:/` 等盘符绝对路径。
- [ ] 图片路径中的文件必须真实存在。
- [ ] 最终输出中不得保留图片占位符。

---

## B 组：格式合规（硬性条件）

- [ ] 课件未使用 HTML 尺寸控制代码。
- [ ] 2 张及以上图片按自然顺序排列或用 Markdown 表格并列。
- [ ] 图片引用位于题目描述之后、分析思路或提示之前。

---

## C 组：内容合规（建议性）

- [ ] 图片配有图注。
- [ ] 图片来源已标注教材或习题库。
- [ ] 未单独设置 `height` 破坏原始宽高比。

---

## 验证结果

| 结果 | 条件 | 处理 |
|------|------|------|
| 通过 | A 组全部通过，B 组违规 0 项 | 进入下一环节 |
| 有条件通过 | A 组全部通过，B 组违规不超过 2 项 | 附修复建议 |
| 不通过 | A 组任一项不通过，或 B 组违规 3 项及以上 | 回退来源阶段 |

---

## 验证方式

### 方式一：脚本自动化检查（推荐）

硬性条件由 `tools/validate_output.py` 同步检查：

```text
python tools/validate_output.py 输出/{课时名}_课件.md
```

### 方式二：人工逐项检查

用于少量课时或脚本无法覆盖的情况，参考上述 A/B/C 组验证项逐项检查。
