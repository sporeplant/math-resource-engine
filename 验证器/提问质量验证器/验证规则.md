# 提问质量验证器

## 验证项

- [ ] 每个提问是否有明确材料锚点、学生动作、数学指向、预期证据和后续出口
- [ ] 是否不存在看图说话、事实复读、封闭确认、伪开放和无指向泛问
- [ ] 每个提问是否能对应学习结果或本课核心任务
- [ ] 每个提问是否能产生可观察的评价证据
- [ ] 每个提问是否对结论负责，并提供分级备用提示
- [ ] 提问顺序是否符合教材原文和教学活动顺序
- [ ] 平均分是否达到提问质量技能设定的通过线

---

## 验证流程

1. 按提问质量技能的五要素逐题检查材料、动作、数学指向、证据和出口
2. 扫描低质问题类型：看图说话、事实复读、封闭确认、伪开放、无指向泛问
3. 对照学习结果、评价任务和活动流程检查提问是否游离
4. 检查每个提问是否有分级备用提示，且提示能把偏差回答引回核心结论
5. 计算平均分、4分提问占比和低分提问数量

---

## 验证结果

- 通过：所有硬检查满足，平均分和高质量占比达到要求
- 有条件通过：仅存在少量可重写问题，附修改建议
- 不通过：存在看图说话、事实复读、封闭确认、提问不对结论负责或平均分不达标
