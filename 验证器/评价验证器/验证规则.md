# 评价验证器

## 验证项

- [ ] 是否能检测学习结果（评价任务能直接反映目标是否达成）
- [ ] 是否存在真实评价证据（有可观察、可记录的评价产出）
- [ ] 是否可分层评价（不同认知层有差异化的评价标准）
- [ ] 是否体现数学思维（评价任务要求展示思维过程，而非仅答案）
- [ ] 是否避免纯刷题（评价任务包含理解性、应用性内容，而非机械练习）
- [ ] 是否支持形成性评价（评价结果能反馈到教学调整中）

---

## 验证流程

1. 对每条评价任务，判断其是否能检测对应目标
2. 检查评价产出是否具体可记录
3. 确认不同分层使用不同评价尺度
4. 分析评价任务是否要求学生展示思维过程
5. 识别是否包含过多机械性题目
6. 判断评价结果是否能用于调整后续教学

---

## 验证结果

- 通过：所有检查项均满足
- 有条件通过：存在个别可优化项，附修改建议
- 不通过：存在严重问题，需退回重新设计
