# 参考答案生成技能 检查清单

## 基础检查项（强制通过）

- [ ] 文件元数据是否完整（content_type、lesson_id、lesson_name、source_files、created_at）
- [ ] 标题是否正确（{课时名} 课堂提问参考答案）
- [ ] 随机池选取记录是否存在且表格格式正确
- [ ] 板块A【ASK课堂提问参考答案】是否存在
- [ ] 板块B【例题与练习解答】是否存在
- [ ] 评分量规是否存在且表格格式正确
- [ ] 所有题目是否标注了 source_id、source_type、question_id
- [ ] 所有题目来源是否为知识库中的真实题源（禁止LLM自行生成题目）
- [ ] 数学公式是否使用 LaTeX 格式（如 $x^2$、$\because$、$\therefore$）

---

## 板块A质量检查项

### 结构完整性
- [ ] 是否按基础层、中间层、拓展层分组
- [ ] 每个问题是否包含：问题编号、所属活动、学生姓名、问题原文、参考答案、评分要点
- [ ] 问题编号格式是否正确（ASK-B-XX、ASK-M-XX、ASK-E-XX）
- [ ] 学生姓名是否与分层提问技能的选取结果一致

### 内容质量
- [ ] 参考答案是否拆分为清晰要点（3-5个要点）
- [ ] 基础层答案是否侧重事实性、记忆性内容
- [ ] 中间层答案是否侧重推理性、应用性内容
- [ ] 拓展层答案是否侧重开放性、创新性内容
- [ ] 评分要点是否明确可观察、可量化
- [ ] 每个评分项是否配有具体分值
- [ ] 评分要点总分是否控制在 2-5 分
- [ ] 是否区分了"准确性"、"完整性"、"表达力"维度

### 数学准确性
- [ ] 所有数学概念、定义表述是否准确
- [ ] 所有公式、定理引用是否正确
- [ ] 所有推理步骤是否逻辑完整、无断裂
- [ ] 所有计算结果是否正确

---

## 板块B质量检查项

### 覆盖范围
- [ ] 是否覆盖了教学设计 lesson_flow 中引用的所有教材例题
- [ ] 是否覆盖了教学设计 lesson_flow 中引用的所有课堂练习/检测题目（教材练习栏目全部 + 习题A组1题 + 习题B组1题）
- [ ] 是否覆盖了教学设计 lesson_flow 中引用的所有习题库题目
- [ ] 台阶递进式核心题是否包含完整解题步骤（不仅是最终答案）

### 题源对应
- [ ] 每个题目是否标注了 source_id
- [ ] 每个题目是否标注了 source_type（TEXTBOOK / WORKBOOK）
- [ ] 每个题目是否标注了 question_id
- [ ] 题目标注是否与教学设计中的标注一致

### 解答质量
- [ ] 证明题是否包含完整推理链，每步有依据
- [ ] 计算题是否展示关键计算步骤和最终结果
- [ ] 概念题是否包含准确定义和辨析说明
- [ ] 应用题是否包含建模过程、求解步骤、结论解释
- [ ] 解答语言是否使用规范数学术语

### 数学准确性
- [ ] 所有证明步骤逻辑严谨、无错误
- [ ] 所有计算过程正确、无计算错误
- [ ] 所有几何推理符合规范
- [ ] 所有数学符号使用正确

---

## 页面长度控制

- [ ] 板块A是否控制在 3-4 页 A4 纸内
- [ ] 板块B是否控制在 2-3 页 A4 纸内
- [ ] 总长度是否不超过 6 页 A4 纸

---

## 协作一致性检查

- [ ] 学生姓名是否与分层提问技能输出的选取结果完全一致
- [ ] 问题内容是否与教学设计中的问题完全一致
- [ ] 题目解答是否与习题分析技能的8维度分析一致
- [ ] 评分量规是否与评价设计技能的评分标准一致

---

## 最终确认

- [ ] 通读全文，确认无数学概念错误
- [ ] 通读全文，确认无公式、定理错误
- [ ] 通读全文，确认无推理逻辑错误
- [ ] 通读全文，确认无计算错误
- [ ] 确认所有题目来自题源，无LLM自行生成题目
