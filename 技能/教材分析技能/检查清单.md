# 教材分析技能 检查清单

## Step01 教材知识结构分析

- [ ] 是否梳理了知识点的纵向逻辑链条
- [ ] 是否分析了知识点的横向联系（与已学、与后续）
- [ ] 是否明确了知识点在本单元/本册中的位置和功能

## Step02 数学本质分析

- [ ] 是否给出了核心概念的定义和数学内涵
- [ ] 是否分析了知识点的数学结构
- [ ] 是否说明了概念的形成过程和逻辑基础
- [ ] 是否从知识库/数学本质/ 中注入了对应知识点的本质分析

## Step03 核心素养分析

- [ ] 是否分析了主要培养的核心素养维度
- [ ] 是否明确了素养的具体表现要求
- [ ] 素养指向是否覆盖了数学抽象、逻辑推理、数学建模、直观想象、数学运算、数据分析中的相关维度

## Step04 学业要求映射

- [ ] 是否映射到了课程标准中的学业要求
- [ ] 是否明确了学生应达到的学习水平
- [ ] 是否区分了不同层次的要求

## Step05 认知起点分析

- [ ] 是否分析了学生已有的知识基础
- [ ] 是否分析了学生已有的生活经验或直觉
- [ ] 是否从知识库/学情理论/认知分层.md 获取了当前学生的认知水平
- [ ] 是否识别了潜在认知障碍（从数学认知障碍.md 获取）

## Step06 教学重难点分析

- [ ] 是否明确了教学重点（聚焦于核心概念的建立）
- [ ] 是否明确了教学难点（来自抽象程度跨越或推理链过长）
- [ ] 是否给出了难点的脚手架和分解策略

## Step07 易错点分析

- [ ] 是否识别了学生常见的错误理解或操作
- [ ] 是否从知识库/常见错误/ 注入了对应知识点的常见错误
- [ ] 是否分析了错误的原因

## Step08 数学思想方法分析

- [ ] 是否识别了知识点蕴含的主要数学思想方法
- [ ] 是否基于课标、教材原文和数学本质进行分析
- [ ] 是否说明了思想方法在课堂中的渗透方式

## Step09 后续学习衔接分析

- [ ] 是否分析了本知识点对后续学习的影响
- [ ] 是否明确了为后续哪些内容做铺垫
- [ ] 是否指出了需要为后续学习埋下的伏笔
