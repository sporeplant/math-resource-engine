# 习题分析技能 检查清单

## Step01 题目考点分析

- [ ] 是否明确了题目考查的核心知识点
- [ ] 是否分析了知识点在课标中的对应要求
- [ ] 是否判断了题目属于单一知识点还是综合知识点考查

## Step02 认知层级分析

- [ ] 是否根据认知分层判断了题目认知水平
- [ ] 是否标注了每道题所处的认知层级（基础层/中间层/拓展层）
- [ ] 基础层是否对应辨认、复述、模仿
- [ ] 中间层是否对应解释、归纳、应用
- [ ] 拓展层是否对应分析、评价、创造

## Step03 易错点分析

- [ ] 是否分析了学生容易出错的环节和原因
- [ ] 是否识别了常见的错误类型（概念混淆、计算错误、推理断裂）
- [ ] 是否分析了错误背后的认知障碍
- [ ] 是否从知识库/常见错误/ 注入了对应知识点的常见错误

## Step04 数学思维分析

- [ ] 是否分析了题目涉及的数学思想方法
- [ ] 是否分析了解题过程中的思维策略（数形结合、分类讨论、转化等）
- [ ] 是否基于题目结构、教材原文和数学本质进行分析
- [ ] 是否标注了题目培养的核心素养维度

## Step05 解法比较规则

- [ ] 存在多种解法时是否列出并比较
- [ ] 比较维度是否包含简洁性、通用性、思维要求
- [ ] 是否给出了每种解法的适用学生层次建议
- [ ] 是否指出了最优解法及其选择理由

## Step06 变式设计规则

- [ ] 是否基于原题设计了变式
- [ ] 变式方向是否覆盖条件变化、结论变化、情境变化、逆向问题
- [ ] 是否说明变式的认知层级递进关系
- [ ] 变式是否服务于加深对核心概念的理解

## Step07 分层使用建议

- [ ] 是否标注了题目适合哪个认知层的学生
- [ ] 是否给出了课堂使用时机（新授课、练习课、复习课）
- [ ] 是否标注了题目是必做、选做还是挑战题
- [ ] 是否给出了分层布置的建议

## Step08 与教学目标映射

- [ ] 是否标注了题目对应哪条学习目标
- [ ] 是否标注了题目对应哪个评价任务
- [ ] 是否标注了题目在目标-评价-活动链条中的位置
- [ ] 是否确保每道题都有明确的教学意图
