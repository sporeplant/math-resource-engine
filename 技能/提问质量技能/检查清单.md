# 提问质量技能检查清单

## A. 五要素硬检查

- [ ] 每个提问都有明确材料锚点：教材文字、数据、表格、图形、题干、已完成步骤或学生产出。
- [ ] 材料锚点与问题位于同一课件页；隐藏上一页后仍能明确问题对象和作答依据。
- [ ] “这四类、这些、各数字、该图、前一个环节”等指代在同页有明确先行对象。
- [ ] 每个提问都有明确学生动作，且动作服务当前目的：为统计比例而分类计数、为确定分组而比较范围、为表图转化而作图、为判断方案而提取证据、计算、填表、改写、说明依据、判断、举反例等。
- [ ] 每个提问都有数学指向：概念、关系、条件、步骤、图表对应、证据或结论。
- [ ] 每个提问都有预期证据：数值、式子、图形、判断、依据、错误暴露点等。
- [ ] 每个提问都有后续出口：能进入下一步活动、方法、结论、评价或迁移。

## B. 低质问题拦截

- [ ] 无“哪里卡住了”“为什么不容易”“有什么困难”等主观困难追问。
- [ ] 无“你发现了什么”“你有什么想法”等无指向泛问。
- [ ] 无“是不是”“对不对”“可以吗”等封闭确认。
- [ ] 无可直接抄教材原句的事实复读问题。
- [ ] 无只问原因但不给材料范围和回答标准的问题。
- [ ] 无只问结论但不要求证据的问题。
- [ ] 无提问无法引向核心结论或后续活动的情况；每个提问均有具体指向性提示。

## C. 教学评一致性检查

- [ ] 每个提问能对应学习结果或本课核心任务。
- [ ] 每个提问能产生可观察的评价证据。
- [ ] 每个提问能推动活动流程，不是游离追问。
- [ ] 每个提问均对结论负责，学生出现偏差回答时有可执行的分级备用提示。
- [ ] 提问没有重新生成或改变已审核教学设计的目标、评价和活动。

## D. 提问链检查

- [ ] 全课问题链能写成一条清晰主线。
- [ ] 同一活动环节通常不超过1个主问题，确需递进时不超过2问。
- [ ] 同一环节不存在同层堆积。
- [ ] 问题顺序符合教材原文和教学活动顺序。
- [ ] 引导性问题出现在答案、结论或规范过程之前。
- [ ] 小结或出口检测问题明确承担回看功能。

## E. 分层匹配检查

- [ ] 基础层问题有低起点且有目的的材料动作，不只要求机械复述，也不使用无目的的圈、画、标、找。
- [ ] 中间层问题要求说明依据、完成转化或处理易错点。
- [ ] 拓展层问题要求数据、条件、图表或推理证据。
- [ ] 三层问题均有覆盖，且问题难度与分层规则一致。

## F. 评分通过检查

- [ ] 平均分 ≥ 3.2。
- [ ] 4分问题占比 ≥ 50%。
- [ ] 1分问题为0个。
- [ ] 2分问题已重写或合并，无原样保留。
- [ ] 五要素完整率 ≥ 90%。
- [ ] 删除或重写清单已给出具体替代方向。
- [ ] 每个课堂提问是否先写提问，再写分级备用提示，最后写教师参考预期
- [ ] 备用提示是否注明仅在学生停滞或答偏时使用
