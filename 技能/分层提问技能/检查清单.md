# 检查清单

## 基础检查项

- [ ] 是否读取了学生成绩数据（学生近期数据/学生成绩数据.md）
- [ ] 学生姓名是否从数据源提取（禁止虚构学生姓名）
- [ ] 分层是否正确（基础A 0-24分 / 基础B 25-49分 / 中间层 50-69分 / 拓展层 70-100分）
- [ ] 基础A学生是否只分配了记忆/回忆类问题（无权分配理解/应用类问题）
- [ ] 基础B学生是否仅分配了基础层问题（未分配中间层以上问题）
- [ ] 是否执行了跨子层提问（记忆类问题可配基础A/B，理解类问题仅配基础B）
- [ ] 教学设计中是否写死了具体学生姓名（应只标注层级与认知难度，不写具体姓名）
- [ ] 同一课时内是否有学生被重复提问
- [ ] 是否生成了学生选取结果表格
- [ ] 是否生成了问题-学生映射表

## 质量检查项

- [ ] 基础层问题类型是否匹配（基础A→记忆/回忆类，基础B→理解/简单应用类）
- [ ] 基础A低分段学生(0-24分)是否只分配了记忆回忆类问题（背公式、读图、复述、选择判断）
- [ ] 基础B学生是否分配了与其分数匹配的理解类问题（25-34分偏简单计算，35-49分可解释推导）
- [ ] 中间层问题类型是否匹配（思路解释/方法迁移/逻辑分析/书写证明）
- [ ] 拓展层问题类型是否匹配（本质分析/特点归纳/变式创新/思想提炼）
- [ ] 各环节是否至少覆盖一个层次（探究引入/例题讲解/课堂练习/课堂小结）
- [ ] 问题链是否形成台阶递进：起点低→逐步深→收束高
- [ ] 第①问（起点台阶）是否为基础层可答
- [ ] 第②问（推进台阶）是否为中间层可答
- [ ] 第③问（收束台阶）是否为拓展层可答
- [ ] 台阶之间的跨度是否适中，中间层能独立完成跳跃
- [ ] 提问总人次是否控制在18-22人次（约全班60%）
- [ ] 各层提问占比是否合理（基础~35%/提升~35%/拓展~30%）
- [ ] 是否执行了Fisher-Yates洗牌（三个池独立执行）
- [ ] 跨课时历史记录是否写入（学生近期数据/提问历史记录.json）
- [ ] 连续3课时未被提问的学生是否给予了更高权重（加权因子1.5）
- [ ] Fisher-Yates是否对三个池独立执行，每次执行独立
- [ ] 学生选取结果表格是否格式正确
- [ ] 问题-学生映射表是否格式正确，包含所有问题编号
