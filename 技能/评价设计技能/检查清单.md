# 检查清单

## 基础检查项

- [ ] 每条学习目标是否都有对应的评价任务
- [ ] 评价任务的行为动词是否与目标匹配
- [ ] 三层评价标准是否差异化
- [ ] 成功标准是否具体可观察
- [ ] 是否包含过程性评价
- [ ] 评价任务是否在课时内可完成
- [ ] 评价结果是否能指导教学调整

## 质量检查项

- [ ] 评价任务所用题目是否来源明确（标注教材题/习题库题号/改编来源）
- [ ] 基础层题目是否优先选用教材练习栏目题目
- [ ] 中间层题目是否优先选用教材习题A组题目
- [ ] 拓展层题目是否优先选用教材习题B组题目
- [ ] 每层是否至少选用了 1 道题目（教材不足时可从练习册补充）
- [ ] 是否避免了模糊的成功标准（如"基本正确"）
- [ ] 是否避免了三层使用同一评价标准
- [ ] 评价任务是否与学习目标脱节
