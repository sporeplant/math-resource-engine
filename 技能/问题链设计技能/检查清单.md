# 问题链设计技能 检查清单

## Step01 核心问题设计

- [ ] 是否设计了一个核心问题统领全课
- [ ] 核心问题是否与知识主题高度相关
- [ ] 核心问题是否具有探究性和开放性
- [ ] 核心问题是否能激发学生思考

## Step02 子问题设计

- [ ] 是否设计了逐步推进的子问题
- [ ] 子问题是否形成了认知台阶
- [ ] 子问题是否服务于学习目标
- [ ] 子问题之间是否有逻辑递进关系

## Step03 认知冲突设计

- [ ] 是否识别了学生的已有认知
- [ ] 是否识别了学生可能的误解
- [ ] 是否设计了冲突问题
- [ ] 冲突问题是否能引发认知失衡

## Step04 问题链结构

- [ ] 问题链结构是否为「核心问题 → 子问题1 → 子问题2 → 子问题3 → 结论形成」
- [ ] 问题链是否能引导学生逐步得出最终结论
- [ ] 是否避免了活动碎片化

## 输出格式检查

- [ ] 是否使用了 YAML 格式输出
- [ ] 是否包含了核心问题、子问题（1-3、认知冲突、最终结论
- [ ] 认知冲突是否包含已有认知、可能误解、冲突问题、目标结论
