# 1.1 正数和负数（第二课时）课堂提问调度稿

## 板块A：教材原文对应题

### A1. 一起探究（对应 ACT-02）

**ASK_B_01** — 页3 📖 回顾引入
- 提问：上节课我们用带"+"和"−"的数表示了什么？谁能举一个例子？
- 备用提示1：想一想那天地面温度是零上几度，地窖温度是零下几度
- 备用提示2：我们在数前面加了什么符号来表示？
- 层级：基础层
- 教师参考预期：学生说出"表示相反意义的量"，举例+30°C/-20°C等。为定义正负数做铺垫。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导

**ASK_B_02** — 页4 📖 正负数定义
- 提问：默读教材第一段后——什么叫正数？什么叫负数？
- 备用提示1：教材说的是"在什么数前面加什么符号"
- 备用提示2：正数前面是什么号？负数前面是什么号？有没有例外？
- 层级：基础层
- 教师参考预期：学生复述"在已学过的数（0除外）前面加'+'是正数，加'−'是负数"。教师板书并强调"0除外""+可省略"。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导

**ASK_M_01** — 页5 📖 符号约定辨析
- 提问：+3和3是一样的数吗？-5能省略"−"写成5吗？为什么？
- 备用提示1：正数的"+"号有什么特殊规定？
- 备用提示2：如果-5省略了"−"变成5，它表示的意义变了吗？
- 层级：中间层
- 教师参考预期：区分"正号可省略（+3=3），负号不可省略（-5≠5）"，体会符号约定的不可替换性。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导

**ASK_B_03** — 页7 🤔 一起探究(1)(2)
- 提问：警戒水位5m，超过记正、低于记负——"+1m"表示什么？实际水深是多少？
- 备用提示1：+1m是比警戒水位高还是低？高了多少？
- 备用提示2：警戒水位5m，高了1m，实际水深是多少？
- 层级：基础层
- 教师参考预期：回答"+1m表示比警戒水位高1m，实际水深6m"。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导
- question_id: 1.1.2-一起探究-1

**ASK_M_02** ⭐ — 页7 🤔 一起探究(1)(2)
- 提问："0m"表示____？此时实际水深是多少米？
- 备用提示1：0m是比警戒水位高了还是低了？
- 备用提示2：既不高也不低说明什么？
- 备用提示3：警戒水位是多少米？0m就是正好在这个位置，实际水深呢？
- 层级：中间层
- 教师参考预期：学生说出"0m表示既不高也不低，实际水深就是5m（警戒水位本身）"——为"0既不是正数也不是负数"建立直观基础。这是本课最核心的提问，停顿2分钟让学生充分思考后再给提示。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导
- question_id: 1.1.2-一起探究-2

**ASK_M_03** — 页8 💡 0的归属
- 提问：刚才+1m比警戒水位高，记为正；-2m比警戒水位低，记为负。那0m不高不低——它应该属于正数还是负数？
- 备用提示1：正数表示超过，负数表示低于，0表示什么？
- 备用提示2：0站在正数和负数中间，既不属于左边也不属于右边
- 层级：中间层
- 教师参考预期：学生推出"0既不是正数也不是负数，是正数和负数的分界"。教师板书结论。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导

**ASK_B_04** — 页9 🤔 一起探究(3)
- 提问："-2m"表示什么意义？实际水深是多少？
- 备用提示1：-2m比警戒水位高还是低？低了多少？
- 层级：基础层
- 教师参考预期：回答"-2m表示比警戒水位低2m，实际水深3m"。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导
- question_id: 1.1.2-一起探究-3

### A2. 分类建构与大家谈谈（对应 ACT-03、ACT-04）

**ASK_B_05** — 页10 📖 分类①
- 提问：教材说正整数、0、负整数统称什么？正分数、负分数统称什么？它们合起来又叫什么？
- 备用提示1：看教材分类段落，依次读出三类整数的统称
- 层级：基础层
- 教师参考预期：学生回答"整数=正整数+0+负整数；分数=正分数+负分数；有理数=整数+分数"。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导

**ASK_M_04** — 页11 📖 知识链接
- 提问：-6.5是不是分数？说说你的理由。
- 备用提示1：判断是不是分数的关键是什么？——看它能不能写成a/b（b≠0）
- 备用提示2：6.5能化成分数吗？前面再加个负号呢？
- 层级：中间层
- 教师参考预期：学生说出"-6.5=-65/10，能化成分数形式，所以是分数（负分数）"，打破"只有带分数线才是分数"的误解。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导

**ASK_M_05** — 页12 🤔 大家谈谈
- 提问：刚才我们按"整数/分数"给有理数分了类。想一想，还能按什么别的标准来分？
- 备用提示1：看看这些数前面带的符号有什么不同？
- 备用提示2：按符号来分——正号、负号，还有没有符号的数？
- 层级：中间层
- 教师参考预期：学生提出按"正/0/负"分类，教师引导完成第二种分类结构图。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导
- question_id: 1.1.2-大家谈谈-1

**ASK_E_01** — 页13 📖 两种分类对比
- 提问：同一些有理数，为什么有两种不同的分类结果？是分错了吗？
- 备用提示1：两种分法用的标准一样吗？
- 备用提示2：把全班同学按性别分和按身高分，结果一样吗？为什么？
- 层级：拓展层
- 教师参考预期：学生归纳"分类标准不同，结果就不同。两种分法都正确，但每次分类只能用一个标准"。
- 答案来源：教材原文_1.1.2_正数和负数 / AI参考推导
