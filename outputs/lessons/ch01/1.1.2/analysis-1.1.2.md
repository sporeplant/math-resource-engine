# 1.1 正数和负数 第2课时 教材分析与教学决策

---

## 一、课标要求

| 维度 | 内容 |
|:---|:---|
| 课标条目 | 4.1.1 ①理解负数的意义；理解有理数的意义，能用数轴上的点表示有理数，能比较有理数的大小 |
| 学业要求 | 理解负数的意义，会用正数和负数表示具体情境中具有相反意义的量；理解有理数的意义 |
| 核心素养 | 抽象能力、运算能力、推理能力、应用意识 |

> **说明**：本课时为有理数概念建构的基础课时，核心任务是"理解有理数的意义"。课标要求"理解"层次——学生应能描述有理数的由来、内涵和特征，阐述有理数与整数、分数之间的区别和联系，并能举例说明。课标条目中"能用数轴上的点表示有理数、能比较有理数的大小"部分将在后续课时（1.2 数轴、1.3 绝对值与相反数）落实，本课时为其提供概念基础。

## 二、教材分析

### 2.1 内容结构

| 环节 | 内容 | 作用 |
|:---|:---|:---|
| 定义形成 | 正数和负数的形式化定义："在已学过的数（0除外）的前面加上'-'得到的数叫作负数，在前面加上'+'得到的数叫作正数"；正数中的"+"可以省略不写 | 从第一课时"用带'+''-'的数表示相反意义的量"过渡到概念的形式化定义 |
| 一起探究 | 警戒水位情境：已知警戒水位为 5 m，规定超过部分记为正、低于部分记为负，判断 +1 m、0 m、-2 m 的意义和实际水深 | 通过真实情境引出"0"的双重身份——既表示警戒线参考点，又是正数和负数的分界 |
| 概念明确 | 0 既不是正数，也不是负数；0 是正数和负数的分界 | 明确 0 的归属，建立正-0-负三分结构 |
| 分类体系建立 | 列出正整数、0、负整数、正分数、负分数五类数；定义整数 = 正整数 + 0 + 负整数；分数 = 正分数 + 负分数；有理数 = 整数 + 分数 | 建立有理数的第一套分类体系（按"是否为分数"标准） |
| 知识链接 | 任何分数都可以化为有限小数或无限循环小数；任何有限小数或无限循环小数都能表示成分数（例：0.16 = 16/100 = 4/25） | 打通分数与小数的关系，为后续有理数运算中小数处理做铺垫 |
| 大家谈谈 | 将有理数按整数/分数进一步分类；探索第二种分类方法（按正/0/负） | 分类思想深化：同一集合可按不同标准得到不同分类结果 |
| 做一做 | 4 道分类与判断练习（正数/负数圈填、判断正负、0 的特殊位置） | 当堂巩固正负数判断与分类技能 |
| 习题 A 组 | 3 题：用正负数表示实际情境中的量（温度、海拔）；任意写正数和负数；探究无限循环小数化分数 | 基础应用与拓展探究 |
| 习题 B 组 | 2 题：交叉条件分类（"整数但不是正数""分数但不是负数"）；正分数/负分数圈填 | 深化分类理解，区分"整分"与"正负"两个维度 |
| 习题 C 组 | 1 题（2 问）：零件标注 Φ30⁺⁰·⁰²₋₀·₀₂ 的实际意义与反推标准尺寸 | 综合应用：正负偏差表示的实际意义与逆向推理 |

### 2.2 知识生长链

```
小学已学的数（自然数、小数、分数及其四则运算）
    ↓ 数的扩充需求：表示相反意义的量
相反意义的量（1.1.1）——用带"+"和"-"的数统一表示
    ↓ 形式化抽象
正数和负数的形式化定义（1.1.2 开头）
    ↓ 边界明确
0 的归属：既不是正数也不是负数，是正数和负数的分界
    ↓ 体系建构
有理数的第一种分类（按整数/分数）：
  整数 {正整数, 0, 负整数} + 分数 {正分数, 负分数} → 有理数
    ↓ 维度拓展
有理数的第二种分类（按正/0/负）：
  正有理数 {正整数, 正分数} + 0 + 负有理数 {负整数, 负分数}
    ↓ 知识打通
分数 ↔ 有限小数/无限循环小数 的互化
    ↓ 后续衔接
数轴上的点表示有理数 → 相反数、绝对值 → 有理数大小比较 → 有理数四则运算
```

### 2.3 与已学知识的联系

| 已学知识 | 联系点 |
|:---|:---|
| 小学自然数、小数、分数 | 有理数是在这些数基础上的扩充——在已学过的数（0 除外）前面加"+"或"-"构成正数和负数，连同 0 一起组成有理数体系 |
| 1.1.1 相反意义的量 | 第一课时建立了"用带'+'和'-'的数表示相反意义的量"的操作性方法；本课时将其抽象为"正数""负数"的形式化概念定义 |
| 1.1.1 做一做和练习 | 学生已接触过 +1.8 km、-3 km、+30%、-20%、+30 cm、-50 cm 等正负数实例，本课时需从这些具体实例中提炼出正数、负数的定义 |
| 小学分数与小数互化 | 教材在引入有理数定义后立即建立"分数 ↔ 有限小数/无限循环小数"的双向通道，学生在小学阶段已具备分数化小数的计算能力（分子÷分母），本课时需要补上"有限小数化分数"和"无限循环小数化分数"的方法 |

## 三、学情分析

### 3.1 学生已有基础

| 知识基础 | 说明 |
|:---|:---|
| 正有理数的四则运算 | 小学已掌握自然数、小数、分数的运算，但仅限于正有理数范围 |
| 相反意义的量 | 上节课（1.1.1）已学习如何用带"+"和"-"的数表示相反意义的量，并在做一做和练习中多次操作 |
| 数的分类经验 | 小学阶段经历过自然数、整数、小数、分数的分类学习，具备基本的分类意识 |
| 分数化小数的技能 | 小学阶段已掌握分子除以分母化小数的方法 |

**班级学情画像**（依据 `students/scores.md`）：
- 基础层学生（0-49 分）占全班约 65%，是本课时的主要关注对象；其中标记 `skip` 的学生（12 人）不参与随机提问但仍需设计可参与的基础层任务
- 中间层学生（50-69 分）占约 22%，是课堂推理活动的主力
- 拓展层学生（70-100 分）占约 13%，可在分类探究和习题 C 组中承担挑战性任务

### 3.2 可能存在的困难

| 困难点 | 表现 | 对策 |
|:---|:---|:---|
| 0 的归属理解 | 学生可能认为 0 是正数（因为 0 前面可以加"+"写成 +0），或认为 0 是正整数（混淆"自然数包含 0"与"正整数的范围"） | 通过一起探究中警戒水位情境使学生直观体验"0 m 既不高也不低"，再过渡到"0 既不是正数也不是负数"的数学表述 |
| 分类标准混淆 | 在同一分类体系中混用"整数/分数"和"正数/负数"两种标准，例如在按"整分"分类时将 0 归入正数栏，或在按"正负"分类时将正分数归入整数栏 | 大家谈谈环节明确展示两种分类是"同一集合按不同标准的两种划分"，强调每次分类只能用一个标准；做一做和 B 组习题分别从不同维度练习 |
| 有限小数化分数 | 学生可能忘记小数点位置与分母 10ⁿ 的对应关系（如 0.16 = 16/100），尤其是小数位数较多时 | 教材已给出范例 0.16→16/100→4/25，课堂可追加 1-2 个类似例子让学生模仿，再逐步过渡到自主转化 |
| 无限循环小数化分数 | 习题 A 组第 3 题为探究题，方法涉及设未知数、乘 10ⁿ、相减消循环节等步骤，推理链较长 | 本题定位为拓展探究，可在课堂简要提示思路后留作课后自主查阅；中间层和拓展层学生可在下节课分享方法 |
| "整数但不是正数"等交叉条件 | B 组第 4 题同时涉及"整分"和"正负"两个维度，学生需先判断整数性再判断正负性，容易漏选或多选 | 引导学生分两步：首先圈出所有整数，再从中排除正数；或首先圈出所有非正数，再从中挑出整数 |

### 3.3 易错点分析

| 易错点 | 错误表现 | 原因分析 | 预防策略 |
|:---|:---|:---|:---|
| 0 是正数 | 将 0 填入"正数圈" | 对"正数是已学过的数（0 除外）前面加'+'"的定义中"0 除外"的条件关注不够 | 在定义阶段强调"0 除外"并板书标注；做一做填圈题中将 0 作为陷阱故意放入选项，让学生体验"0 不属于任何一个圈" |
| 分数 = 所有小数 | 将 -6.5、-0.01 等小数排除在"分数"之外，认为只有带分数线的才是分数 | 未建立"有限小数可以化为分数"的联结 | 在"分数与小数互化"环节后，明确告知：判断一个数是否为分数，关键看能否化为 a/b（b≠0）的形式，而非看书写形式 |
| 分类双重标准 | 在完成"按整分分类"时，同时在子类别中按正负标注，导致分类混乱 | 对"分类标准唯一性"的理解不足 | 在大家谈谈环节对比展示两种分类结果，强调"标准不同，分出的类别也不同，但同一分类中标准必须唯一" |
| 正数"+"省略后的遗忘 | 看到 3、12、19 等不带"+"号的数时，忘记它们是正数 | 对"正数中的'+'可以省略"理解不牢固，误以为省略后不再是正数 | 在板书正负数定义时同时给出带"+"和不带"+"的两组示例，明确标注"它们是同一个数" |

> **说明**：`knowledge/common-errors/number-algebra.md` 对应第 14 章（分式），不含第 1 章"有理数"相关内容；`knowledge/math-essence/` 当前仅有"正方形"条目，无"有理数"本质分析文件。以上易错点基于教材原文分析和教学经验梳理。

## 四、教学目标（基于2022年新课标 · 教学评一体化）

| 目标类型 | 学习经验表述 | 核心素养指向 | 对应评价证据 |
|:---|:---|:---|:---|
| 概念形成 | 经历从"用带'+''-'的数表示相反意义的量"到"正数和负数的形式化定义"的抽象过程，能用教材原文的语言说出正数和负数的定义，并能举出正数和负数的实例 | 抽象能力 | 1. 口头复述正数、负数定义；2. 每人写 3 个正数和 3 个负数（习题 A 组第 2 题）；3. 能解释"+"为什么可以省略 |

| 有理数分类 | 通过阅读教材原文和"大家谈谈"交流，能用两种分类方法（按整数/分数分、按正/0/负分）对给定的有理数进行分类，并能说明两种分类方法的标准不同 | 推理能力、抽象能力 | 1. 完成做一做第 1、3 题和填圈题；2. 能独立画出有理数的两种分类结构图；3. 小组讨论：两种分类结果为什么不同？ |
| 分数与小数互化 | 通过教材范例和模仿练习，能将有限小数化为分数（约简到最简分数），知道无限循环小数也可以化为分数（留待探究） | 运算能力 | 1. 独立完成 0.24 → 24/100 → 6/25 等 2-3 个有限小数化分数的练习；2. 能判断给定的小数是否可以化为分数 |
| 综合应用 | 能运用正负数定义和有理数分类知识，解决交叉条件分类（如"整数但不是正数""分数但不是负数"）和实际情境中的正负数表示问题 | 推理能力、应用意识 | 1. B 组第 4 题的正确分类；2. C 组第 6 题的正负数理解与逆向推理；3. 习题 A 组第 1 题的正负数表示 |

### 目标达成预期：通过上述经验，学生能理解负数和有理数的意义，掌握有理数的两种分类方法，能判断一个数属于有理数中的哪一类别，体会分类思想在数学中的运用。

## 五、教学重点与难点

| 类型 | 内容 | 依据 |
|:---|:---|:---|
| **教学重点** | 正数和负数的形式化定义；有理数的概念与两种分类方法 | 核心概念建立——有理数是整个第 1 章的基石概念，分类方法是后续数轴表示、大小比较和运算的基础 |

## 六、方法分析

### 6.1 概念建构方法

| 方法 | 具体做法 |
|:---|:---|
| 回顾引入 | 快速回顾 1.1.1 的"相反意义的量"和"用带'+''-'的数表示"，自然过渡到"这些带'+''-'的数分别叫作正数和负数"的形式化定义 |
| 情境抽象 | 通过一起探究"警戒水位"情境，让学生在实际问题中体验 0 作为参考点的含义，再抽象到"0 既不是正数也不是负数"的数学结论 |
| 阅读+提炼 | 让学生阅读教材中"正整数、0、负整数、正分数、负分数→整数、分数→有理数"的定义段落，教师用板书逐步画出分类结构图 |

### 6.2 探究活动设计

| 活动 | 时间 | 目的 |
|:---|:---:|:---|
| 回顾与引入 | 3 min | 从 1.1.1 的相反意义的量自然过渡到正负数定义 |
| 定义形成 | 5 min | 教师引导阅读教材定义→学生复述→板书标注"0 除外""+可省略"等关键条件 |
| 一起探究 | 7 min | 逐问讨论警戒水位情境，重点突破"0 m 的含义"→引出 0 既不是正数也不是负数 |
| 有理数分类阅读与建构 | 8 min | 学生阅读教材分类段落→教师板书分类结构图→学生复述分类逻辑 |
| 大家谈谈 | 5 min | 小组讨论两种分类方法→代表发言→教师对比总结两种分类标准 |
| 做一做 | 7 min | 当堂练习 4 道分类与判断题，重点关注 0 的归属处理 |
| 课堂小结 | 3 min | 回顾正负数定义、0 的意义、有理数两种分类，强调易错点 |
| 布置作业 | 2 min | 明确习题 A/B/C 组要求，提示 C 组为选做 |

### 6.3 数学思想渗透

| 思想 | 渗透点 | 方式 |
|:---|:---|:---|
| **分类思想** | 有理数的两种分类方法（按整分/按正负） | 大家谈谈环节明确展示"同一集合按不同标准得到不同分类结果"，板书时并列呈现两种分类树形图，引导学生比较异同 |
| **抽象思想** | 从警戒水位情境中具体的 0 m → 数学中"0 既不是正数也不是负数"的抽象命题 | 一起探究三步走：先算实际水深（具体）→再观察 0 m 的含义（半抽象）→最后归纳 0 的数学性质（抽象） |
| **对应思想** | 分数与有限小数/无限循环小数的双向互化 | 教师用"有限小数 → 除以 10ⁿ → 约分 → 分数"和"分数 → 分子÷分母 → 小数"的双向箭头板书，建立对应关系 |
| **符号意识** | 正数中"+"可省略、负数中"-"不可省略的符号约定 | 通过对比板书"3 和 +3 是同一个数，-3 不能写成 3"强化符号规则的约定性质 |

### 6.4 例题与习题教学关键动作

| 栏目 | 关键动作 | 注意事项 |
|:---|:---|:---|
| 一起探究 | 逐问讨论，第(2)问是核心——让学生先说"0 m 是什么意思"，教师引导归纳出"不高也不低"，再上升到"0 既不是正数也不是负数" | 不要直接给出结论，要通过追问让学生自己体会 0 的特殊性 |
| 有理数分类构建 | 板书从左到右逐步生长分类树：有理数→整数/分数→正整数/0/负整数 + 正分数/负分数 | 每一步都让学生先说，教师只在必要处补充 |
| 做一做填圈 | 故意将 0 放入选项，观察学生是否将 0 填入任一圈内；如有，请其他学生纠正 | 0 在两个圈之外——这是学生对 0 认识的第一个"仪式性时刻" |
| 大家谈谈 | 分两步：先让学生完成"整分"分类的细化（正向建构），再追问"还能怎么分"引出第二种分类（逆向启发） | 第二步可能需要提示"想一想按符号能怎么分" |
| B 组第 4 题 | 引导学生分两步判断：①先判断整数/分数属性，②再判断正/负/0 属性，最后取交集 | 这是两个维度交叉的练习，对基础层学生可简化条件（如只要求找出"整数但不是正数"） |
| C 组第 6 题 | 第(1)问可直接运用正负偏差概念；第(2)问需要逆向推理，建议作为选做题 | 第(2)问涉及不等式思想（d - 0.4 ≤ 所有合格直径 ≤ d + 0.4），对多数学生有挑战 |

## 七、流程分析

### 7.1 教学环节划分

| 教学环节 | 时间 | 核心任务 |
|:---|:---:|:---|
| 课前回顾与引入 | 3 min | 回顾相反意义的量 → 引出正负数定义 |
| 新课学习 | 25 min | 定义形成(5') → 一起探究(7') → 有理数分类建构(8') → 大家谈谈(5') |
| 当堂练习 | 7 min | 做一做 4 题（含填圈和判断） |
| 课堂小结 | 3 min | 回顾正负数定义、0 的意义、两种分类方法；强调易错点 |
| 布置作业 | 2 min | A 组必做、B 组选做、C 组挑战 |

**总时长：40 min**

### 7.2 环节衔接

- **课前回顾 → 新课学习**：通过"上节课我们用带'+'和'-'的数表示相反意义的量，今天我们来给这些数起个正式的名字"自然过渡到定义形成
- **定义形成 → 一起探究**："正数负数有了定义，那 0 属于哪一边呢？我们通过一个水库水位的例子来研究"
- **一起探究 → 有理数分类建构**："有了正数、负数、0，我们学过的数可以怎么分类？请阅读教材这一页"
- **分类建构 → 大家谈谈**："刚才我们按整数/分数分了类，想一想，还有别的分类方法吗？"
- **大家谈谈 → 当堂练习**："学会了两种分类方法，现在来试一试——做一做"
- **当堂练习 → 课堂小结**：练习讲评完毕后用三个问题回看：①正数是什么？负数是什么？②0 属于正数吗？属于负数吗？③有理数有哪两种分类方法？
- **课堂小结 → 布置作业**：小结后明确习题分层要求
