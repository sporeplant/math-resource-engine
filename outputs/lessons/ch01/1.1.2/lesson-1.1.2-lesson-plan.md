# 1.1 正数和负数（第二课时）教学设计

## 一、教学内容

冀教版七年级上册第一章《有理数》1.1 正数和负数第二课时。核心内容：正数和负数的形式化定义、0 既不是正数也不是负数（是正负数的分界）、有理数的概念与两种分类方法（按整数/分数、按正/0/负）、分数与有限小数的互化。本课时为概念建构课，承接第一课时"用带'+''-'的数表示相反意义的量"，完成概念的抽象化与体系化。

## 二、教学目标

1. 能说出正数和负数的定义，辨认给定各数中的正数和负数。
2. 能解释 0 既不是正数也不是负数的理由，用两种标准对有理数进行分类。
3. 能分析交叉条件（如"整数但不是正数"），筛选满足条件的数。

## 三、教学重点与难点

**教学重点**：正数和负数的形式化定义；有理数的概念与两种分类方法。

**教学难点**：0 的意义（既不是正数也不是负数，是正负数的分界）；两种分类标准不混淆。

## 四、教学准备

教材、练习本、黑板。学生已在第一课时学习了用带"+"和"−"的数表示相反意义的量，生活中接触过警戒水位、零件误差等涉及正负基准的情境。

## 五、教学过程

### （一）回顾与定义形成（5分钟）

教师提问："上节课我们学会了用带'+'和'−'的数表示什么？谁举例？"基础层学生举手举例（温度、海拔等）。过渡："今天给这些数起正式名字。"学生默读教材第一段定义。

提问："什么叫正数？什么叫负数？"学生用教材语言复述，教师板书定义，标注"0 除外""+ 可省略"。追问："+3 和 3 一样吗？-5 能省略'−'写成 5 吗？为什么？"辨析符号约定的不可替换性——正号可省略，负号不能省略。

### （二）一起探究——0 的归属（7分钟）

教师口述情境（警戒水位 5 m，超过记正、低于记负），板书简图。

第(1)问："+1 m 表示什么？实际水深多少？"基础层学生快速回答（高 1 m，实际 6 m）。

⭐ 第(2)问："0 m 表示____？实际水深多少？"停顿 2 分钟让学生独立思考。必要时逐级提示：0 m 是比警戒水位高了还是低了？既不高也不低说明什么？学生说出"不高不低，实际水深就是 5 m"后，追问："0 m 不高不低——它属于正数还是负数？"引导归纳：0 既不是正数也不是负数，是正数和负数的分界。教师板书结论。

第(3)问："-2 m 表示什么？实际水深多少？"快速通过（低 2 m，实际 3 m）。

### （三）有理数分类建构（8分钟）

"有了正数、负数和 0，我们学过的数怎么整理？"学生默读教材分类段落。教师逐步板书分类树：正整数举例 → 负整数 → 0 → 完成"整数"分支 → 正分数 → 负分数 → 完成"分数"分支 → 汇合为"有理数"。

知识链接："0.16 怎么化成分数？"学生模仿教材例子：$0.16 = \frac{16}{100} = \frac{4}{25}$。辨析："-6.5 是不是分数？"——打破"只有带分数线才是分数"的误解，有限小数可化为分数，$-6.5 = -\frac{65}{10}$，是负分数。

### （四）大家谈谈——第二种分类（5分钟）

同桌讨论："有理数还可以按什么标准分类？"学生代表发言，教师板书第二种分类树（正有理数 / 0 / 负有理数）。两种分类树并列对比："为什么同一堆数有两种分法？"结论：分类标准不同，结果不同；每次分类只能用一个标准。

### （五）当堂练习（10分钟）

1. **做一做填圈题**（2.5分钟）：将 $3, -\frac{1}{2}, 0, 12, -6.5, \frac{7}{8}, -24$ 填入正数圈和负数圈。独立完成 → 全班核对。重点：0 不填入任何一个圈。
2. **做一做第 1 题**（2分钟）：判断 9 个数的正负。独立完成 → 同桌互查。
3. **做一做第 2 题**（0.5分钟）：有没有既不是正数也不是负数的有理数？口头抢答。
4. **做一做第 3 题**（2分钟）：将 6 个数填入正数圈 / 负数圈。独立完成 → 核对。
5. **习题 B 组第 4 题**（2.5分钟）：找出"整数但不是正数"和"分数但不是负数"的数。独立完成 → 选人展示筛选思路：先判断整/分属性，再判断正/负/0 属性。

### （六）课堂小结与布置作业（5分钟）

三问回顾（2分钟）：① 正数 / 负数定义？② 0 属于哪边？③ 有理数两种分类方法？强调易错点（1分钟）：0 不填入正数圈也不填入负数圈；-6.5 是分数；分类标准不能混用。布置作业（2分钟）。

## 六、当堂检测

1. 做一做填圈题：将 7 个数填入正数圈和负数圈。检测目标：基础层——辨认正负数，0 不填圈。限时 2.5 分钟。
2. 做一做第 1 题：判断 9 个数正负。检测目标：基础层——辨认正负数。限时 2 分钟。
3. 做一做第 3 题：将 6 个数填入正数圈 / 负数圈。检测目标：基础层——正负数分类。限时 2 分钟。
4. 习题 B 组第 4 题：交叉条件筛选。检测目标：拓展层——分析交叉条件。限时 2.5 分钟。

检测题全部取自教材做一做和习题 B 组，共 5 道，当堂核对答案。成功标准：前 3 题全部正确；第 4 题"整数但不是正数"和"分数但不是负数"各至少选对一个。

## 八、板书设计

**1.1 正数和负数（第二课时）**

**1. 定义**
正数：已学数（0 除外）前加"+"，+ 可省略（+3 = 3）
负数：已学数（0 除外）前加"−"，− 不可省略

**2. 分类①：按整数/分数**
```
          有理数
          ↙    ↘
       整数    分数
      ↙ │ ↘   ↙  ↘
  正整数 0 负整数 正分数 负分数
```

**3. 分类②：按正/0/负**
```
          有理数
       ↙    │    ↘
   正有理数  0  负有理数
   ↙    ↘      ↙    ↘
 正整数 正分数 负整数 负分数
```

★ 0 既不是正数也不是负数，是正负数的分界。
★ 分类标准不同，结果不同；每次分类只能用一个标准。

## 九、设计依据简记

本节遵循教材原文顺序：定义形成 → 一起探究 → 分类构建（+ 知识链接）→ 大家谈谈 → 做一做 → 习题（课后）。练习和习题全部取自教材，无自编题。

一起探究中警戒水位情境是突破"0 的归属"的关键——先让学生体验"0 m 不高不低"的直观意义，再抽象到"0 既不是正数也不是负数"。两种分类树并列呈现，帮助学生建立"分类标准决定分类结果"的思想。

课内练习 5 道（做一做 4 题 + B 组第 4 题），课后必做 3 题（A 组第 1、2 题 + B 组第 5 题），选做 2 题（A 组第 3 题 + C 组第 6 题）。教材全部 13 道题均已分配，无遗漏。无练习册资源（ch01 未入库）。

## knowledge_analysis

**知识本质**：概念建构——在正有理数的基础上完成"正数""负数"的形式化定义，建立有理数的两种分类体系。核心数学关系是：数的符号（+/−/无）决定数在分类体系中的位置。

**知识生长路径**：
小学自然数、小数、分数 → 1.1.1 用带"+"和"−"的数表示相反意义的量 → **1.1.2 正负数形式化定义 + 有理数两种分类** → 1.2 数轴 → 1.3 绝对值与相反数 → 1.4 有理数大小比较 → 1.5+ 有理数运算

**数学思想**：分类思想（同一集合按不同标准得到不同划分）、抽象思想（从警戒水位情境中具体的 0 m 抽象到"0 既不是正数也不是负数"）、符号意识（正号可省略、负号不可省略的约定性质）

**学习困难**：
- 0 的归属：学生可能认为 0 是正数，或认为 0 是正整数
- 分类标准混淆：在同一分类中混用"整/分"和"正/负"两个标准
- 小数=分数的认知断裂：认为只有带分数线的才是分数，忽略有限小数可化分数

**关键教学策略**：
- 一起探究中先让学生体验"0 m 不高不低"，再抽象到数学结论
- 两种分类树并列板书，强化"标准决定分类"
- 辨析"-6.5 是不是分数"打破书写形式的限制

**为什么教材只选这两种分类**：理论上可按任意明确标准对有理数再次划分（如按"大于1/等于1/0到1之间/等于0/负数"等），但教材选定"整/分"和"正/0/负"两个维度有其设计意图——第一种回答"有理数由什么组成"（定义其基本结构），第二种回答"有理数在数轴上怎么分布"（直接衔接下一课数轴）。二者分别从"类型"和"符号"两个最核心的维度建立有理数概念体系。教师可用"全班按性别分 vs 按身高分"类比，帮助学生理解同一集合因标准不同而产生不同划分结果，"有没有第三种"可作为拓展层学生的课后思考。

## boardwork

**1.1 正数和负数（第二课时）**

**1. 定义**
正数：已学数（0 除外）前加"+"，+ 可省略（+3 = 3）
负数：已学数（0 除外）前加"−"，− 不可省略

**2. 分类①：按整数/分数**
```
          有理数
          ↙    ↘
       整数    分数
      ↙ │ ↘   ↙  ↘
  正整数 0 负整数 正分数 负分数
```

**3. 分类②：按正/0/负**
```
          有理数
       ↙    │    ↘
   正有理数  0  负有理数
   ↙    ↘      ↙    ↘
 正整数 正分数 负整数 负分数
```

★ 0 既不是正数也不是负数，是正负数的分界。
★ 分类标准不同，结果不同；每次分类只能用一个标准。
