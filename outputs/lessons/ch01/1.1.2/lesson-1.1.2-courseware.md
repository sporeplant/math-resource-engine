### 📖 有理数的分类 ① —— 按整数 / 分数

```
          有理数
          ↙    ↘
       整数    分数
      ↙ │ ↘   ↙  ↘
  正整数 0 负整数 正分数 负分数
```

正整数、0 和负整数统称为 **整数**。

正分数和负分数统称为 **分数**。

整数和分数统称为 **有理数**。

---

### 📖 知识链接：分数与小数

**任何分数都可以化为有限小数或无限循环小数；反过来也成立。**

例：$0.16 = \dfrac{16}{100} = \dfrac{4}{25}$

$-6.5$ 是不是分数？说说理由。

### 📖 两种分类方法

| 分类 ①：按整 / 分 | 分类 ②：按正 / 0 / 负 |
|:---|:---|
| 有理数 → 整数 + 分数 | 有理数 → 正有理数 + 0 + 负有理数 |
| 整数 → 正整数 + 0 + 负整数 | 正有理数 → 正整数 + 正分数 |
| 分数 → 正分数 + 负分数 | 负有理数 → 负整数 + 负分数 |

同一堆数，为什么有两种分类结果？

请一位同学回答。
