# 22.4 频数分布与直方图 HTML课件中间稿

## 课件定位

本文件是 MRE 课件结构文本与 guizang HTML deck 之间的中间稿，不是最终教学设计，也不是最终 HTML。

MRE 负责教学顺序、题源追溯、课堂活动和评价一致性；guizang-ppt-skill 只负责把页面结构渲染成横向翻页网页课件。

本案例采用 Style B 瑞士国际主义风，适合数据整理、统计表、直方图、对比辨析等信息型页面。

## 生成约束

- 必须遵循教材原文顺序，先从数据整理需求进入，再建立频数分布表，最后过渡到频数分布直方图。
- 每页只承载一个核心教学意图，避免把完整讲稿堆进页面。
- 课堂练习和检测页必须保留题源字段。
- 页面文案优先短句、问题、任务指令，不写大段教师讲述。
- HTML 生成时优先使用 guizang Swiss 版式，不临时发明复杂页面结构。

## Page 01 课题导入

### 页面目标

让学生意识到：面对一组较多数据，直接观察很难发现整体分布，需要整理数据。

### 页面文本

主标题：频数分布与直方图

副标题：从“一堆数据”到“看见分布”

页脚：22.4 · 数据的整理与表达

### 画面建议

使用大标题加少量散点数字作为背景元素。不要放完整表格，避免开场信息过密。

### 教师提示

如果给你全班同学的身高数据，你能一眼看出主要集中在哪个范围吗？

## Page 02 问题情境

### 页面目标

制造认知冲突：原始数据能记录事实，但不方便看整体。

### 页面文本

大句：数据越多，越需要整理。

辅助文本：原始数据告诉我们“每一个是多少”，统计图表帮助我们看到“整体怎样分布”。

### 学生活动

学生观察一组原始数据，回答：哪个范围的数据最多？你能快速判断吗？

### guizang版式建议

使用 Statement 页面，左上标题，右下放一组小字号原始数据块。

## Page 03 数据整理任务

### 页面目标

呈现从原始数据到频数分布表的操作步骤。

### 页面文本

步骤一：确定数据范围

步骤二：合理分组

步骤三：统计每组频数

步骤四：填写频数分布表

### 学生活动

学生完成一个小数据集的分组统计。

### 页面要求

保留一个待填写区域或空表结构，不直接把完整答案呈现出来。

## Page 04 核心概念

### 页面目标

明确三个核心概念，并让学生能用自己的话解释。

### 页面文本

频数：落在某一范围内的数据个数。

组距：每一组两个端点之间的距离。

频数分布表：用表格表示各组数据出现次数的整理结果。

### 页面要求

每个概念配一个小例子，不堆定义。

## Page 05 直方图生成

### 页面目标

建立“频数分布表”和“频数分布直方图”的对应关系。

### 页面文本

横轴：数据分组

纵轴：频数

矩形高度：表示该组频数

### 图示说明

左侧放频数分布表，右侧放频数分布直方图，中间用箭头连接。

### 教师追问

为什么直方图中的矩形通常是连续排列的？

## Page 06 易错辨析

### 页面目标

区分条形统计图和频数分布直方图。

### 页面文本

条形统计图：比较不同类别的数量。

频数分布直方图：观察连续数据在不同范围内的分布。

### 对比表

| 对比点 | 条形统计图 | 频数分布直方图 |
|---|---|---|
| 横轴 | 类别 | 数据范围 |
| 矩形 | 可分开 | 通常连续 |
| 目的 | 比较类别 | 观察分布 |

## Page 07 课堂练习

### 页面目标

让学生根据频数分布表完成直方图，或根据直方图读取信息。

### 题源标记

```yaml
source_id: textbook-22.4
source_type: textbook
question_id: "22.4-practice-1"
```

### 页面文本

任务：根据频数分布表，完成频数分布直方图，并回答人数最多的数据范围。

### 反馈点

检查横轴分组是否等距。

检查纵轴频数是否准确。

检查矩形高度是否对应频数。
