## overview

**核心问题**：如何从统计图表中提取有效信息，并用数据描述和论证结论？

**教学流程**：
1. 回顾唤醒（4分钟）：复习统计表、条形图、直方图、折线图的基本概念
2. B组第7题（6分钟）：统计表填写 + 条形图绘制
3. B组第9题（5分钟）：图表信息提取（口头计算）
4. B组第8题（6分钟）：频数分布分析（教师展示直方图）
5. B组第10题（7分钟）：分组方案设计 + 频数表填写
6. C组第12题（5分钟）：数据对比分析（口头回答）
7. 小结（2分钟）：总结数据描述与分析的关键方法

**能力层次**：
- 基础层：频数统计、条形图绘制
- 中间层：信息提取、分布描述
- 拓展层：数据论证、对比分析

## objectives

### 基础层

- **LO-B-01**：能观察给定的数据表，说出各类数据的频数和频率，填写完整统计表
- **LO-B-02**：能模仿教材示例，画出条形统计图或折线统计图表示数据

### 中间层

- **LO-M-01**：能归纳数据的分组方案，列出频数分布表并画出频数分布直方图
- **LO-M-02**：能从不完整的扇形统计图和条形统计图中概括出隐含的数量关系，计算出缺失数据
- **LO-M-03**：能描述频数分布直方图或折线统计图所反映的数据分布特征和变化趋势

### 拓展层

- **LO-E-01**：能分析两组数据的统计图表，论证不同数据集的差异，用数据支撑判断结论

## questions

### 全课问题主线

```
回顾四种统计图用途（ASK_B_01）
  → 原始数据→统计表→条形图转化（ASK_B_02 → ASK_M_01）
  → 不完整图表提取信息反推数据（ASK_M_02）
  → 频数分布表求未知频数+画直方图（ASK_M_03）
  → 自主确定分组方案+描述分布特征（ASK_M_04 → ASK_E_01）
  → 折线图多组数据对比+用数据论证差异（ASK_E_02）
  → 归纳统计tools-2选择策略（ASK_M_05）
```

### 逐题详情

**ASK_B_01**
- 提问：上节课我们学了四种统计图。请看着教材第1课时的知识结构图，说出条形图、扇形图、折线图、直方图分别用来描述数据的什么特征？
- 备用提示1：想一想每种图的横轴和纵轴分别表示什么。
- 备用提示2：条形图比较数量，扇形图表示比例，折线图看变化，直方图看分布。
- 教师参考预期：学生能说出四种统计图的基本用途，为后续各题选择合适统计tools-2做铺垫。

**ASK_B_02**
- 提问：第7题的40袋面粉数据中，误差为0kg的有多少袋？你是怎么数出来的？
- 备用提示1：先看表中哪些数据是50.0kg，一袋一袋数。
- 备用提示2：数完后检查总数是否为40。
- 教师参考预期：学生能逐袋对照标准质量50kg，说出误差为0kg的有12袋。

**ASK_M_01**
- 提问：画第7题的条形统计图时，横轴标什么、纵轴标什么？条形的高度对应统计表中的哪一列？
- 备用提示1：横轴是"误差/kg"的各个取值，纵轴是"袋数"。
- 备用提示2：条形高度 = 该误差对应的袋数，不是百分比。
- 教师参考预期：学生能区分条形统计图与直方图的横轴差异，正确完成作图。

**ASK_M_02**
- 提问：第9题的扇形统计图中，"步行占40%"对应条形统计图中步行人数为16人。根据这两个信息，怎样求出全班总人数？

**ASK_M_04**
- 提问：第10题有48个橘子维生素C含量数据，最小26.2、最大32.8。请计算极差，并说明你打算分几组、组距取多少？
- 备用提示1：极差 = 最大值 - 最小值 = 32.8 - 26.2 = 6.6。
- 备用提示2：数据个数48，通常分8组左右。组距 = 极差 ÷ 组数，取整。
- 教师参考预期：学生能说出极差=6.6，分组方案合理（如分7-8组，组距≈1），不同学生的分组方案可以有差异。
