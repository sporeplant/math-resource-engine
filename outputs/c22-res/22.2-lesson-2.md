## meta

- grade: 八年级
- topic: 22.2 数据的收集(第2课时)——抽样调查的代表性
- duration: 35分钟

## 知识分析

```yaml
知识主题: 抽样调查的代表性

课前强调:
  ★ 总体和个体都是"数据"，不是"人/物"本身。
    错误说法：总体是3000名学生，个体是每名学生。
    正确说法：总体是3000名学生选择的项目（数据），个体是每名学生选择的项目（数据）。
    本节课涉及的所有总体、个体、样本，都必须指向"数据"。

知识本质:
  学生要学的核心是：为什么不同的抽样方式会导致调查结果差异很大，
  以及如何设计抽样方案才能使样本对总体具有较好的代表性。
  本节课最重要的数学关系是：样本代表性 ↔ 抽样方式的科学性。

知识来源:
  源于第1课时已建立的"总体、个体、样本、样本容量、简单随机抽样"
  等基本概念。第1课时解决了"什么是抽样调查"，第2课时解决
  "怎样抽样才合理"——从概念认知走向方法判断。

知识生长路径:
  第1课时：认识抽样调查的概念和术语
    → 第2课时：反思不同抽样方式的差异，发现代表性问题
    → 归纳出保证样本代表性的原则（样本容量适当、按比例分层抽样）
    → 应用：设计具有较好代表性的抽样方案

前置知识:
  - 普查与抽样调查的区别（22.2第1课时）
  - 总体、个体、样本、样本容量的概念（22.2第1课时）
    ★ 关键辨析：总体是要考察的"数据的全体"而非"对象的全体"，
      个体是"每个对象对应的数据"而非"对象本身"。
  - 简单随机抽样的含义（22.2第1课时）
  - 百分数与比例的计算

后续知识:
  - 22.3 数据的整理与描述
  - 22.4 频数分布与直方图
  - 用样本估计总体（样本平均数估计总体平均数）

核心思想:
  - 统计思想：通过样本推断总体
  - 代表性思想：样本结构与总体结构一致时，推断才可靠
  - 分类思想：按不同特征对总体分层，按比例抽样

学习难点:
  1. 理解"为什么同一总体不同抽样方式结果差异巨大"
  2. 理解"按比例分层抽样"的逻辑
  3. 从具体案例抽象出判断样本代表性的通用标准
  4. ★ 辨析总体/个体是"数据"而非"对象"

常见误解:
  - 误认为"样本容量越大，代表性越好"
  - 混淆"简单随机抽样"与"随意抽样"
  - 认为互联网调查一定比小样本调查更可靠（忽略自愿回应偏差）
  - ★ 将"人/物"误认为总体和个体，忽略了总体和个体都是指"数据"
```

**课型**：新授课

### 拓展层

- LO-E-01：能**设计**一个具有较好代表性的分层抽样方案，说明分层依据和各层抽样数量的计算过程

## lesson_flow

### 问题链

```yaml
问题链ID: ChainQ_01

核心问题:
  怎样抽样才能保证样本对总体有较好的代表性？

子问题1:
  为什么同样的节目，不同人调查得到的收视率差别很大？

子问题2:
  什么样的样本对总体有较好的代表性？判断标准是什么？

子问题3:
  如何设计一个具有较好代表性的抽样方案？

认知冲突:
  已有认知：第1课时学了简单随机抽样，认为随机抽取就够了
  可能误解：样本容量越大越好
  冲突问题：小强调查了100人，电视台调查了1000人，但小强的结果偏差更大——为什么？
  目标结论：代表性取决于样本结构与总体结构是否一致，而非仅看样本容量

最终结论:
  保证样本代表性的原则：样本容量适当 + 按比例分层抽样（样本结构与总体结构一致）
```

### 活动三：分析归纳样本代表性标准（规律形成）— 9分钟

活动目标：LO-M-01 解释样本代表性好坏的标准；LO-M-02 归纳保证样本代表性的基本原则

活动类型：归纳

**关联问题链ID**：ChainQ_01

活动步骤：
1. 学生阅读教材"大家谈谈"第1-4题，独立思考
2. 台阶①：教师提示"从**调查的人群**和**调查的数量**两个角度去看"，提问："第1题，小红调查的是全班同学，小亮调查的是火车站的人——他们调查的**人群**有什么不同？调查的**数量**差多少？"（基础层可答）
3. 台阶②：提问："第2题，电视台和四名同学相比，在**人群构成**上做了什么不同的事？"（中间层可答——按不同年龄段、文化背景按比例确定调查对象）
4. 台阶③：提问："第3题，综合以上分析，抽样调查应该注意什么？"（中间层可答——归纳原则：人群要有代表性、数量不能太少）
5. 教师板书归纳：样本容量适当 + 按比例分层抽样 → 样本结构与总体结构一致 → 代表性好
6. 提问："第4题，抽样调查的优点和缺点各是什么？"（中间层可答——优点：节省时间、比较经济；缺点：结果不如普查准确）
7. 教师追问辨析："回头看刚才的收视率案例——小强调查了100人，结果40%；电视台调查了1000人，结果只有3.5%。有同学可能会想：样本容量大的结果应该更接近真实吧？但为什么100人的结果比1000人的结果差这么多？这说明决定代表性的关键到底是什么？"（呼应认知冲突——引导学生从案例数据中得出"代表性取决于样本结构与总体结构是否一致，而非仅看样本容量"）

反馈节点：步骤5归纳后追问辨析，步骤6优缺点回答后教师补充教材原文总结，步骤7追问后确认认知冲突已解决

学生产出：口头归纳

成功标准：能说出"样本结构与总体结构一致时代表性好"，能归纳出"样本容量适当、按比例分层"两个原则，能说出抽样调查的优缺点

教材对应位置：大家谈谈（第1-4题）+ 教材正文总结段落

题源字段：
- source_id: 教材原文_22.2_数据的收集(第2课时)
- source_type: textbook
- question_id: 22.2-2-大家谈谈-1,2,3,4

活动升华：
- 核心要点：样本代表性取决于样本结构与总体结构是否一致；抽样调查省时经济但不如普查准确
- 思想方法：统计推断思想——用样本推断总体需要样本有代表性
- 关联模型：简单随机抽样保证每个个体机会均等
- 后续应用：设计抽样方案时必须考虑代表性
