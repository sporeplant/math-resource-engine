# 21.6 菱形 第1课时 教学设计

### ——当边变得特殊

## 一、课前速览

### 1.1 课时信息

| 项目 | 内容 |
|:---|:---|
| 课题 | 21.6 菱形（第1课时） |
| 课型 | 新授课 |
| 课时 | 40分钟 |

### 1.2 教学目标

| 目标编号 | 目标内容 | 核心素养指向 |
|:---:|:---|:---|
| 目标① | 能用自己的语言描述菱形与平行四边形的区别，准确给出菱形定义 | 几何直观 |
| 目标② | 能通过折叠实验发现菱形的对称性，并用自己的话描述对角线的关系 | 抽象能力 |
| 目标③ | 能完成性质定理的证明，理解"为什么要证"和"怎么想到这样证" | 推理能力 |
| 目标④ | 能运用菱形性质解决计算问题，体会"性质是用来用的" | 运算能力 |

### 1.3 教学重难点

| 类型 | 内容 |
|:---|:---|
| **教学重点** | 菱形性质定理的发现与运用 |
| **教学难点** | 性质定理证明中"为何想到SSS全等"的思维路径 |

### 1.4 整体设计思路

**核心线索**：平行四边形加一个条件会怎样？矩形加了"一个直角"——角特殊；菱形加了"一组邻边相等"——边特殊。用矩形课上走过的方法再走一遍，让学生自己发现"研究方法完全一样"。

**学生认知路径**：
```
我知道菱形长什么样
    ↓ 需要说清楚"什么样"
我给菱形下定义
    ↓ 定义只是开始——它还有哪些隐藏性质
我通过折叠发现菱形的全部性质
    ↓ 折叠只是操作感受，我需要逻辑保证
我用推理证明这些性质
    ↓ 性质是用来解决新问题的
我运用性质做题
```

## 二、教学流程

### 环节一：从"认识"到"定义"（约5分钟）

**核心问题**：菱形你认识，但你能说清楚它"特殊"在哪儿吗？

**教学展开**：

教师依次展示生活中的菱形图片（风筝、地砖、窗花），每张图片只停留3秒，然后说：

"上节课我们学了矩形——矩形是什么？**平行四边形加一个直角**。那如果我给平行四边形换一个'加料'——不是加直角，而是让**一组邻边相等**——会得到什么？"

学生自然会说"菱形"。教师不满足于此，而是追问："好，但我要你说清楚——菱形和平行四边形到底差在哪？用一句话。"

**关键追问链**：

1. "我们已经有一个框架了——矩形是平行四边形加'一个直角'。那菱形可以看成平行四边形加什么？"
   - 预期：学生说"邻边相等"或"四边相等"
   
2. "如果我说'一组邻边相等'就够了，不需要四边相等，你同意吗？为什么？"
   - 这是第一个思维转折点。学生需要思考：平行四边形已经有"对边相等"了，所以"一组邻边相等"自然推出"四边相等"。
   - 教师不直接说，而是让学生自己推理出来。

3. "好，现在你能用一句话说清楚什么是菱形吗？"
   - 学生给出定义后，教师板书：**有一组邻边相等的平行四边形叫作菱形**。
   - 全班齐读一遍，教师强调两个关键词：**平行四边形** + **一组邻边相等**。缺一不可。

**过渡语**："定义告诉了我们菱形从哪里来。但它长什么样，还有哪些我们没发现的秘密？我们需要动手看看。"

**设计意图**：定义不是教师给的，而是学生在"想说服别人"的需求中自己提炼的。学生经历了从直觉判断到逻辑定义的跨越——这比直接背诵定义深刻得多。

---

### 环节二：折叠发现——菱形"藏"着三条秘密（约10分钟）

**核心问题**：一张菱形纸片，不借助任何tools-2，你能看出来它藏着几条性质？

**教学展开**：

教师手拿一张菱形纸片（课前剪好），展示给学生看，然后说："我用折叠的方法来探索这张菱形纸片，你们仔细观察——看看我每折一次，能发现什么。"

**演示与追问**：

**第一折**：沿一条对角线AC对折。
- 教师："看好了——两边完全重合了。这说明什么？"
- 学生：它是对称的。
- 追问："对称轴在哪儿？"（学生指出折痕AC）
- 追问："B和D跑到哪儿去了？"（学生：重合在一起了）
- 教师不急着说出结论，而是先放在这儿。

**第二折**：沿另一条对角线BD对折。
- 教师："也重合了。所以菱形有几条对称轴？"
- 学生：两条。
- 追问："普通的平行四边形有几条？"（学生：0条）"所以这是菱形特有的性质。"

教师顺势对比："上节课的矩形也有两条对称轴——但它们是对边中点的连线。今天菱形的对称轴却是两条对角线。有意思——'角特殊'的矩形，对称轴在边上；'边特殊'的菱形，对称轴在对角线上。"

**第三折**：绕两条折痕的交点O旋转纸片。
- 教师把纸片绕O点转180°——"跟原来一样吗？"
- 学生：一样。
- "所以O是中心对称点。菱形既是轴对称也是中心对称。"

**关键追问**（此时回到第一折的悬念）：

"刚才第一折时，B和D重合在一起了。B和D是什么关系？"

学生：它们是一对**对应点**。

"对。轴对称里有一条性质——对称轴和对应点连线的关系，完整怎么说？"

学生可能只记得"垂直"，教师追问："垂直就够了？对称轴经过连线的哪个特殊位置？"

引导学生完整回忆：**"对称轴是对应点连线的垂直平分线。"**——既垂直，又平分。

"好！那放在菱形里——AC是对称轴，B和D是对应点，BD就是对应点的连线。所以AC对BD既垂直又平分。也就是说——"

学生：AC ⟂ BD，且AC平分BD。

"同理，BD作为对称轴时，也垂直于AC并平分AC。这就同时得到了两个结论——对角线互相垂直，互相平分。其中'互相平分'本就是平行四边形的性质，折叠再次确认了。"

教师板书：**菱形对角线互相垂直**。

**继续深挖**（利用同一事实）：

"再看——B重合到D了。那∠BAC和∠DAC是什么关系？"

学生应该能回答：B落在D上，所以∠BAC和∠DAC重合了，它们相等。

"所以AC不仅垂直于BD，还把∠A平分了。类似地，它平分∠C吗？"
- 学生：平分（道理相同）。

"那沿着BD对折的时候呢？"
- 学生：BD平分∠B和∠D。

教师板书：**菱形对角线平分一组对角**。

**完整总结**（全部性质已经通过折叠揭示）：

"一张纸、两下折叠，我们就发现了菱形三条隐藏性质——"
1. 四条边都相等（从定义已可推出，折叠时也观察到）
2. 对角线互相垂直（B和D重合时）
3. 对角线平分一组对角（同理）

**过渡语**：

"折叠让我们'看到'了这些性质。但你可能会想——万一我这张菱形纸片剪得不够标准呢？或者换一个菱形，这些性质还成立吗？数学不能靠'一个纸片'说了算，我们需要证明——对所有的菱形，这些性质都成立。接下来我们就用逻辑推理来验证。"

**设计意图**：只折两下，读出了三条性质。关键不是"折"本身，而是帮学生回忆轴对称的旧知（对应点连线垂直于对称轴），然后用这个旧知解锁折叠中的新发现。学生经历的不是凭空想象，而是"旧知→新知"的推理——这比用"纸上的点怎么飞"的比喻严谨得多。

### 环节三：逻辑证明——用已知推出未知（约10分钟）

**核心问题**：从定义和平行四边形性质出发，你能证明折叠中发现的菱形性质吗？

**教学展开**：

教师板书证明题：
- 已知：菱形ABCD，对角线AC、BD相交于点O
- 求证：(1) AB=BC=CD=DA；(2) AC⊥BD

**证(1)的思维引导**：

"要证明四条边都相等——我们已经知道哪些边相等？"

学生应该能说出：
- 平行四边形 → 对边相等 → AB=CD，AD=BC
- 菱形定义 → 一组邻边相等 → 比如AB=AD
- 所以：AB=BC=CD=DA

教师让一名学生口述，全班核对，教师板书规范格式。

**证(2)的思维引导（难点）**：

这是本节课的思维高潮。教师的引导要慢、要精细。

**第一步：转换问题**
"刚才折叠时，我们发现AC和BD垂直——这是从'B和D重合'推理出来的。现在要用几何推理再证明一遍。要证AC⊥BD，在几何里怎么表示两条线垂直？"
- 学生：夹角为90°。
- 教师："对，AC和BD的交点是O。要证AC⊥BD，只需要证什么？"
- 学生：∠AOD=90° 或 ∠COD=90°（两者互为邻补角，证一个即可）

**第二步：寻找全等三角形**
"∠AOD怎么算？它和哪个角有关系？"
- 学生可能想到邻补角∠AOD+∠COD=180°，所以只要∠AOD=∠COD=90°就行。
- 教师："∠AOD和∠COD相等吗？你有什么办法证明它们相等？"
- 学生：可以证明△ADO≌△CDO。

**第三步：找全等条件（此处是学生最容易卡住的地方）**
"△ADO和△CDO——它们有哪些边或角可能相等？"

教师先给一个提示："上节课证矩形的对角线相等时，我们用了SAS全等——联系直角三角形斜边来推。今天证菱形的对角线垂直，思路一样——还是找全等三角形。但用的全等判定方法不同。你来找。"

教师不直接列出SSS，而是让学生自己找。可能的回答：
- AO=CO（平行四边形对角线互相平分）
- DO是公共边
- AD=CD（菱形邻边相等）
- 三个条件都有了：SSS全等 ✓

教师适时点评："矩形对角线相等的证明用的是SAS（两条边夹一个角），今天菱形用了SSS（三条边）。虽然都是看对角线分出的三角形，但矩形分出来的是直角三角形（有直角可用），菱形分出来的却是三边可求（邻边相等+对角线平分）。同一个思路——找全等——用在不同图形上，具体选哪个判据要看已知条件。"

**第四步：得出结论**
"现在你能完整地写出证明过程吗？"

学生独立书写，教师请一名学生板演，全班对照修正。板演者可能漏写"∠AOD+∠COD=180°"——这是常见错误，教师抓住这个生成点强调：**全等只能说明两角相等，结合邻补角关系才能推出90°**。

**第五步：补充证明——对角线平分一组对角**
"刚才折叠时，我们根据'B和D重合'直接判断了对角线平分对角。现在能不能用已证的性质推理出来？"
- 学生：△ADO≌△CDO → ∠ADO=∠CDO（一对对角被平分）
- 再结合平行线内错角可推出另外一对
- 这一问作为快速思考题，学生口答即可

**穿插练习**（A组第1题，快速口答）：

"刚刚我们证明了对角线平分一组对角。来，用一下这个结论——"

出示题目：菱形ABCD中，AC、BD为对角线，∠BAC=50°。求各内角度数。

"不需要写过程，心里想一下，告诉我答案就行。"

**思维路径**：
- ∠BAC=50° → 对角线平分对角 → ∠BAD=100°
- 邻角互补 → ∠ABC=80°
- 对角相等 → ∠BCD=100°，∠CDA=80°

这道题放在这里，不是为了练计算，而是让学生**第一时间用上刚学的性质**——证明完了就用，形成"性质是用来解决问题"的直觉。

**过渡语**：

"证完了，现在我们手里有三件'武器'（四条边相等、对角线垂直、对角线平分对角）。接下来看看这些武器怎么用。"

**设计意图**：证明不是"教师写学生抄"，而是让学生**自己找路径**。找全等条件的每一步都有推理依据，不是"因为书上这么写"。教师的作用不是在黑板上写满证明过程，而是问对问题——让学生在关键处被"卡"一下，然后自己跨过去。

### 环节四：例题——让性质"活"起来（约6分钟）

**核心问题**：知道了菱形的性质，怎么把它们用在计算里？

**教学展开**：

出示教材例1：菱形ABCD的周长为16cm，∠ABC=120°，求对角线BD和AC的长。

教师不直接讲，而是让学生**自己先读题**，然后问：

"这个题目给了两个条件：周长16cm和∠ABC=120°。你觉得哪个条件先派上用场？"

这不是随便一问——它训练学生判断条件的**使用顺序**：
- 周长 → 边长（4cm），这是四边相等最直接的应用
- 有了边长之后，再处理∠ABC=120°：BD平分∠ABC → ∠ABD=60°，结合AB=AD（邻边相等）→ △ABD是等边三角形 → BD=4cm

学生回答后，教师再问：

"我们已经知道边长了，∠ABC=120°这个条件怎么用？"
- 学生：BD平分∠ABC → ∠ABD=60°
- 教师追问："我们需要∠ABD=60°，还是需要知道别的？"
- 学生：有了∠ABD=60°，△ABD就有两个角60°+60°（等边三角形？）
- 教师追问："AB=AD吗？"
- 学生：是的（菱形邻边相等）→ 所以△ABD是等边三角形 → BD=AB=4cm

"BD求出来了，AC呢？"
- 学生：用勾股定理，在Rt△AOB中，OB=2，AB=4 → AO=2√3 → AC=4√3

教师板书完整过程，然后在旁边画一个菱形，标示所有的已知和求出的量，形成"数字图形"。

**过渡语**（自然引出练习）：

"例1帮我们梳理了菱形的解题路径：边相等→等边三角形→直角三角形→勾股定理。下面两道题考考你们——看能不能自己走通这条路。"

**设计意图**：例题的作用不是"演示答案"，而是**示范思维路径**。教师的追问让学生看到：拿到一个几何计算题，先明确"有什么条件"，再想"能推出什么"，最后问"要算什么"。这个元认知训练比答案本身重要得多。

---

### 环节五：练习——独立走一遍（约8分钟）

**核心问题**：刚才看我走了这条路，现在你自己走。

**练习1**（A组第2题）：菱形ABCD中，AC=8，BD=6，求周长。

这道题是例1的"逆用"：例1是从边求对角线，这道题是从对角线求边。教师要让学生在对比中体会"勾股定理是桥梁"。

**关键追问**（练习后反馈）：
"例1和这道题有什么相同的数学结构？"
- 学生：都用到了对角线垂直平分 → 直角三角形 → 勾股定理
- 教师：**"菱形的对角线垂直平分"是一个性质，"勾股定理"是一个tools-2——它们的组合能解决一大类问题。**

**练习2**（A组第3题）：菱形ABCD边长为2cm，E为AB中点，DE⊥AB，求面积。

这道题有难度。学生可能卡在"DE怎么求"上。

**预设困难与应对**：

- **困难1**：学生不知道怎么利用"E是中点且DE⊥AB"这个条件。
  - 引导："如果DE⊥AB，△ADE是什么三角形？"
  - 学生：直角三角形。
  - 教师："已知AE=1cm（中点），AD=2cm（菱形边长），能求DE吗？"
  - 学生：用勾股定理！DE=√(2²-1²)=√3。

- **困难2**：学生算出DE=√3后，不知道面积怎么算。
  - 引导："菱形面积你能想到几种算法？"
  - 学生可能会说：底×高 = 2×√3 = 2√3；或者对角线乘积÷2（但不知道对角线长）。
  - 教师："底×高现在有（底AB=2，高DE=√3），试试？"

**设计意图**：两道题的设计有递进——第一道是"直用"（勾股定理），第二道是"拐个弯"（先构造直角三角形再算面积）。学生需要在第一题中找到成功感，在第二题中"被卡一下"再突破，这样的节奏才不会平铺直叙。

---

### 环节六：拓展——面积还能这样算（约4分钟）

**核心问题**：同一个菱形，面积有好几种算法。你能找到几种？

**练习**（B组第4题）：菱形ABCD中，AC=6cm，BD=8cm，AE⊥BC于E，求AE。

教师引导：

"这道题不给边长，只给对角线。你有什么办法求AE？"
- 学生：需要用到面积法！

"菱形的面积可以怎么表示？"
- 学生1：S = ½×AC×BD = ½×6×8 = 24（对角线乘积的一半）
- 学生2：S = BC×AE（底×高）
- 教师："两个式子都等于S，所以呢？"
- 学生：½×AC×BD = BC×AE

"BC怎么求？"
- 学生：在Rt△AOB中，AO=3，BO=4 → AB=5 → BC=5

"所以AE = 24÷5 = 4.8cm"

教师总结："碰到求高的题，优先想**等积法**——用面积搭桥，把未知量转化成已知量。"

**设计意图**：这道题的价值不在于它有多难，而在于它展示了**面积的双重表达**——同一个菱形，可以用不同方式计算面积，两者相等就构成了一个方程。这是等积法的核心思想，也是后续几何学习的重要方法。

---

### 环节七：小结——我们经历了什么（约3分钟）

教师用三个问题带着学生回看这节课：

**问题1**："这节课开始时我们只'认识'菱形，现在你能用数学语言定义它吗？"
- 目的：回顾定义，强调"平行四边形 + 一组邻边相等"

**问题2**："我们发现了菱形的哪些秘密？是怎么发现的？"
- 学生梳理：折叠发现三条隐藏性质 → 从"一个纸片"到"所有菱形"的疑问 → 证明保证普适性
- 目的：回顾完整的研究路径

教师追问："回想一下——矩形那节课，我们也是同样的路径：从平行四边形加一个条件出发，折叠发现性质，用全等证明。不同的是：矩形加的是'一个直角'，菱形加的是'一组邻边相等'；矩形证明对角线相等用了SAS，菱形证明对角线垂直用了SSS。**研究方法完全一样——给平行四边形加条件，看性质怎么变。**这就是几何图形研究的一般方法。"

**问题3**："如果让你用一句话告诉没来上课的同学'菱形是什么样'，你会怎么说？"
- 这个问题比"菱形有什么性质"更有趣——学生需要把三条性质用通俗的语言打包outputs
- 预期："菱形就是四条边一样长的平行四边形，对角线垂直还平分对角。"
- 教师板书对应性质定理的标准表述

**最后1分钟**：教师展示菱形定义和两条性质定理的板书，让学生闭眼默述一遍。

**设计意图**：小结不是重复知识点，而是让学生把"碎片"整合成"结构"。第三个问题的设计，迫使学生在头脑中重组知识——这是比复述更深层的认知加工。同时，这也检查了目标①和②的达成情况。

## 三、弹性时间分配

| 环节 | 节拍 | 大致时间 | 弹性调整策略 |
|:---|:---:|:---:|:---|
| 环节一：从认识到定义 | 铺垫 | 5min | 定义提炼顺利则压缩至4min |
| 环节二：折叠发现 | 探究 | 10min | **核心**，不压缩；关键在"B和D重合说明了什么"的讨论 |
| 环节三：逻辑证明 | 突破 | 10min | **核心**，不压缩；A组第1题口答控制在2min内 |
| 环节四：例题应用 | 示范 | 6min | 精简板书时间，重点在追问 |
| 环节五：练习巩固 | 应用 | 8min | 时间不够时A组第3题改为口述思路 |
| 环节六：拓展提升 | 延伸 | 4min | 可弹性压缩至3min或并入机动时间 |
| 环节七：课堂小结 | 收束 | 3min | 可压缩至2min |
| **机动** | | **约0-2min** | 用于处理课堂生成的意外讨论 |

## 四、板书设计

黑板分两半：**主板**（左半，保留整节课）和**副板**（右半，随环节擦写）。

---

### 环节一（定义）结束时

```
┌── 主板 ────────────────────┐
│ 课题：21.6 菱形（第1课时）      │
│                                │
│ 一、菱形定义                    │
│   有一组邻边相等的平行四边形     │
│   叫作菱形                      │
│                                │
│ 关键词：平行四边形+一组邻边相等  │
└────────────────────────────────┘
（副板此时为空）
```

---

### 环节二（折叠发现）结束时

```
┌── 主板 ────────────────────┐  ┌── 副板 ──────────┐
│ 课题：21.6 菱形（第1课时）      │  │ 二、对称性          │
│                                │  │   轴对称：2条对称轴  │
│ 一、菱形定义                    │  │   对称轴=对角线     │
│   有一组邻边相等的平行四边形     │  │   中心对称 ✓        │
│   叫作菱形                      │  │                     │
│                                │  │ 三、折叠发现→性质    │
│                                │  │ ① 四边相等         │
│                                │  │ ② 对角线互相垂直   │
│                                │  │ ③ 对角线平分对角   │
└────────────────────────────────┘  └────────────────────┘
```

> 副板写"对称性"和"折叠发现"的结论。折叠操作不用写——教师用纸片演示。

---

### 环节三（证明）结束时

```
┌── 主板 ────────────────────┐  ┌── 副板 ──────────┐
│ 课题：21.6 菱形（第1课时）      │  │ 三、逻辑证明         │
│                                │  │ 已知：菱形ABCD      │
│ 一、菱形定义                    │  │ 求证：①四边相等     │
│   有一组邻边相等的平行四边形     │  │      ② AC⊥BD      │
│   叫作菱形                      │  │                     │
│                                │  │ 证② ∵ 等腰△AOD=COD │
│ 二、菱形性质定理                │  │ → ∠AOD=∠COD       │
│   ① 四条边都相等               │  │ 又 ∠AOD+∠COD=180° │
│   ② 对角线互相垂直             │  │ → ∠AOD=90° → AC⊥BD│
│     且每条对角线平分一组对角    │  │                 SSS │
│                                │  │                     │
│                                │  │ 穿插：∠BAC=50°     │
│                                │  │ → ∠BAD=100°       │
└────────────────────────────────┘  └────────────────────┘
```

> 擦掉副板上"折叠发现"的草稿结论，写正式证明过程（关键步骤即可，不写全文）。性质定理移到主板——这是核心保留内容。

---

### 环节四（例题）结束时

```
┌── 主板 ──────────────┐  ┌── 副板 ──────────────────────┐
│ ...      │ 擦掉证明，写：                                    │
│          │                                                     │
│          │ 四、例1 菱形ABCD，周长16cm，∠ABC=120°            │
│          │   → 边长=4cm                                      │
│          │   → BD平分∠ABC → ∠ABD=60°                      │
│          │   → AB=AD → △ABD等边 → BD=4cm                   │
│          │   → OB=2 → OA=2√3 → AC=4√3                     │
│          │                                                     │
│          │ 解题路径：边相等→等边△→勾股定理                   │
└────────────────────────┘  └─────────────────────────────────────┘
```

---

### 环节五-六（练习+拓展）时

副板擦掉例题过程，学生板演或教师写关键算式。不需要详细板书——这些题已在课件上展示答案。

---

### 环节七（小结）时

用红粉笔在主板框出三行核心内容——定义和两条性质定理。板书最终格局即环节三末的状态。

---

### 板书要点速记

| 主板（永远不擦） | 副板（分阶段擦写） |
|:---|:---|
| 课题 | 环节二：对称性 + 折叠发现的性质草稿 → 环节三开始时擦掉 |
| 定义 | 环节三：证明关键步骤 + A组第1题 |
| 两条性质定理 | 环节四：例1解题过程 → 环节五前擦掉 |
| — | 环节五-六：学生板演或关键算式 |
| — | 小结时副板清空 |

## 五、设计检查清单

| 检查项目 | 状态 | 检查结果 |
|:---|:---:|:---|
| 有没有一根主线贯穿全课？ | ✅ | "菱形特殊在哪里"——从定义到性质到应用 |
| 每个环节的"入口"是否自然？ | ✅ | 从生活直觉切入，过渡到数学定义，再到折叠发现，再到逻辑证明 |
| 学生是否被问题"卡"住过？ | ✅ | 追问"B和D重合说明了什么？"——空间想象 |
| 数学思想是"贴标签"还是"融在问题中"？ | ✅ | 特殊到一般通过"折叠一个纸片→证明所有菱形"体现；转化思想通过"证垂直→证邻补角相等"体现 |
| 学生活动是否具体可操作？ | ✅ | 观察折叠、思考回答、独立书写证明、口答、闭眼默述 |
| 教师引导是否有层次？ | ✅ | 先演示→再追问→再提炼；不是"教师说学生听" |
| 课堂节奏是否有变化？ | ✅ | 慢→快→慢（定义展开慢、证明细致慢、练习节奏快、小结慢） |
| 预设了学生的困难了吗？ | ✅ | 证明中"B和D重合怎么推出垂直"卡住、A组第3题"不会用中点条件"卡住 |
