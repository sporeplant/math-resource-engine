# 21.7 正方形 第1课时 教学设计

## 一、课前速览

### 教学目标（教学评一体化）

| 目标类型 | 学习经验表述 | 核心素养指向 | 对应评价证据 |
|:---|:---|:---|:---|
| 概念形成 | 经历"从矩形加邻边相等"和"从菱形加直角"两个方向，逐步抽象出正方形定义的过程，能用自己的语言描述"正方形是同时具有平行四边形、矩形、菱形的交集" | 几何直观、抽象能力 | 1. 举2个不同方向的正例+1个反例；2. 独立完成"定义三条件缺一不可"的判别练习；3. 同桌互说"矩形→菱形→正方形的关系" |
| 性质探索 | 在"大家谈谈"活动中，通过对比、归纳，从边、角、对角线、对称性四个维度，系统梳理正方形的性质，并能在图形上用语言表达 | 几何直观、归纳能力 | 1. 小组讨论"大家谈谈"的完整回答；2. 能独立整理出正方形继承自三类图形的性质表格；3. 能在正方形图上标出对角线、对称轴 |
| 性质应用 | 运用正方形性质与全等三角形，独立完成正方形中线段相等、角相等的证明，能说出每一步的依据 | 推理能力 | 1. 能独立写出例1、A组第2题的完整证明；2. 能指出证明中关键全等条件的来源；3. 小组内互相批改并点评 |
| 综合推理 | 综合运用正方形性质（45°角、邻边相等）与等边/等腰三角形性质，完成角度推算，能按逻辑链条表述每一步角度关系 | 推理能力、运算能力 | 1. 例2的完整推理书写；2. 能独立解决练习中的角度计算问题；3. 小组间互相讲解推理步骤 |
| 结构化认知 | 感受"矩形加邻边相等"与"菱形加直角"殊途同归指向正方形，体会"一般到特殊"的研究方法，能画出四类图形关系图 | 数学思想、应用意识 | 1. 课堂小结时能口头表述四类图形的关系；2. 独立完成四边形关系思维导图 |

### 教学重难点

- **教学重点**：正方形的定义与性质；例1、例2的证明思路
- **教学难点**：例2的角度计算（涉及正方形+等边三角形+等腰三角形的多步推理）；做一做的证明方向选择

### 认知路径图

```
我知道正方形长什么样（生活经验），我熟悉矩形和菱形的定义与性质
    ↓ 需要说清楚"什么样"——不能只靠"看起来像"
我给正方形下定义：邻边相等 + 一个角是直角 → 缺一不可
    ↓ 定义只是开始——它继承了什么？比矩形和菱形多了什么？
我梳理正方形的全部性质：继承矩形 + 继承菱形 + 独有的对称性
    ↓ 性质是用来解决问题的——最简单的用法是什么？
我用正方形性质 + SAS 直接证线段相等（例1）
    ↓ 性质 + 其他图形的条件，还能搞定吗？
我综合正方形 + 等边三角形 + 等腰三角形推理求角度（例2）
    ↓ 没有现成的全等，需要自己构造
我在开放图形中寻找证明路径（做一做）
    ↓ 今天学的和之前学的到底是什么关系？
我画出平行四边形→矩形→菱形→正方形的关系图
```

## 二、教学流程

### 环节一：定义形成——"它到底是什么？"（探究·8min）

**教师的起点动作**：教师在黑板上画出一个平行四边形，然后问（口述）："我们已经知道，有一个角是直角的平行四边形是矩形，有一组邻边相等的平行四边形是菱形。现在我问你们——有一组邻边相等的矩形，是什么图形？"

**学生可能的反应**：
- **正常反应**：部分学生能说出"正方形"，但说不清理由
- **典型困难**：学生可能会说"它既是矩形又是菱形"，但没意识到需要两个条件同时满足
- **意外反应**：个别学生可能直接背出定义（预习过），但无法解释为什么"邻边相等"和"一个角是直角"缺一不可

**教师的跟进策略**：
- 教师先在黑板上画一个"邻边相等的矩形"——标出已知矩形条件（四个直角）和新增条件（邻边相等），问（口述）："这个图形是什么？你怎么判断的？"
- 再从另一个方向画一个"有一个角是直角的菱形"——标出菱形条件（四边相等）和新增条件（一个直角），问（口述）："这个又是什么？"
- 追问（口述）："这两个图形是同一个东西吗？你能用一句话说清楚它的定义吗？"

**核心问题**：学生尝试表述定义后，教师追问（口述）："'有一组邻边相等'和'有一个角是直角'这两个条件，能去掉一个吗？"——在黑板上画一个只满足"邻边相等"但不满足"直角"的菱形，再画一个只满足"直角"但不满足"邻边相等"的矩形，让学生判断它们是不是正方形（口述），从而体会定义中两个条件**缺一不可**。

**教师规范定义**："有一组邻边相等且有一个角是直角的平行四边形叫作正方形。"让学生齐读一遍（口述），并标注关键词：**邻边相等**、**一个角是直角**、**平行四边形**。

**过渡语**："刚才我们从'矩形加条件'和'菱形加条件'两个方向都走到了正方形。那正方形到底有哪些性质？它比矩形多什么？比菱形又多什么？"

---

### 环节二：性质探索——"正方形到底有哪些性质？"（突破·10min）

**教师的起点动作**：打开教材"大家谈谈"的三个问题，出示在黑板上（PPT或板书）：

> 问题1：正方形是中心对称图形吗？对称中心在哪里？是轴对称图形吗？有几条对称轴？
> 
> 问题2：谈谈正方形与平行四边形、矩形和菱形的关系。
> 
> 问题3：正方形有哪些性质？

教师说（口述）："这三个问题，我不直接给答案。四人一组讨论，每组拿出一张纸，从**边、角、对角线、对称性**四个维度整理出正方形的性质。要求：说清楚哪些性质是从矩形继承来的，哪些是从菱形继承来的，哪些是正方形独有的。"

**学生活动**：
- **(小组讨论，口述)**：学生分组讨论，在白纸上整理性质
- 教师在教室内巡视，倾听典型讨论，但不急于纠正

**学生可能的反应**：
- **正常反应**：大部分组能从"继承矩形"写出"四个角都是直角、对角线相等"，从"继承菱形"写出"四条边相等、对角线互相垂直、对角线平分对角"，从"独有"写出"四条对称轴"
- **典型困难**：部分学生漏掉"对角线平分对角"（这是菱形性质，正方形继承但不直观）；部分学生写不出"中心对称图形"或写错对称轴数量
- **混乱点**："对角线相等"和"对角线互相垂直"写反——矩形对角线相等，菱形对角线垂直，部分学生记混

**教师的跟进策略**：

找一组展示整理结果（口述），其他组补充。教师逐条确认，板书形成**性质清单**：

```
正方形性质（平行四边形+矩形+菱形）：
  边：四条边都相等（继承菱形）
  角：四个角都是直角（继承矩形）
  对角线：相等（继承矩形）、互相垂直（继承菱形）、互相平分（继承平行四边形）、平分每一组对角（继承菱形）
  对称性：中心对称图形（对称中心：对角线交点）
          轴对称图形（4条对称轴：两条对角线 + 两组对边中点连线）
```

追问（口述）："刚才有同学说正方形有2条对称轴，有同学说4条。到底几条？谁来验证？"——请学生上台，在黑板上的正方形图上画出所有对称轴（动笔），边说边画。其他学生在练习本上画（动笔）。

**核心问题**：在学生完成性质梳理后，教师指着性质清单问（口述）："现在回头看——'正方形是特殊的矩形'和'正方形是特殊的菱形'，这句话对吗？从性质上怎么验证？"——引导学生发现：正方形满足矩形的全部性质，也满足菱形的全部性质，因此它**同时是**特殊的矩形和特殊的菱形。

**过渡语**："梳理清楚了性质，我们来看看这些性质怎么用。最简单的用法——给你一个正方形和对角线，你能证一对线段相等吗？"

---

### 环节三：性质直接用——例1（示范·5min）

**教师的起点动作**：出示例1图形（正方形ABCD，E在对角线AC上），问（口述）："条件只有'ABCD是正方形，E在AC上'，你来说服我BE=DE——不能靠眼睛看，用逻辑说话。"

**核心问题**："要证BE=DE，你打算证哪两个三角形全等？全等的三个条件分别从哪里来？"

**学生活动**：
- **(动笔)**：学生尝试独立写证明，限时2分钟
- **(口述)**：请一位学生口述证明思路

**学生可能的反应**：
- **正常反应**：大部分能识别△AEB≅△AED，条件：AB=AD（正方形邻边相等）、∠BAC=∠DAC（对角线平分对角）、AE=AE（公共边）
- **典型困难**：部分学生找不到∠BAC=∠DAC，因为"对角线平分对角"是菱形性质，学生不习惯从正方形中提取菱形性质来用
- **错误反应**：个别学生想用△BEC≅△DEC或△ABE≅△CBE，方向偏离

**教师的跟进策略**：
- 对于找不到对角相等的学生，引导（口述）："正方形对角线有什么性质？回顾一下刚才的性质清单——正方形从菱形继承了哪条性质？"
- 对于方向偏离的学生，反问（口述）："你看△BEC和△DEC，它们有几个边能确定相等？怎么证明？"——让学生自己发现这条路走不通或更复杂

**教师板书标准证明（示范书写规范）**（动笔）：

```
∵ 四边形ABCD是正方形，
∴ AB = AD，∠BAC = ∠DAC。
又∵ AE = AE，
∴ △AEB ≅ △AED（SAS）。
∴ BE = DE。
```

追问（口述）："全等的依据是什么？"——学生回答"SAS"后，教师标注每一步的条件来源。

**练习穿插**：变式——教师问（口述）："如果E是AC延长线上一点，BE还等于DE吗？为什么？"（口答，快速过）

**过渡语**："刚才正方形里只有它自己，条件单纯。如果正方形里再放一个等边三角形呢？难度就上来了。"

### 环节四：综合推理——例2（突破·10min）

**教师的起点动作**：出示例2图形（正方形ABCD，△BCE是等边三角形），标注已知条件。问（口述）："现在正方形内部有一个等边三角形，要证∠EAD=∠EDA=15°。15°不是特殊角，它很可能来自某个已知角的差或和。我们先来找找能确定的角。"

**核心问题（分段推进）**：

**第1问（口述）**："等边三角形给了我们什么条件？"——学生答：BC=BE=CE，∠EBC=∠ECB=∠BEC=60°。

**第2问（口述）**："BC同时是正方形的边，这告诉我们什么？"——学生发现：AB=BC=BE，所以△ABE是等腰三角形。

**第3问（口述）**："∠ABE是多少度？怎么算的？"——学生答：∠ABE=∠ABC−∠EBC=90°−60°=30°。

**第4问（口述）**："已知等腰△ABE的顶角∠ABE=30°，底角是多少？"——学生答：∠BAE=∠BEA=½×(180°−30°)=75°。

**第5问（口述）**："∠EAD是哪个角和哪个角的差？"——学生发现：∠EAD=∠BAD−∠BAE=90°−75°=15°。

**学生活动**：
- **(动笔)**：学生跟随教师分步书写证明过程，每一步写出角度计算的依据
- **(默述)**：在写完一半时，教师说"闭眼，把前四步在脑子里过一遍"——让学生形成推理链条的完整印象

**学生可能的反应**：
- **正常反应**：跟着教师的分步引导，大部分能跟上
- **困难①**：在"AB=BE"这一步卡住——不理解为什么BC既是正方形的边又是等边三角形的边可以传递
  - 矫正：教师在图形上用彩色粉笔标出AB、BC、BE三条线段，直接指给学生看（口述）："AB=BC（正方形），BC=BE（等边三角形），所以AB=BE，这叫什么？"——学生答"等量代换"
- **困难②**：在"∠ABE=30°"这一步卡住——看不出是∠ABE=∠ABC−∠EBC
  - 矫正：引导（口述）："∠ABE包含在哪一个大角里？这个大角减去哪个角就是∠ABE？"——在图上标注出∠ABC和∠EBC

**教师板书完整过程（示范多步推理书写）**（动笔）：

```
证明：
∵ 四边形ABCD是正方形，
∴ AB = BC，∠ABC = ∠BAD = 90°。
∵ △BCE是等边三角形，
∴ BC = BE，∠EBC = 60°。
∴ AB = BE，∠ABE = 90° − 60° = 30°。
∴ ∠BAE = ∠BEA = ½×(180° − 30°) = 75°。
∴ ∠EAD = 90° − 75° = 15°。
同理 ∠EDA = 15°。
∴ ∠EAD = ∠EDA = 15°。
```

**追问拓展（口述）**："证完∠EAD=∠EDA=15°后，你还能求出哪些角的度数？"——学生自由回答（口述），可能发现∠CED、∠DCE等，不要求全部求对，重在打开思路。

**练习穿插**：教材"练习"——正方形ABCD的对角线AC为菱形AEFC的一边，求∠FAB的度数。（动笔·3min）这道题将正方形与菱形叠加，是例1例2的综合变式。学生独立完成后同桌互批。

**过渡语**："例1和例2都是直接告诉了我们图形关系，下面这道题要你自己在图形中寻找证明的方向——条件更开放了。"

### 环节五：开放探究——做一做（应用·7min）

**教师的起点动作**：出示做一做的图形和条件（正方形ABCD，点E、F、G分别在AB、AD、CD上，EF⊥FG，AF=DG），让学生先**独立思考2分钟**（动笔），找找证明思路。

**核心问题**："要证EF=FG，你想证哪两个三角形全等？你的全等条件够吗？"

**学生可能的反应**：
- **良好反应**：部分学生能识别目标—证△AEF≅△DGF，找到两个条件（AF=DG，∠EAF=∠FDG=90°），但第三个条件卡住
- **中等反应**：学生尝试连接EG或作垂线，方向不明确
- **优秀反应**：少数学生发现"一线三垂直"模型，利用A、F、D共线的角度关系，推出ASA全等

**关键追问（"一线三垂直"引导）**：

**第1问**："A、F、D这三个点有什么关系？"——学生发现：F在AD上，所以A、F、D三点共线。

**第2问（核心）**：教师在图上标出∠AFE和∠GFD，追问（口述）："∠AFE、∠EFG、∠GFD这三个角在一条直线上，它们有什么关系？"

引导学生发现：∠AFE + ∠EFG + ∠GFD = 180°（平角），已知∠EFG=90°，所以∠AFE + ∠GFD = 90°。

**第3问**："在Rt△DGF中，∠DGF和∠GFD又有什么关系？"——学生答：∠DGF = 90° − ∠GFD。

**第4问（归纳）**："比较∠AFE和∠DGF，你发现了什么？"——∠AFE = 90° − ∠GFD = ∠DGF。

**完整证明引导**：

```
在△AEF和△DGF中：
  ∠EAF = ∠FDG = 90°（正方形角）
  AF = DG（已知）
  ∠AFE = ∠DGF（由A、F、D共线 + 三角形内角和推得）
∴ △AEF ≅ △DGF（ASA）
∴ EF = FG
```

**教师小结**（口述）："这题用的是'一线三垂直'模型——A、F、D三点共线，产生了三个直角：∠EAF=90°、∠EFG=90°、∠FDG=90°。利用平角和三角形内角和，找到角相等关系，从而用ASA证全等。"

**过渡语**："做一做让我们体会到'构造'的重要性。现在我们来检测一下今天学得怎么样。"

### 环节六：当堂检测与小结（收束·5min）

**当堂检测（动笔·3min）**：

选2道题，学生独立完成：

1. **基础题**（练习册选择第5题）：正方形ABCD的对角线AC、BD交于点O，结论正确的个数：①AB=BC=CD=DA；②AO=BO=CO=DO；③AC⊥BD。——快速判断，口答

2. **提高题**（A组第2题）：两个正方形共顶点B，求证AE=CG。——动笔写完整证明

**课堂小结（口述·2min）**：

教师用三个问题串起全课：

> 问题1："今天学的正方形，定义是什么？两个条件缺一不可——哪两个？"

学生齐答或指名回答。

> 问题2："正方形的性质从哪儿来？从平行四边形继承了什么？从矩形继承了什么？从菱形继承了什么？"

请一位学生完整回答，其他学生补充。

> 问题3："你能画一张图说清楚平行四边形、矩形、菱形、正方形的关系吗？"

**(动笔)**：学生在笔记本上画四边形关系图（集合图或箭头图）。

## 五、课后反思点

1. **定义形成环节**：学生是否能从"矩形加邻边相等"和"菱形加直角"两个方向自然得出正方形的定义？还是需要教师强推？如果学生卡在"殊途同归"的理解上，明天是否需要补一个类比练习？

2. **例2的推理链长度**：5步角度计算，学生在哪一步最容易断？今天课上反映出来的主要卡点在"AB=BE"的等量代换还是"∠ABE=∠ABC−∠EBC"的角度差识别？明天第2课时的判定教学中是否有类似的长推理链需要提前设计支架？

3. **做一做的方向预警**：有多少学生掉进了"Rt△AEF≅Rt△DGF"的陷阱？这个错误能否在后续教学中用来强化"全等三角形对应关系"的意识？

4. **教学评一致性检查**：对照课前速览中的5条目标，哪些目标的评价证据在今天的课堂上确实收集到了？如果某条目标缺乏证据，是设计问题还是时间问题？
