# 21.7 正方形（第2课时）

---

## 一、知识点引入

根据正方形与平行四边形、矩形、菱形的关系，说一说如何判定一个四边形是正方形。

可以先判定这个四边形是矩形，再证明这个矩形是菱形；也可以先判定这个四边形是菱形，再证明这个菱形是矩形。因此，判定一个四边形是正方形，只要判定这个四边形既是矩形又是菱形即可。

---

## 二、例题

### 例3

**已知**：如图21.7-4，分别延长正方形ABCD的边AB，BC，CD，DA到点E，F，G，H，使BE = CF = DG = AH，连接EF，FG，GH，HE，得到四边形EFGH。

**求证**：四边形EFGH是正方形。

![图21.7-4](images/450a373887a96d6d8c4fdb8a91b8b9c8b5279b351d909064456c3305b7931e04.jpg)

**证明**：
∵ 四边形ABCD是正方形，
∴ AB = BC = CD = DA，
∠EBF = ∠FCG = ∠GDH = ∠HAE = 90°。
又∵ BE = CF = DG = AH，
∴ AE = BF = CG = DH，
∴ △EBF ≌ △FCG ≌ △GDH ≌ △HAE，
∴ EF = FG = GH = HE，∠EFB = ∠HEA。
∴ 四边形EFGH是菱形。
又∵ ∠EFB + ∠FEB = 90°，
∴ ∠FEB + ∠HEA = 90°，即∠FEH = 90°。
∴ 菱形EFGH是正方形。

---

### 例4

**已知**：如图21.7-5，在菱形ABCD中，对角线AC，BD相交于点O，点E，F在对角线BD上，且BE = DF，OE = OA。

**求证**：四边形AECF是正方形。

![图21.7-5](images/64ae06f2f765bde050a674570f78b058efa927cb78e097541494e25cecf39256.jpg)

**证明**：
∵ 四边形ABCD是菱形，
∴ AC ⊥ BD，OA = OC，OB = OD。
又∵ BE = DF，
∴ OE = OF，
∴ 四边形AECF是菱形。
又∵ OE = OA，
∴ OE = OF = OA = OC，即EF = AC，
∴ 菱形AECF是正方形。
