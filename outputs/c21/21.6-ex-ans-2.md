## 二、逐题解析

### 序号1 [一起探究]

**原题**：如图 21.6-6，画两条等长的线段 $AB, AD$，分别以点 $B, D$ 为圆心，以 $AB$ 长为半径画弧，两弧相交于点 $C$。连接 $BC, CD$，得到四边形 $ABCD$。四边形 $ABCD$ 是菱形吗？

**解析**：
- **考察知识点**：菱形的判定定理1的发现
- **思路分析**：通过尺规作图，让学生直观感受"四条边相等的四边形是菱形"
- **关键步骤**：
  1. 由作图可知 $AB = AD$（已知）
  2. 由作图可知 $BC = AB$，$CD = AD$（半径相等）
  3. 所以 $AB = BC = CD = DA$
  4. 四边形 $ABCD$ 是菱形（四条边相等的四边形是菱形）
- **探究结论**：四条边相等的四边形是菱形

**答案**：是菱形。

---

### 序号2 [判定定理1证明]

**原题**：证明：四条边相等的四边形是菱形。

**已知**：如图21.6-7，在四边形 $ABCD$ 中，$AB = BC = CD = DA$

**求证**：四边形 $ABCD$ 是菱形

**解析**：
- **考察知识点**：菱形判定定理1的证明
- **思路分析**：要证菱形，需先证平行四边形，再证邻边相等
- **关键步骤**：利用"两组对边分别相等的四边形是平行四边形"

**证明**：

$\because AB = CD$，且 $BC = DA$，

$\therefore$ 四边形 $ABCD$ 是平行四边形（两组对边分别相等的四边形是平行四边形）

又 $\because AB = DA$，

$\therefore$ 四边形 $ABCD$ 是菱形（一组邻边相等的平行四边形是菱形）

**核心思路**：四边相等 → 对边相等 → 平行四边形 + 邻边相等 → 菱形

### 序号3 [大家谈谈]

**原题**：如图21.6-8，$\square ABCD$ 的两条对角线 $AC$，$BD$ 互相垂直，$O$ 是这两条对角线的交点。

(1) 你能说明图中的 $\text{Rt}\triangle ABO$，$\text{Rt}\triangle CBO$，$\text{Rt}\triangle CDO$，$\text{Rt}\triangle ADO$ 都是全等的吗？

(2) $\square ABCD$ 的四条边都相等吗？

(3) 请证明你的猜想。

**解析**：
- **考察知识点**：菱形判定定理2的发现
- **思路分析**：通过全等三角形证明四边相等
- **关键步骤**：
  1. 平行四边形对角线互相平分 → $OA=OC$，$OB=OD$
  2. 对角线垂直 → 四个直角三角形
  3. SAS全等 → 四边相等 → 菱形

**解答**：

**(1) 全等证明**：

在 $\triangle ABO$ 和 $\triangle CBO$ 中：
- $OA = OC$（平行四边形对角线互相平分）
- $\angle AOB = \angle COB = 90°$（对角线垂直）
- $OB = OB$（公共边）

$\therefore \triangle ABO \cong \triangle CBO$（SAS）

同理可证 $\triangle CBO \cong \triangle CDO \cong \triangle ADO$

**(2) 结论**：四条边都相等。

**(3) 证明**：

由 (1) 知 $\triangle ABO \cong \triangle ADO$

$\therefore AB = AD$

$\therefore$ 四边形 $ABCD$ 是菱形（一组邻边相等的平行四边形是菱形）

**核心思路**：平行四边形 + 对角线垂直 → 全等三角形 → 邻边相等 → 菱形

### 序号4 [判定定理2证明]

**原题**：证明：两条对角线互相垂直的平行四边形是菱形。

**已知**：$\square ABCD$ 中，$AC \perp BD$，$O$ 为交点

**求证**：四边形 $ABCD$ 是菱形

**多角度证明**：

**角度一：SAS全等证邻边相等**
> **证明**：
> $\because$ 四边形 $ABCD$ 是平行四边形
> $\therefore OA = OC$，$OB = OD$
> 又 $\because AC \perp BD$
> $\therefore \angle AOB = \angle AOD = 90°$
> 在 $\triangle AOB$ 和 $\triangle AOD$ 中：
> - $OA = OA$
> - $\angle AOB = \angle AOD$
> - $OB = OD$
> $\therefore \triangle AOB \cong \triangle AOD$（SAS）
> $\therefore AB = AD$
> $\therefore$ 四边形 $ABCD$ 是菱形
> 
> **核心思路**：利用对角线垂直平分，证邻边相等

**角度二：垂直平分线性质**
> **证明**：
> $\because$ 四边形 $ABCD$ 是平行四边形
> $\therefore OA = OC$，$OB = OD$
> $\therefore$ $AC$ 垂直平分 $BD$，$BD$ 垂直平分 $AC$
> $\therefore AB = AD$，$AB = CB$（垂直平分线上的点到两端距离相等）
> $\therefore AB = BC = CD = DA$
> $\therefore$ 四边形 $ABCD$ 是菱形
> 
> **核心思路**：利用垂直平分线性质直接得四边相等

**方法对比**：

| 角度 | 优点 | 适用场景 |
|:---:|:---|:---|
| 角度一 | 步骤清晰，易理解 | 基础巩固 |
| 角度二 | 更简洁，一步到位 | 拓展提升 |

### 序号5 [例2]

**原题**：已知：如图 21.6-9，在 $\triangle ABC$ 中，$AD$ 是 $\angle BAC$ 的平分线，$DE \parallel AC$，交 $AB$ 于点 $E$，$DF \parallel AB$，交 $AC$ 于点 $F$。求证：四边形 $AEDF$ 是菱形。

**解析**：
- **考察知识点**：菱形判定定理的综合应用
- **思路分析**：先证平行四边形，再证邻边相等
- **关键技巧**：角平分线 + 平行线 → 等腰三角形

**证明**：

$\because DE \parallel AC$，$DF \parallel AB$

$\therefore$ 四边形 $AEDF$ 是平行四边形（两组对边分别平行）

$\because DE \parallel AC$

$\therefore \angle EDA = \angle CAD$（内错角相等）

又 $\because AD$ 平分 $\angle BAC$

$\therefore \angle EAD = \angle CAD$

$\therefore \angle EAD = \angle EDA$

$\therefore AE = DE$（等角对等边）

$\therefore$ 四边形 $AEDF$ 是菱形（一组邻边相等的平行四边形是菱形）

**核心思路**：平行四边形 + 角平分线+平行线→等腰三角形→邻边相等→菱形

### 序号6 [练习1]

**原题**：如图，$AB = AC$，画出点 $A$ 关于 $BC$ 的对称点 $A'$，请用两种不同的方法证明四边形 $ABA'C$ 是菱形。

**解析**：
- **考察知识点**：菱形判定定理的灵活应用
- **思路分析**：根据对称性质，选择不同判定方法

**多角度证明**：

**方法一：四条边相等**
> **证明**：
> $\because A'$ 是 $A$ 关于 $BC$ 的对称点
> $\therefore A'B = AB$，$A'C = AC$
> 又 $\because AB = AC$
> $\therefore AB = AC = A'B = A'C$
> $\therefore$ 四边形 $ABA'C$ 是菱形（四条边相等的四边形是菱形）
> 
> **核心思路**：利用对称性质直接得四边相等

**方法二：对角线互相垂直的平行四边形**
> **证明**：
> $\because A'$ 是 $A$ 关于 $BC$ 的对称点
> $\therefore AA' \perp BC$，且 $BC$ 平分 $AA'$
> 设 $AA'$ 与 $BC$ 交于点 $O$
> 则 $OA = OA'$，$OB = OC$（等腰三角形三线合一）
> $\therefore$ 四边形 $ABA'C$ 是平行四边形（对角线互相平分）
> 又 $\because AA' \perp BC$
> $\therefore$ 四边形 $ABA'C$ 是菱形（对角线互相垂直的平行四边形是菱形）
> 
> **核心思路**：对称 → 对角线垂直平分 → 平行四边形 + 垂直 → 菱形

**方法对比**：

| 方法 | 优点 | 适用场景 |
|:---:|:---|:---|
| 方法一 | 更简洁，步骤少 | 推荐首选 |
| 方法二 | 巩固判定定理2 | 对比理解 |

### 序号7 [练习2]

**原题**：如图，在 $\square ABCD$ 中，$\angle D = 60°$，以顶点 $A$ 为圆心、$AB$ 长为半径画弧，交 $BC$ 于点 $E$，交 $AD$ 于点 $F$，连接 $AE$，$EF$。请指出图中的等腰三角形、平行四边形和菱形。

**解析**：
- **考察知识点**：识图能力，等腰三角形、平行四边形、菱形的判定
- **思路分析**：根据作图条件和已知条件，逐一判断

**解答**：

**等腰三角形**：
- $\triangle ABE$：$AB = AE$（半径相等）
- $\triangle AEF$：$AE = AF$（半径相等）
- $\triangle ABF$：$AB = AF$（半径相等）

**平行四边形**：
- $\square ABCD$（已知）
- $\square ABEF$：$AB \parallel EF$（$\angle D = 60°$，$\triangle AEF$是等边三角形）

**菱形**：
- 四边形 $ABEF$ 是菱形
  - 理由：$AB = BE = EF = FA$（四条边相等）
  - 或由 $AB = AF$，$\angle D = 60°$ 推出等边三角形，进而四边相等

**关键步骤说明**：
- 由 $\angle D = 60°$，$\square ABCD$ 得 $\angle B = 120°$
- $\triangle ABE$ 中，$AB = AE$，$\angle B = 120°$，则 $\angle BAE = 30°$
- 同理可证其他角的关系，最终得出四边相等

### 序号8 [A组1]

**原题**：已知：如图，$E$ 是菱形 $ABCD$ 的边 $AD$ 的中点，$EF \perp BD$ 于点 $H$，交 $BC$ 的延长线于点 $F$，交 $DC$ 于点 $G$。求证：$DC$ 与 $EF$ 互相平分。

**解析**：
- **考察知识点**：菱形性质、平行四边形判定、全等三角形
- **思路分析**：证 $AEFC$ 为平行四边形得 $AE = CF$，又 $AE = DE$ → $CF = DE$；再证 $\triangle CFG \cong \triangle DEG$（AAS）→ $DG = CG$ 且 $EG = FG$ → 互相平分

**证明**：

连接 $AC$。

$\because$ 四边形 $ABCD$ 是菱形

$\therefore AD \parallel BC$，$AC \perp BD$

$\because EF \perp BD$

$\therefore EF \parallel AC$（垂直于同一直线的两直线平行）

又 $\because AE \parallel FC$（$AD \parallel BC$）

$\therefore$ 四边形 $AEFC$ 是平行四边形（两组对边分别平行）

$\therefore AE = CF$

$\because E$ 是 $AD$ 的中点

$\therefore AE = DE$

$\therefore CF = DE$

在 $\triangle CFG$ 和 $\triangle DEG$ 中：

$\because EF \parallel AC$，$AD \parallel BC$

$\therefore \angle DEG = \angle DAC = \angle BCA = \angle CFG$

又 $\because \angle DGE = \angle CGF$（对顶角相等）

$CF = DE$（已证）

$\therefore \triangle DEG \cong \triangle CFG$（AAS）

$\therefore DG = CG$，$EG = FG$

$\therefore DC$ 与 $EF$ 互相平分

**核心思路**：平行四边形 → $CF = DE$ → $\triangle CFG \cong \triangle DEG$（AAS）→ 直接得 $DG = CG$ 且 $EG = FG$

### 序号9 [A组2]

**原题**：已知：如图，在 $\square ABCD$ 中，$O$ 为对角线 $AC$ 的中点，过点 $O$ 作 $AC$ 的垂线与边 $AD$，$BC$ 分别交于点 $E$，$F$，连接 $AF$，$CE$。求证：四边形 $AFCE$ 是菱形。

**解析**：
- **考察知识点**：菱形判定定理2的直接应用
- **思路分析**：证平行四边形 + 对角线垂直

**证明**：

$\because$ 四边形 $ABCD$ 是平行四边形

$\therefore AD \parallel BC$

$\therefore \angle EAO = \angle FCO$（内错角相等）

在 $\triangle AOE$ 和 $\triangle COF$ 中：
- $\angle EAO = \angle FCO$
- $OA = OC$（$O$ 是 $AC$ 中点）
- $\angle AOE = \angle COF$（对顶角相等）

$\therefore \triangle AOE \cong \triangle COF$（ASA）

$\therefore OE = OF$

$\therefore$ 四边形 $AFCE$ 是平行四边形（对角线互相平分）

又 $\because EF \perp AC$

$\therefore$ 四边形 $AFCE$ 是菱形（对角线互相垂直的平行四边形是菱形）

**核心思路**：证平行四边形（对角线互相平分）+ 对角线垂直 → 菱形

### 序号10 [A组3]

**原题**：已知：如图，四边形 $ABCD$ 是菱形，两条对角线相交于点 $O$，$DE$ 为 $\angle ADB$ 的平分线，交 $AC$ 于点 $E$，$DF$ 为 $\angle CDB$ 的平分线，交 $AC$ 于点 $F$，连接 $BE$，$BF$。求证：四边形 $DEBF$ 是菱形。

**解析**：
- **考察知识点**：菱形性质、角平分线、全等三角形、菱形判定
- **思路分析**：证四边相等或对角线互相垂直平分

**证明**：

$\because$ 四边形 $ABCD$ 是菱形

$\therefore AC \perp BD$，$OA = OC$，$OB = OD$

$\angle ADB = \angle CDB$（菱形对角线平分对角）

$\because DE$ 平分 $\angle ADB$，$DF$ 平分 $\angle CDB$

$\therefore \angle ADE = \angle CDF = \frac{1}{2}\angle ADB$

在 $\triangle ADE$ 和 $\triangle CDF$ 中：
- $AD = CD$（菱形四边相等）
- $\angle DAE = \angle DCF$（$AD = CD$，$\triangle ADC$ 是等腰三角形，底角相等）
- $\angle ADE = \angle CDF$

$\therefore \triangle ADE \cong \triangle CDF$（ASA）

$\therefore AE = CF$

$\therefore OE = OF$

$\therefore$ 四边形 $DEBF$ 是平行四边形（对角线互相平分）

又 $\because EF \perp BD$

$\therefore$ 四边形 $DEBF$ 是菱形（对角线互相垂直的平行四边形是菱形）

**核心思路**：利用菱形性质 + 角平分线 + 全等，证对角线互相垂直平分

### 序号11 [B组4]

**原题**：如图，在菱形 $ABCD$ 中，$\angle BAD = 60°$，$M$ 为 $AB$ 的中点，$P$ 为对角线 $AC$ 上的一个动点，$PM + PB$ 的最小值是 3。求 $AB$ 的长。

**解析**：
- **考察知识点**：菱形性质、轴对称、最短路径（将军饮马模型）
- **思路分析**：利用菱形对称性，将 $PM + PB$ 转化为两点间距离

**解答**：

$\because$ 四边形 $ABCD$ 是菱形

$\therefore AC$ 是对称轴，$B$ 关于 $AC$ 的对称点是 $D$

$\therefore PB = PD$

$\therefore PM + PB = PM + PD$

当 $P$ 在 $MD$ 与 $AC$ 的交点时，$PM + PD = MD$ 最小

$\therefore MD = 3$

$\because \angle BAD = 60°$，$AB = AD$

$\therefore \triangle ABD$ 是等边三角形

$\because M$ 是 $AB$ 的中点

$\therefore DM \perp AB$（等边三角形三线合一）

在 $\text{Rt}\triangle AMD$ 中：

$\angle DAM = 60°$，$\angle AMD = 90°$

$\therefore \angle ADM = 30°$（直角三角形两锐角互余）

在含 $30°$ 角的直角三角形中，$30°$ 角所对的直角边等于斜边的一半：

$\therefore AM = \frac{1}{2}AD$

又由勾股定理：$AM^2 + MD^2 = AD^2$

代入 $MD = 3$：$AM^2 + 9 = (2AM)^2 = 4AM^2$

$\therefore 3AM^2 = 9$，$AM^2 = 3$

$\therefore AM = \sqrt{3}$

$\therefore AB = 2AM = 2\sqrt{3}$

**核心思路**：轴对称转化 + 将军饮马模型 + 等边三角形性质

### 序号12 [B组5]

**原题**：如图，绿丝带下部重叠部分是什么图形？请说明理由。

**解析**：
- **考察知识点**：菱形判定的生活应用
- **思路分析**：观察绿丝带的结构，判断重叠部分的形状

**解答**：

绿丝带下部重叠部分是**菱形**。

**理由**：

绿丝带的两条边缘是两条平行线，它们之间的**垂直距离**就是丝带的宽度——这个距离是恒定的，不随你从哪个角度去量。两条丝带交叉重叠，重叠区域是一个平行四边形。

这个平行四边形的一组对边分别落在第一条丝带的两条平行边缘上——这两条边缘之间的**垂直距离**就是第一条丝带的宽度。

另一组对边分别落在第二条丝带的两条平行边缘上——垂直距离就是第二条丝带的宽度。

两条丝带宽度相等，所以这个平行四边形的两组对边分别相等，四条边全相等，是菱形。

**核心思路**：丝带宽度 = 两条平行边缘之间的垂直距离 → 重叠平行四边形两组对边分别等于两条丝带的宽度 → 四边相等 → 菱形
