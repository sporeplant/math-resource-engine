# 第05讲-一次函数2_复习讲义 答案解析

## 一、知识点填空答案

### 知识点01 一次函数图像的平移

1. 左右平移时，平移规律是在**自变量**上加减平移单位，左加右减。

若 $y=kx+b$ 向左平移 $a$ 个单位长度，则平移后解析式为 $y=k(x+a)+b$。

若 $y=kx+b$ 向右平移 $a$ 个单位长度，则平移后解析式为 $y=k(x-a)+b$。

2. 上下平移时，平移规律是在**函数解析式**上加减平移单位，上加下减。

若 $y=kx+b$ 向上平移 $a$ 个单位长度，则平移后解析式为 $y=kx+b+a$。

若 $y=kx+b$ 向下平移 $a$ 个单位长度，则平移后解析式为 $y=kx+b-a$。

### 知识点04 一次函数与一元一次方程

求 $y=k_1x+b_1$ 与 $y=k_2x+b_2$ 的交点坐标，只需建立方程 $k_1x+b_1=k_2x+b_2$，先求交点的横坐标，再代入任意一个函数求纵坐标。

若 $y=kx+b$ 经过点 $(m,n)$，则方程 $kx+b=n$ 的解为 $x=m$。

若两个一次函数图像的交点为 $(m,n)$，则方程 $k_1x+b_1=k_2x+b_2$ 的解为 $x=m$。

### 知识点05 一次函数与方程组

若两个一次函数图像的交点为 $(m,n)$，则方程组

```math
\begin{cases}
k_1x+b_1-y=0,\\
k_2x+b_2-y=0
\end{cases}
```

的解为

```math
\begin{cases}
x=m,\\
y=n
\end{cases}
```

解析：当 $0\le x\le400$ 时，图像过 $(0,200)$、$(200,180)$，所以 $k=\frac{180-200}{200-0}=-\frac{1}{10}$，得 $y=-\frac{1}{10}x+200$。当 $x>400$ 时，图像为水平线，$y=160$。设甲种面积为 $x$，乙种面积为 $1000-x$。在 $0\le x\le400$ 时，$x\left(-\frac{1}{10}x+200\right)+100(1000-x)=121000$，解得 $x=300$ 或 $700$，取符合区间的 $x=300$。在 $x>400$ 时，方程 $160x+100(1000-x)=121000$ 得 $x=350$，不符合 $x>400$。所以甲 $300m^2$，乙 $700m^2$。
