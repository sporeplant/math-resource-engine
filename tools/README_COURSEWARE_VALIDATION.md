# 课件教材一致性验证工具

## 概述

本工具用于验证生成的课件是否与教材原文内容一致，特别检查：

1. **图片引用完整性** - 确保教材中的图片在课件中都有对应
2. **内容完整性** - 检查关键图表（如条形图、扇形图等）是否都已包含
3. **上下文准确性** - 避免图片被错误地引用到不相关的内容区域

## 使用方法

### 方法一：直接运行验证工具

```
python tools/validate_courseware_consistency.py <课件文件> --textbook <教材原文文件>
```

示例：
```
python tools/validate_courseware_consistency.py "输出/22.1第一课时_课件.md" --textbook "知识库/教材原文/教材原文_22.1_统计的初步认识.md"
```

### 方法二：集成到主验证流程

该工具现在也已经集成到 `validate_output.py` 中，当验证课件时会自动进行一致性检查：

```
python tools/validate_output.py <课件文件>
```

## 检测功能

### 已实现的检测项目

| 检测项目 | 说明 |
|---------|------|
| 图片缺失检测 | 检查课件是否包含教材中的所有关键图片 |
| 扇形统计图检测 | 统计类内容确保同时包含条形图和扇形图 |
| 章节关键词检测 | 确保关键教学内容（如"统计的一般过程"）被包含 |
| 智能上下文识别 | 识别特定图片被正确内容替代的情况（如章首插图替换为真实数据） |

### 检测结果说明

- **✅ Validation PASSED** - 课件与教材一致
- **⚠️ Validation passed with warnings** - 有需要注意的变化（可能是合理的改进）
- **❌ Validation FAILED** - 发现严重不一致

## 问题修复案例

### 问题：课件图片引用错误

**原始问题（22.1第一课时）：**
- 教材中的调查数据表格在课件中被错误地引用为章首插图
- 缺少教材中配套的扇形统计图
- 统计表中的占位值具有误导性

**修复内容：**
1. 将错误引用的章首插图替换为真实的调查数据表格
2. 补充了缺失的扇形统计图，与条形统计图并排显示
3. 修正了统计表中的占位值

**验证结果对比：**
- 修复前：验证失败，提示缺少扇形统计图
- 修复后：验证通过（有提示说明章首插图被合理替换）

## 注意事项

1. 章首插图（如 `a4a09b2c...jpg`）如果被真实教学内容替代是合理的修改，工具会识别这种情况
2. 统计类课程必须同时包含条形统计图和扇形统计图
3. 验证工具通过文件名匹配图片，不检查图片内容本身
