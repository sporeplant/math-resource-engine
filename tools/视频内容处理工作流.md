# 视频内容处理工作流

## 概述

本工作流用于将教学视频与教学设计对接，实现视频内容的智能识别、片段推荐和自动分割。

## 工作流步骤

### 步骤1：生成口播稿 📝
**目标**: 将视频中的音频转换为文字稿，并附带时间戳

**输入**: 视频文件（MP4格式）
**输出**: 带时间戳的口播稿（Markdown格式）

**工具**: `video_transcribe.py`

**使用方法**:
```bash
python tools/video_transcribe.py video/用统计图描述数据.mp4
```

**输出文件**:
- `video/transcripts/[视频名].wav` - 提取的音频文件
- `video/transcripts/[视频名].vtt` - 原始字幕文件
- `video/transcripts/[视频名]_口播稿.md` - 带时间戳的口播稿

---

### 步骤2：视频片段使用推荐 📋
**目标**: 根据教学设计和口播稿，推荐哪些视频片段适合用于教学的哪个环节

**输入**:
1. 教学设计文件（教学设计.md）
2. 口播稿文件（口播稿.md）

**输出**: 视频片段使用推荐表

**推荐内容**:
- 教学活动与视频片段的对应关系
- 每个片段的起止时间
- 使用建议

---

### 步骤3：视频分割 ✂️
**目标**: 根据推荐的时间点或自定义时间点，将视频分割为多个片段

**输入**:
1. 原视频文件
2. 分割点时间列表

**输出**: 分割后的视频片段

---
