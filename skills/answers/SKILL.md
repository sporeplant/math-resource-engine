# 参考答案生成skills（已归档）

本skills已由 `skills/question-dispatch/SKILL.md` 替代。

当前 `/courseware-collab` 不再生成 `{课时名}_课堂提问参考答案.md`，而是生成：

```text
outputs/{课时名}_课堂提问调度稿.md
```

请使用：

- `skills/question-dispatch/SKILL.md`
- `skills/question-dispatch/checklist.md`
- `validators/question-dispatch/rules.md`

核心变更：

- 学生课件不显示具体学生姓名。
- 具体点名、备用学生、观察证据、即时调整规则、参考答案和评分要点只进入教师课堂提问调度稿。
- 旧“课堂提问参考答案”命名不再作为正式产物使用。
