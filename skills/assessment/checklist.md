# 检查清单

## 基础检查项

- [ ] 每条学习目标是否都有对应的评价任务
- [ ] 评价任务的行为动词是否与目标匹配
- [ ] 三层评价标准是否差异化
- [ ] 成功标准是否具体可观察
- [ ] 评价任务、评价标准、成功标准中是否避免了不可观察动词：理解、掌握、了解、熟悉、体会、感受、知道
- [ ] 评价任务、评价标准、成功标准中是否避免了空泛素养标签：提升核心素养、培养数学素养、提高综合能力
- [ ] 是否包含过程性评价
- [ ] 评价任务是否在课时内可完成
- [ ] 评价结果是否能指导教学调整

## 质量检查项

- [ ] 评价任务所用题目是否来源明确（标注教材题/习题库题号/改编来源）
- [ ] 多道题共用同一 `source_id`、`source_type` 时是否使用 `question_ids` 列表合并，且未逗号拼接编号
- [ ] 基础层题目是否优先选用教材练习栏目题目
- [ ] 中间层题目是否优先选用教材习题A组题目
- [ ] 拓展层题目是否优先选用教材习题B组题目
- [ ] 每层是否至少选用了 1 道题目（教材不足时可从练习册补充）
- [ ] 练习册补充选题是否通过练习册索引按 `tier` 字段分层匹配，而非随机选取
- [ ] 练习册题是否携带 `answer_id` 字段，答案是否从 `workbook-answers/` 复用
- [ ] 已标记 `is_open_answer` 的练习册题是否未被选为唯一评价题
- [ ] 是否避免了模糊的成功标准（如"基本正确"）
- [ ] 是否避免了三层使用同一评价标准
- [ ] 评价任务是否与学习目标脱节
- [ ] 是否存在跳过评价设计直接进入活动设计的链条断裂
