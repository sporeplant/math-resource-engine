# 检查清单

## 基础检查项

- [ ] 每条学习目标是否都有对应的评价任务
- [ ] 评价任务的行为动词是否与目标匹配
- [ ] 三层评价标准是否差异化
- [ ] 成功标准是否具体可观察
- [ ] 评价任务、评价标准、成功标准中是否避免了不可观察动词：理解、掌握、了解、熟悉、体会、感受、知道
- [ ] 评价任务、评价标准、成功标准中是否避免了空泛素养标签：提升核心素养、培养数学素养、提高综合能力
- [ ] 是否包含过程性评价
- [ ] 评价任务是否在课时内可完成
- [ ] 评价结果是否能指导教学调整

## 质量检查项

- [ ] 评价任务所用题目是否来源明确（标注教材题或练习册逐题索引题号）
- [ ] 多道题共用同一 `source_id`、`source_type` 时是否使用 `question_ids` 列表合并，且未逗号拼接编号
- [ ] 基础层题目是否优先选用教材练习栏目题目
- [ ] 中间层题目是否优先选用教材习题A组题目
- [ ] 拓展层题目是否优先选用教材习题B组题目
- [ ] 每层是否审查了至少 1 道练习册候选题
- [ ] 被采用的练习册题是否能在 `knowledge/workbook-index/` 中找到，且答案映射非空
- [ ] 每层是否至少选用了 1 道题目
- [ ] 是否避免了模糊的成功标准（如"基本正确"）
- [ ] 是否避免了三层使用同一评价标准
- [ ] 评价任务是否与学习目标脱节
- [ ] 是否存在跳过评价设计直接进入活动设计的链条断裂
- [ ] 数学表达式是否使用 LaTeX 格式（`$...$`），无裸文本公式
