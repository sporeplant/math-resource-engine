# 教材原文拆解技能 检查清单

## Step01 HTML表格转换

- [ ] 是否扫描了文件中所有 `<table>` 标签
- [ ] `colspan` 属性是否正确展开为重复列
- [ ] `rowspan` 属性是否正确向下传递单元格内容
- [ ] 生成的 Markdown 表格行数列数是否正确
- [ ] 分隔线格式是否为 `| --- | --- | ... |`

## Step02 章标题识别

- [ ] 是否识别了 `## 第X章` 格式的章标题
- [ ] 中文数字是否正确转换为阿拉伯数字
- [ ] 章节引言的行范围是否正确（从章标题开始，到第一个课时标题之前结束）

## Step03 课时边界识别

- [ ] 是否识别了 `## X.Y ...（第N课时）` 格式的课时标题
- [ ] OCR 错误（如 `05` 误识别为章号）是否正确修正
- [ ] 课时号（中文数字）是否正确转换
- [ ] 小节名称是否正确提取

## Step04 回顾与反思识别

- [ ] 是否识别了 `## ...回顾与反思（第N课时）` 格式
- [ ] 是否兼容"第X章回顾与反思"和"回顾与反思"两种格式

## Step05 子标题过滤

- [ ] 做一做、大家谈谈、观察与思考 是否被过滤
- [ ] 练习、习题 是否被过滤
- [ ] A组、B组、C组（含带空格的变体）是否被过滤
- [ ] 读一读、一起探究 是否被过滤
- [ ] 法则类标题（以"法则"结尾）是否被过滤
- [ ] 知识结构、总结与反思、注意事项 是否被过滤（回顾与反思内的子标题）
- [ ] 中文编号（一、二、三、）是否被过滤
- [ ] 数字编号（1. 2. 3.）是否被过滤

## Step06 用户确认

- [ ] 是否以结构化格式呈现了课时分配表
- [ ] 是否明确标注了章节引言归入第一课时
- [ ] 是否等待用户确认（除非使用 `-y` 参数）
- [ ] 用户否认时是否正确终止

## Step07 文件拆分

- [ ] 章节引言是否归入第一个课时文件的开头
- [ ] 每个课时文件是否以对应的 `##` 标题开头
- [ ] 数学活动文件是否独立输出
- [ ] 回顾与反思是否按课时拆分
- [ ] 文件命名是否符合 `textbook-{章节}-{课时}.md` 规则

## Step08 内容保真

- [ ] 教材原文内容是否完整保留（无删节）
- [ ] LaTeX 公式是否完整保留
- [ ] 图片引用路径是否完整保留
- [ ] 段落间空行是否保留

## Step09 OCR修正

- [ ] 课时号前缀数字小于章号时，是否自动补全章号
- [ ] 修正后文件内容中的 OCR 错误标题是否同步修正
