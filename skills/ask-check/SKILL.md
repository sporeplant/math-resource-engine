# 提问质量skills

## 0. 全局定位

本skills是教学资源系统的**提问质量总闸门**，不是润色器，也不是单题评分表。

它负责在教学设计、课件、课堂提问调度稿生成前，判断一组提问是否真正推动学生完成数学学习任务。任何提问若不能推动“目标达成、证据产生、活动推进”，必须删除、合并或重写。

本skills适用于：

- `/lesson-collab` 中的问题链、活动提问、评价提问
- `/courseware-collab` 中的课堂提问页、小结提问、练习反馈提问
- 任何局部 Skill outputs中包含的课堂提问

### 0.1 课堂呈现顺序

每个课堂提问强制按以下顺序书写和实施：

1. `提问`：先呈现，保留学生独立思考空间。
2. `备用提示1/2/3`：仅在学生停滞或答偏时逐级提供，不得提前全部呈现。
3. `教师参考预期`：供教师判断学习证据，不向学生直接展示。

禁止“提示→提问”的写法，禁止把参考预期作为课堂提示。

## 1. 最高原则

### 1.1 提问服务教学评一致性

每个保留的问题都必须同时回答三件事：

| 问题 | 必须能回答 |
|------|------------|
| 服务哪个学习结果 | 对应 `LO-B/M/E-XX` 或本课核心学习任务 |
| 产生什么评价证据 | 学生回答、书写、作图、计算、解释、比较中的哪一项可被观察 |
| 推动哪一步学习活动 | 回答后进入分组、计算、证明、归纳、作图、判断、迁移中的哪一步 |

不能说明这三项的问题，不进入下游。

### 1.2 提问对结论负责

提问不是让课堂“热闹”，而是把学生推到关键数学结论前。

一个合格问题必须满足：

- 学生有明确材料可看
- 学生有明确动作可做，且该动作服务当前统计、推理、作图或评价目的
- 回答结果能成为下一步结论、方法或判断的依据
- 若学生答偏，有可执行的提示或追问

如果学生无论怎样回答，都无法自然走向核心结论，该问题无效。

### 1.3 提问尊重认知边界

问题表述必须贴合教材原文、课标要求和学生当前学习进度。

禁止用超前术语替代教材表达；禁止用成人化学科表述包装普通问题；禁止把教材中尚未出现的概念提前放入提问。

## 2. 合格提问的五要素

每个提问必须显式具备以下五项。缺少任意一项，最高评为2分；缺少两项及以上，直接删除或重写。

| 要素 | 检查问题 | 合格表现 |
|------|----------|----------|
| 材料锚点 | 学生看哪里 | 指向教材数据、图形、表格、题干、已写步骤或学生刚完成的产出 |
| 学生动作 | 学生为达成什么目的做什么 | 为统计比例而分类计数、为确定分组而比较范围、为表图转化而作图、为判断方案而提取证据、计算、填表、改写、说明依据、判断、举反例 |
| 数学指向 | 指向哪个数学对象 | 概念、关系、条件、步骤、图表对应、证据、结论 |
| 预期证据 | 回答留下什么证据 | 一个数值、一个式子、一句依据、一个图形、一个判断、一个错误暴露点 |
| 后续出口 | 回答后去哪一步 | 进入新概念、下一步推理、作图规则、评价标准、迁移任务 |

### 2.1 禁止只问主观困难

下列问法一律视为低质问题：

- “哪里卡住了？”
- “为什么不容易？”
- “有什么困难？”
- “你发现了什么？”
- “你有什么想法？”
- “这样可以吗？”
- “对不对？”

这些问法只能作为教师现场追问口语，不能作为结构化提问进入教学设计或课件。

### 2.2 改写原则

| 低质问法 | 改写方向 |
|----------|----------|
| 问困难 | 设计一个会暴露困难的小任务 |
| 问发现 | 指定两个对象或两个维度进行比较 |
| 问原因 | 先给材料范围，再要求说明依据 |
| 问判断 | 要求给出数据、条件、步骤或反例 |
| 问特点 | 指定观察维度，并说明该特点用于哪一步 |

## 3. 提问功能分类

提问不是按“像不像开放题”分类，而是按课堂功能分类。

| 类型 | 功能 | 合格问法特征 | 常见场景 |
|------|------|--------------|----------|
| 定位型 | 找准对象、条件、数据范围 | 指向具体材料和对象 | 导入、读题、读图 |
| 转化型 | 把一种表达转成另一种表达 | 明确从什么转到什么 | 表到图、文字到式子、图形到关系 |
| 依据型 | 要求说明理由或来源 | 必须有“依据是什么” | 计算、证明、判断 |
| 冲突型 | 暴露易错点或边界问题 | 让学生处理边界、反例、漏项 | 分组边界、条件不足 |
| 归纳型 | 从多个结果收束方法 | 要求提炼步骤或规则 | 小结、方法形成 |
| 评价型 | 用证据支持判断 | 必须引用数据、条件或图表信息 | 实际问题、方案比较 |
| 迁移型 | 换情境使用同一方法 | 明确保持什么、变化什么 | 练习、拓展、机动题 |

一个课时的问题链应至少包含“定位型 → 转化/依据型 → 归纳/评价型”的推进。

---

## 4. 分层提问规则

分层不是给学生贴标签，而是控制问题台阶。

| 层级 | 可承担的问题 | 问题动作 |
|------|--------------|----------|
| 基础层 | 定位、辨认、简单计算、照样完成第一步 | 为统计或判断目的找数据、读表、计数、计算、填表、完成作图第一步 |
| 中间层 | 解释依据、完成规则转化、处理易错边界 | 比较依据、改写区间、完成表图转化、补全步骤 |
| 拓展层 | 评价方案、迁移方法、提出证据链 | 判断、选择证据、比较方案、迁移应用 |

基础层问题不能只问“是什么”；应让学生围绕明确目的在具体材料上完成可见动作。禁止把圈、画、标、找等无目的操作当作合格提问。

中间层问题必须要求依据或方法，不得只是换个说法复述。

拓展层问题必须要求证据，不得只给结论。

---

## 5. 提问链规则

### 5.1 单环节最多两问

同一活动环节内，通常只保留1个主问题；确需递进时最多2问。

两问必须形成：

```text
低台阶动作 → 方法或依据
材料定位 → 数学转化
局部判断 → 证据表达
```

禁止在同一环节连续追问同一认知层级。

### 5.2 全课问题链必须有主线

全课提问应围绕一个核心学习路径推进。

示例：

```text
原始材料 → 明确任务 → 处理对象 → 建立方法 → 使用方法 → 证据判断 → 迁移回看
```

如果某个问题不能放入主线，只能作为课堂临场追问，不写入结构化outputs。

### 5.3 问题先于答案

凡用于引出方法、结论、证明、计算路径的问题，必须出现在答案页、结论页或规范过程页之前。

小结回看、出口检测、已学内容纠错可在结论之后出现，但必须标明其功能是回看或检测。

## 6. 评分标准

### 6.1 单题评分

| 分数 | 等级 | 判断标准 |
|:---:|------|----------|
| 4 | 核心问题 | 五要素完整，指向关键数学转折点，回答能直接推动下一步结论或方法 |
| 3 | 合格问题 | 五要素基本完整，有明确材料和动作，但结论出口或追问还可更精确 |
| 2 | 弱问题 | 缺少一个关键要素，或与前后问题同层重复 |
| 1 | 删除问题 | 无材料、无动作、无出口；或属于封闭确认、伪开放、主观困难追问、事实复读 |

### 6.2 整体通过条件

| 指标 | 通过标准 |
|------|----------|
| 平均分 | ≥ 3.2 |
| 4分问题占比 | ≥ 50% |
| 1分问题 | 0个 |
| 2分问题 | 必须重写或合并，不得原样保留 |
| 五要素完整率 | ≥ 90% |
| 主线完整性 | 能用一句话说出全课问题链 |

未达到任一硬性条件，必须回退到提问设计环节。

---

## 7. 硬性禁止

- 禁止无材料锚点的问题。
- 禁止无学生动作的问题。
- 禁止无结论出口的问题。
- 禁止“哪里卡住了”“为什么不容易”“你发现什么”“你怎么看”等泛问进入结构化outputs。
- 禁止用封闭确认代替思考，如“是不是”“对不对”“可以吗”。
- 禁止重复询问同一概念。
- 禁止把教材原文可直接抄到的句子作为提问答案。
- 禁止问题早于学生可用材料出现。
- 禁止问题晚于结论出现后再假装引导。
- 禁止拓展层问题不要求证据。
- 禁止基础层问题只让学生机械复述概念。

## 9. 自检流程

1. 逐题标出材料锚点、学生动作、数学指向、预期证据、后续出口。
2. 删除所有无材料、无动作、无出口的问题。
3. 删除或改写所有泛问、封闭确认和事实复读问题。
4. 检查每个问题是否服务对应学习结果和评价证据。
5. 检查同一环节是否超过2问，是否存在同层堆积。
6. 检查基础层、中间层、拓展层问题是否匹配认知台阶。
7. 写出全课问题主线；若写不出，重排问题链。
8. 计算评分；不达标则回退重写。

## 10. 合格问题示例库

### 统计与数据

| 场景 | 合格问题 |
|------|----------|
| 原始数据进入 | 制定阶梯电价时，要判断第一档电价能否覆盖大多数家庭，仅看50个原始用电量数据够不够？还需要统计哪些范围内的户数和比例？ |
| 分组边界 | 如果一个数据正好是120，它应放入哪一组？请用两个相邻区间说明理由。 |
| 表图转化 | 画直方图时，横轴、纵轴、小长方形高度分别对应频数表中的哪一项？ |
| 证据判断 | 判断第一档电价是否覆盖大多数居民，需要把哪些组的频数相加？结果是多少户，占多少？ |

### 几何证明

| 场景 | 合格问题 |
|------|----------|
| 条件定位 | 题中哪些条件能推出平行或相等？请在图中标出对应线段或角。 |
| 证明入口 | 若要证明四边形是平行四边形，当前图中最容易凑出哪一组条件？ |
| 逻辑传递 | 这个中点条件能把哪条边的关系转移到哪条线段上？ |
| 反例辨析 | 若去掉这个条件，结论还成立吗？请画一个可能的图形说明。 |

### 代数与函数

| 场景 | 合格问题 |
|------|----------|
| 式子转化 | 题目中的哪句话对应自变量，哪句话对应函数值？请写成一个等式。 |
| 图象读数 | 从图象上读出当 $x=2$ 时的函数值，并说明你看的是哪一个点。 |
| 参数影响 | 比较两个表达式中变化的系数，图象的哪个特征随之改变？ |
| 实际判断 | 若要判断方案更省费用，应比较哪两个函数值或哪两个图象位置？ |
