# 练习册栏目图标清除 — 检查清单

## 执行前检查

- [ ] 确认目标是练习册题库文件（`knowledge/workbooks/`），不是教材原文
- [ ] 确认栏目标题为一级 `#`，不是教材的二级 `##`
- [ ] 栏目列表已确认（默认：知识点拨/夯实基础/数学思考/解决问题/开阔视野）
- [ ] 已运行 `--dry-run` 预览将要删除的图标，确认无误

## 执行后验证

- [ ] 图标引用行已移除，栏目标题本身完整保留
- [ ] 正文图片（紧跟题干、选项、序号的图片）未被误删
- [ ] 题号、公式、选项内容未受影响
- [ ] 图片文件已同步删除（独立工具模式）；或图标图片未进入 knowledge/images/（集成模式）

## 两层工具选择

- [ ] 新分割文件 → 使用 `split_workbook.py --remove-icons`（分割时自动清除）
- [ ] 已拆分文件补救 → 使用 `remove_workbook_icons.py`（事后独立清除）

## 禁止事项核对

- [ ] 未使用教材侧 `remove_section_icons.py` 处理练习册
- [ ] 未删除正文插图或公式图片
- [ ] 未修改栏目标题本身
- [ ] 未修改图标引用以外的任何内容
