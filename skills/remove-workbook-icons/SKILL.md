# 练习册栏目图标清除技能

## 1. Skill 定位

清除练习册题库 Markdown 文件中紧邻栏目标题（如"夯实基础""解决问题"等）前方的图标图片引用，同时删除对应的图标图片文件。

本 Skill 只处理练习册题库原文，不处理教材原文。教材原文图标清除使用 `skills/remove-section-icons/SKILL.md`。

## 2. 适用场景

练习册题库文件中，部分栏目标题（`# 夯实基础`、`# 解决问题`、`# 开阔视野` 等）前方有一行图标图片引用，这些图标是排版装饰元素，不属于练习册正文内容。在正式教学资源生成前应予以清除。

## 3. 与教材侧的差异

| 维度 | 教材（remove-section-icons） | 练习册（本 Skill） |
|------|------------------------------|------------------|
| 标题层级 | `## 栏目名`（二级） | `# 栏目名`（一级） |
| 栏目集 | 观察与思考/做一做/练习/一起探究/大家谈谈/读一读/复习题/回顾与反思 | 知识点拨/夯实基础/数学思考/解决问题/开阔视野 |
| 图标覆盖 | 各栏目前普遍有 | 选择性出现（仅夯实基础/解决问题/开阔视野前观测到） |

## 4. 默认目标栏目

以下栏目标题前方的图标图片将被清除：

- `知识点拨`
- `夯实基础`
- `解决问题`
- `数学思考`
- `开阔视野`

## 5. 两层工具架构

与教材侧对齐，本 Skill 提供两层清除能力：

### 5.1 分割时集成清除（推荐）

`tools/split_workbook.py` 内置 `--remove-icons` 标志，在分割写出每个课时文件时自动清除图标引用。清除发生在 `normalize_body` 之后、`rewrite_and_copy_images` 之前，因此图标图片不会被复制进 `knowledge/images/`。

```bash
python tools/split_workbook.py <MinerU文件路径> --remove-icons --outdir knowledge/workbooks -y
```

### 5.2 事后独立清除

`tools/remove_workbook_icons.py` 用于清除已拆分文件中的残留图标引用。适用于分割时未加 `--remove-icons` 的补救场景。

```bash
python tools/remove_workbook_icons.py <目标目录> [--headers 栏目1,栏目2,...] [--dry-run] [--no-delete-images]
```

| 参数 | 说明 |
|------|------|
| `target_dir` | 练习册题库 MD 文件所在目录（必填，如 `knowledge/workbooks`） |
| `--headers` | 自定义栏目列表，逗号分隔（可选，默认使用全部 5 个栏目） |
| `--dry-run` | 仅预览，不实际修改和删除（可选） |
| `--no-delete-images` | 仅删除 MD 中的引用，不删除图片文件（可选） |

## 6. 识别模式

```
![](https://cdn.jsdelivr.net/.../knowledge/images/<hash>.jpg)   ← 图标引用行
                                                                  ← 1 个空行
# 栏目名                                                          ← 一级栏目标题
```

正则允许标题前有可选的 OCR 噪声前缀（如全角 `？`），但**不修改标题本身**。

## 7. 处理规则

1. **精确匹配**：仅删除紧邻栏目标题（允许中间有空行）前方的图片引用行，正文中的插图不受影响
2. **同步删除**：独立工具模式下，删除 MD 文件中图标引用的同时，删除 `knowledge/images/` 下对应的图片文件；集成模式下图标图片根本不会被复制
3. **不改内容**：除图标引用行外，不修改任何练习册原文内容
4. **正文图片保护**：紧跟题干文字（如"甲""乙""第1(1)题""A/B/C 选项"）的图片属于正文插图，绝不能删

## 8. 输出

- 修改后的 MD 文件（移除图标引用行）
- 已删除的图片文件列表（独立工具模式）
- 操作统计：修改文件数、删除引用数、删除图片数

## 9. 禁止事项

- 禁止删除正文中的插图或公式图片
- 禁止修改栏目标题本身
- 禁止修改图标引用以外的任何内容
- 禁止在未确认栏目列表的情况下批量执行
- 禁止使用教材侧 `remove_section_icons.py` 处理练习册文件（标题层级和栏目集不同）

## 10. 自检流程

- [ ] 目标目录是否存在且包含 MD 文件
- [ ] 栏目列表是否已确认（默认 5 个练习册栏目）
- [ ] 是否已预览将要删除的图标（建议先 dry-run）
- [ ] MD 文件修改后，栏目标题是否完整保留
- [ ] 正文图片（紧跟题干、选项的图片）是否未被误删
- [ ] 图片文件是否已同步删除（独立工具模式）
