# 复习课讲义整理检查清单

## 生成前检查

- [ ] 已读取 `AGENTS.md`
- [ ] 已读取 `orchestrator/workflow-registry.md` 中 `/复习讲义` 工作流
- [ ] 已读取 `orchestrator/output-contract.md` 中 `review_lesson` 元数据与outputs规则
- [ ] 已读取 `skills/reviews/SKILL.md`
- [ ] 已读取 `knowledge/types/复习课.md`
- [ ] 已读取用户确认的全部源讲义文件
- [ ] 已确认outputs路径为 `outputs/g8-reviews/`

## 统计确认检查

- [ ] 已对各源文件分别统计知识点数量
- [ ] 已对各源文件分别统计即学即练题数
- [ ] 已对各源文件分别统计各题型N 板块的典例数量
- [ ] 已对各源文件分别统计各题型N 板块的变式数量
- [ ] 已向用户呈现统计表格并等待确认
- [ ] 已确认课后作业固定选用 10 题
- [ ] 用户确认后锁定统计数据，继续生成

## 选题检查

- [ ] 知识点按源讲义顺序提取，未打乱、未合并
- [ ] 已识别源docs中的「题型精讲」区域、`## 题型 NN` 板块、`【典例...】` 和 `【变式...】` 标记
- [ ] 每个知识点至少配 1 道例题（优先来自「即学即练」）
- [ ] 当堂练习已纳入全部 题型N 典例，无遗漏
- [ ] 课后作业从变式中固定选取 10 题，优先级：例题变式 > 其他变式 > 有图综合题 > 原课后练习
- [ ] 变式题保持归属于其前最近的 `## 题型 NN` 和对应典例，不脱离原题型重新归类
- [ ] 题型和难度尽量贴近源讲义原文比例
- [ ] 至少包含基础题、中等题、中等偏难题
- [ ] 未虚构题目、未改写题目、未用仿题替代原题

## 格式检查

- [ ] 文件包含 YAML front matter
- [ ] `content_type: review_lesson`
- [ ] `command: 复习讲义`
- [ ] `source_files` 登记全部源讲义文件
- [ ] 包含讲标题、知识点、例题讲解、当堂练习、课后作业
- [ ] 课后作业恰好 10 题
- [ ] 题号从 `1．` 连续编号，不重置
- [ ] docs末尾包含「原始数量与选用数量对比」表格
- [ ] 对比表格字段为 `来源 / 类别 / 原始数量 / 选用数量 / 备注`
- [ ] 对比表格行数与统计项一致（含合计行）
- [ ] 对比表格中原始数量合计与选用数量合计正确

## 图片检查

- [ ] 只搬运被选用题目的图片
- [ ] 图片均位于outputs文件同级 `images/` 目录
- [ ] 图片引用格式为 `![](images/文件名)`
- [ ] 无绝对路径、跨目录路径或空图片路径

## 验证检查

- [ ] 已运行 `python tools/validate_review_lesson.py {outputs文件}`
- [ ] 已运行 `python tools/validate_output.py {outputs文件}`
- [ ] 验证不通过时已回退修正，未发布正式产物
