# 检查清单

## 基础检查项

- [ ] 每条目标是否以"能……"开头
- [ ] 行为动词是否属于对应层级
- [ ] 目标是否可观察、可评价
- [ ] 三层之间是否有认知递进
- [ ] 学习目标、学习结果表述中是否避免了不可观察动词：理解、掌握、了解、熟悉、体会、感受、知道
- [ ] 学习目标、学习结果表述中是否避免了空泛素养标签：提升核心素养、培养数学素养、提高综合能力
- [ ] 目标总数是否控制在6条以内，不强制每层固定数量
- [ ] 是否与课标要求一致
- [ ] 数学表达式是否使用 LaTeX 格式（`$...$`），无裸文本公式

## 质量检查项

- [ ] 是否可评价
- [ ] 是否可观察
- [ ] 是否有行为动词
- [ ] 是否体现数学活动
- [ ] 是否包含认知分层
- [ ] 是否符合课标
- [ ] 是否支持后续评价设计
