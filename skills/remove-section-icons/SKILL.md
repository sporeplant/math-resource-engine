# 栏目图标清除技能

## 1. Skill 定位

清除教材原文 Markdown 文件中紧邻栏目标题（如"做一做""观察与思考"等）前方的图标图片引用，同时删除对应的图标图片文件。

## 2. 适用场景

教材原文文件中，部分栏目标题（`## 观察与思考`、`## 做一做`、`## 练习` 等）前方有一行图标图片引用，这些图标是排版装饰元素，不属于教材正文内容。在正式教学资源生成前应予以清除。

## 3. 输入要求

| 字段 | 说明 |
|------|------|
| 目标目录 | 教材原文 Markdown 文件所在目录（如 `knowledge/textbooks/ch12/`） |
| 栏目列表 | 需要清除图标的目标栏目名称（可选，默认全部） |

## 4. 默认目标栏目

以下栏目标题前方的图标图片将被清除：

- `观察与思考`
- `做一做`
- `练习`
- `一起探究`
- `大家谈谈`
- `读一读`
- `复习题`
- `回顾与反思`

## 5. 处理规则

1. **精确匹配**：仅删除紧邻栏目标题（允许中间有空行）前方的图片引用行，正文中的插图不受影响
2. **同步删除**：删除 MD 文件中图标引用的同时，删除 `knowledge/images/` 下对应的图片文件
3. **不改内容**：除图标引用行外，不修改任何教材原文内容
4. **识别模式**：`![](https://...)` 行 + 空行 + `## 栏目名`

## 6. 调用方式

```bash
python tools/remove_section_icons.py <目标目录> [--headers 栏目1,栏目2,...]
```

| 参数 | 说明 |
|------|------|
| `target_dir` | 教材原文 MD 文件所在目录（必填） |
| `--headers` | 自定义栏目列表，逗号分隔（可选，默认使用全部 7 个栏目） |
| `--dry-run` | 仅预览，不实际修改和删除（可选） |
| `--no-delete-images` | 仅删除 MD 中的引用，不删除图片文件（可选） |

## 7. 输出

- 修改后的 MD 文件（移除图标引用行）
- 已删除的图片文件列表
- 操作统计：修改文件数、删除引用数、删除图片数

## 8. 禁止事项

- 禁止删除正文中的插图或公式图片
- 禁止修改栏目标题本身
- 禁止修改图标引用以外的任何内容
- 禁止在未确认栏目列表的情况下批量执行

## 9. 自检流程

- [ ] 目标目录是否存在且包含 MD 文件
- [ ] 栏目列表是否已确认
- [ ] 是否已预览将要删除的图标（建议先 dry-run）
- [ ] MD 文件修改后，栏目标题是否完整保留
- [ ] 正文图片是否未被误删
- [ ] 图片文件是否已同步删除
