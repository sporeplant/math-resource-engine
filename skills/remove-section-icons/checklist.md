# 栏目图标清除技能 检查清单

## Step01 目标确认

- [ ] 目标目录路径是否正确
- [ ] 目录下是否存在教材原文 MD 文件
- [ ] 栏目列表是否已确认（默认 7 个或自定义）

## Step02 预览检查（dry-run）

- [ ] 是否先执行了 `--dry-run` 预览
- [ ] 预览结果中的图标引用是否都紧邻目标栏目标题
- [ ] 有无误匹配的非图标图片（如正文插图恰好位于栏目标题前）
- [ ] 图片文件路径是否可在 `knowledge/images/` 下找到

## Step03 执行

- [ ] MD 文件中图标引用行是否已删除
- [ ] 栏目标题（`## 观察与思考` 等）是否完整保留
- [ ] 正文内容（含公式、表格、正文插图）是否未被修改
- [ ] 图片文件是否已从 `knowledge/images/` 删除

## Step04 验证

- [ ] 再次搜索目标栏目，确认前方不再有图标引用
- [ ] 抽查正文插图，确认未被误删
- [ ] 文件行数和结构是否符合预期
