# 练习册题库拆解技能 检查清单

## Step01 类型确认

- [ ] 输入文件是否为练习册题库，而非教材原文
- [ ] 是否包含“知识点拨 / 夯实基础 / 数学思考 / 解决问题”等练习册栏目
- [ ] 是否不套用教材原文的 YAML、lesson_id 和 `textbook_original` 规则

## Step02 边界识别

- [ ] 是否识别章标题 `# 第X章 ...`
- [ ] 是否识别 `## X.Y ...` 课时标题
- [ ] 是否从 `(一)`、`(二)` 等标题中识别课时号
- [ ] 是否识别 `回顾与反思`
- [ ] 是否没有把题型标题或题号标题误判为课时

## Step03 内容保真

- [ ] 题干是否完整保留
- [ ] 选项是否完整保留
- [ ] 公式是否完整保留
- [ ] 题号是否完整保留
- [ ] 栏目顺序是否与原文一致

## Step04 输出位置和命名

- [ ] 输出目录是否为 `knowledge/workbooks/`
- [ ] 课时文件是否命名为 `workbook-{章节}.{小节}-{课时号}.md` 或 `workbook-{章节}.{小节}.md`
- [ ] 回顾与反思是否命名为 `workbook-ch{章号}-review.md`
- [ ] 未经用户允许是否没有覆盖已有文件

## Step05 图片处理

- [ ] 是否扫描所有 Markdown 图片引用
- [ ] 是否将图片复制到 `knowledge/images/`
- [ ] 输出文件中是否全部使用 CDN 图片路径
- [ ] CDN 路径对应的本地图片是否存在

## Step06 验证

- [ ] 是否运行 `python tools/validate-workbook-split.py <输出文件或目录>`
- [ ] 验证器是否通过
