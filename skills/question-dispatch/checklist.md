# 课堂提问调度稿生成skills 检查清单

## 基础检查项（强制通过）

- [ ] 文件元数据是否完整（content_type、lesson_id、lesson_name、source_files、created_at）
- [ ] `content_type` 是否为 `question_dispatch`
- [ ] 标题是否正确（`{课时名} 课堂提问调度稿`）
- [ ] 提问对象建议表是否存在且表格格式正确
- [ ] 板块A【ASK课堂提问调度稿】是否存在
- [ ] 板块B【例题与练习解答】是否存在
- [ ] 评分量规是否存在且表格格式正确
- [ ] 所有题目是否标注了 `source_id`、`source_type`、`question_id`
- [ ] 所有题目来源是否为 knowledge 中的真实题源（禁止 LLM 自行生成题目）
- [ ] 是否已读取对应课时的教材参考解答，且文件存在、课时匹配并通过教材问题解答 validators
- [ ] `source_files` 是否登记教学设计、学生课件和教材参考解答路径
- [ ] 数学公式是否使用 LaTeX 格式（如 `$x^2$`、`$\because$`、`$\therefore$`）

---

## 学生课件隔离检查

- [ ] 学生课件中是否未出现具体学生姓名
- [ ] 学生课件中是否未出现 ASK 编号、后台题源字段、教师参考预期、备用提示和调度说明
- [ ] 调度稿中的问题原文是否与学生课件中的问题一致
- [ ] 调度稿中的课件页码或页面标题是否能回指到学生课件对应页面

---

## 板块A质量检查项

### 结构完整性

- [ ] 是否按课件页码或课堂流程顺序排列
- [ ] 是否按基础层、中间层、拓展层标注问题层级
- [ ] 每个问题是否包含：问题编号、课件页码/页面标题、所属活动、学生姓名、备用学生、问题原文、提问意图、观察证据、分级备用提示、参考答案、即时调整规则、评分要点
- [ ] 每个问题是否按“问题→分级备用提示→参考答案→即时调整规则→评分要点”排列
- [ ] 问题编号格式是否正确（ASK-B-XX、ASK-M-XX、ASK-E-XX）
- [ ] 学生姓名是否与分层提问skills输出的选取结果一致
- [ ] 备用学生是否来自相同或合适层级，且不是本问题建议提问学生本人

### 内容质量

- [ ] 提问意图是否具体，能说明该问题要观察的学习证据
- [ ] 观察证据是否可观察、可判断
- [ ] 参考答案是否拆分为清晰要点（3-5个要点）
- [ ] 基础层答案是否侧重事实性、记忆性内容
- [ ] 中间层答案是否侧重推理性、应用性内容
- [ ] 拓展层答案是否侧重开放性、迁移性内容
- [ ] 分级备用提示是否只提供支架，不直接泄露完整答案
- [ ] 即时调整规则是否覆盖答不出、答偏、基本答出、完整答出等课堂情况
- [ ] 评分要点是否明确可观察、可量化
- [ ] 每个评分项是否配有具体分值
- [ ] 评分要点总分是否控制在 2-5 分
- [ ] 直接对应教材任务的 ASK 答案是否以同一 `question_id` 的教材参考解答为事实基线
- [ ] 是否保留了教材参考解答中的答案来源属性

### 数学准确性

- [ ] 所有数学概念、定义表述是否准确
- [ ] 所有公式、定理引用是否正确
- [ ] 所有推理步骤是否逻辑完整、无断裂
- [ ] 所有计算结果是否正确

---

## 板块B质量检查项

### 覆盖范围

- [ ] 是否覆盖了教学设计 lesson_flow 和学生课件中引用的所有教材例题
- [ ] 是否覆盖了学生课件中引用的所有课堂练习/检测题目
- [ ] 是否覆盖了学生课件中引用的所有习题库题目
- [ ] 台阶递进式核心题是否包含完整解题步骤（不仅是最终答案）

### 题源对应

- [ ] 每个题目是否标注了 `source_id`
- [ ] 每个题目是否标注了 `source_type`（TEXTBOOK / WORKBOOK）
- [ ] 每个题目是否标注了 `question_id`
- [ ] 题目标注是否与教学设计、学生课件中的标注一致
- [ ] 每个教材 `question_id` 是否都能在教材参考解答中找到
- [ ] 教材题是否直接复用已有解答，而非重新独立推导
- [ ] 练习册题是否仍按练习册题源处理，未被教材参考解答错误覆盖

### 解答质量

- [ ] 证明题是否包含完整推理链，每步有依据
- [ ] 计算题是否展示关键计算步骤和最终结果
- [ ] 概念题是否包含准确定义和辨析说明
- [ ] 应用题是否包含建模过程、求解步骤、结论解释
- [ ] 解答语言是否使用规范数学术语

---

## 协作一致性检查

- [ ] 学生姓名是否与分层提问skills outputs 的选取结果完全一致
- [ ] 问题内容是否与学生课件中的问题完全一致
- [ ] 问题内容是否与教学设计中的问题意图一致
- [ ] 题目解答是否与习题分析skills的8维度分析一致
- [ ] 评分量规是否与评价设计skills的评分标准一致

---

## 最终确认

- [ ] 通读全文，确认无数学概念错误
- [ ] 通读全文，确认无公式、定理错误
- [ ] 通读全文，确认无推理逻辑错误
- [ ] 通读全文，确认无计算错误
- [ ] 确认所有题目来自题源，无 LLM 自行生成题目
- [ ] 确认学生课件不显示具体学生姓名，姓名只进入教师调度稿
