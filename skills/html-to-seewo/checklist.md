# html-to-seewo 检查清单

生成 JSON 和截图后，逐项检查：

## 截图完整性

- [ ] 截图数量 = HTML 中 `.slide` 元素数量
- [ ] 每张截图尺寸 1920×1080（允许 ±1px 子像素偏差）
- [ ] 截图内容完整，无截断、无偏移
- [ ] 公式已渲染，无 `$` 源码残留
- [ ] 中文正常显示，无乱码或 tofu 字符

## JSON 合规

- [ ] 文件为合法 JSON，可直接解析
- [ ] 每页 `imagePath` 指向正确的 `assets/slide_*.png`
- [ ] 页码连续，从 01 开始

## assets 目录

- [ ] `assets/` 目录存在且位于 HTML 同级
- [ ] 文件名格式统一：`slide_01.png` ~ `slide_NN.png`
- [ ] 无多余文件混入

## 导入验证（可选）

- [ ] 希沃白板5 中 MrePlugin 能成功导入生成的 `.json` 文件
- [ ] 导入后页数与截图一致
- [ ] 每页图片清晰，缩放正确
