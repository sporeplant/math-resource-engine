# 检查清单

## 基础检查项

- [ ] 课件是Markdown格式，每页用 `<div style="page-break-after: always;"></div>` 分隔
- [ ] 第1页为标题页（含 `### 📐` 标题，章节目录）
- [ ] 学习目标页是否存在，使用编号 `①②③④` 格式
- [ ] 是否已读取对应教材原文，并把支撑作答的背景、原始数据、统计表、方案表、图形、题干摘要转化为课件可见材料
- [ ] 每个课堂提问在出现前或同页是否有可见材料锚点，学生离开教材也能据此作答
- [ ] 每页标题是否包含正确的emoji（📐/🎯/📖/✏️/📝/💡/🤔），无省略
- [ ] 每页最多使用1个emoji，正文中无装饰性emoji
- [ ] 推理证明步骤中无emoji
- [ ] 四边形关系图是否使用mermaid流程图
- [ ] 每张mermaid图节点数是否 ≤ 6，是否配图注说明核心结论
- [ ] 有无使用ASCII字符画替代专业图示（推理链必须用mermaid）
- [ ] 每页正文是否控制在7行以内（不含mermaid图块和图片引用）
- [ ] 无连续3页以上纯文字无图示（mermaid/图片引用/占位符均可）
- [ ] 所有例题是否拆分为：题目页 → 提问环节页 → 推理链分析页 → 规范证明页
- [ ] 两道及以上步骤的例题末尾是否附推理链（`→` 箭头连接）
- [ ] 证明和计算步骤禁止使用数字标号（`1.` `2.` `3.`），用空行分隔区块
- [ ] 课堂小结是否为三问三答表格（基础/提升/拓展各一个问题）
- [ ] 课后作业页是否存在（含必做/选作/挑战三类）
- [ ] 无整页未分段的纯文字墙

## 教学材料专项检查

- [ ] 仅出现"教材""练习册""练习本/纸笔"三种教学材料
- [ ] 无"学习单""任务单""检测卡"等额外物料
- [ ] 无"不同颜色笔""投影胶片""贴纸"等特殊tools-2
- [ ] 所有学生产出对应练习本书写或口头回答，无其他形式
- [ ] 统计/数据课是否呈现原始数据表、频数表或方案表等关键教材材料，未只写“看教材”

## 课堂练习/检测专项检查

- [ ] 课堂练习/检测题目来源是否符合规则：教材练习栏目全部题目、习题A组选1题、习题B组选1题
- [ ] 课堂练习/检测题目总数量是否 ≥ 4 道
- [ ] 练习/检测页标题是否使用 `### 📝 练习/检测`（非 `### 📝 练习：xxx`）
- [ ] 练习/检测页是否**仅出示题目编号位置和作答要求，未显示完整题干**
- [ ] 练习/检测页是否标注了产出形式（`✏️ 请在练习本上完成`）
- [ ] 每道课堂练习/检测题目是否标注了来源（`教材练习第X题 / 教材习题A组第X题 / 教材习题B组第X题`）
- [ ] 每道课堂练习/检测题目是否标注了预估完成时间（`限时 X 分钟`）
- [ ] 每道课堂练习/检测题目是否标注了评分标准
- [ ] 所有课堂练习/检测题目是否均有对应的参考答案页
- [ ] 参考答案页是否紧随题目要求页之后
- [ ] 参考答案是否适当密集排版（同一页可放多道题答案，非每题单独一页）
- [ ] 参考答案是否包含完整解题过程（证明题含推理链、计算题含关键步骤、概念题含定义要点）
- [ ] 参考答案的数学符号是否正确使用 LaTeX 格式
- [ ] 所有题目（含参考答案中的题目）是否携带 `source_id`/`source_type`/`question_id`
- [ ] 未被选为课堂练习/检测的教材习题是否已纳入课后作业
- [ ] 课后作业中教材题目是否标注了教材页码和题号

## 质量检查项

- [ ] 学生活动是否标注了时间限制（格式：`（限时 X 分钟）`）
- [ ] 练习/检测页是否标注了评分标准（格式：`评分：...`）
- [ ] 每个活动页标题下方是否标注了产出形式（`✏️ 请在练习本上完成` / `🗣️ 口头回答` / `🗣️ 小组讨论后回答`）
- [ ] 提问页格式是否为 `**【层提问】**：请[姓名]同学……`
- [ ] 学生姓名是否来自学生成绩数据的三层池，与参考答案文件一致
- [ ] 分层提问标记是否保留（学生需要看到谁被提问）
- [ ] 复杂公式是否独占一行，不嵌入正文堆叠
- [ ] 重点内容是否使用加粗强调，无全文加粗
- [ ] 选择题/判断题的提问是否未透露答案（无"为什么A是错的"等）
- [ ] 每页信息量是否控制在学生能在2-3分钟内消化完毕
- [ ] 所有需配图的页面是否包含图片引用或占位符
- [ ] 图片引用使用 `<img src="..." width="35%">` 格式，非Markdown `![](...)` 语法
- [ ] 图片引用路径是否统一使用 `./images/文件名`
- [ ] 无猜测或编造的假图片文件名（必须从教材源文件读取真实文件名）
- [ ] 课件中无「教师：」「学生：」等教师动作提示或学生操作指令
- [ ] 无直接复制教材完整段落作为页面内容
- [ ] 与教学设计中的核心板块内容无完全重复

## 问题先行专项检查（Question-First）

- [ ] 每个问题是否按“问题页→按需备用提示页→答案/归纳页”排列
- [ ] 问题页是否未提前显示提示、答案或教师参考预期
- [ ] 每个备用提示页是否重复对应的 `ASK_*` 编号，且没有出现在问题页之前
- [ ] 学生课件中是否完全没有 `教师参考预期` 字段或教师手持答案内容
- [ ] 每个复杂例题的子页顺序是否为：题目页 → 提问环节页 → 推理链分析页 → 规范证明页
- [ ] 每个简单例题的课堂提问是否出现在答案页之前
- [ ] 提问页是否在推理链分析页和规范证明页之前（而非之后）
- [ ] 是否存在"提问/讨论在前但问题内容是让学生复述答案"的伪提问
- [ ] 例题中未被提问的学生在提问页是否保留"独立思考后全班交流"的隐性参与空间
- [ ] 例外场景（出口检测、判断题、前置激活）的提问是否确实不剧透后续内容

## 分层提问专项检查

- [ ] 是否读取了学生成绩数据（students/scores.md）
- [ ] 学生姓名是否从数据源提取，非虚构
- [ ] 分层是否正确（基础层0-49分/中间层50-69分/拓展层70-100分）
- [ ] 无跨层提问（基础层问题配基础层学生）
- [ ] 教学设计中未写死具体学生姓名（只标注层级）
- [ ] 同一课时内无学生被重复提问
- [ ] 参考答案文件是否控制在3-4页A4纸内
- [ ] 参考答案末尾是否附有评分量规（准确性/完整性/表达力）
- [ ] 跨课时历史记录是否已写入（students/ask-log.json）
