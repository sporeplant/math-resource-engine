# 教材问题解答skills检查清单

## 课时与题源

- [ ] 是否唯一匹配教材课时并读取完整教材原文
- [ ] 是否读取并核对所有相关教材图片
- [ ] 是否按教材原文顺序建立任务清单
- [ ] 是否同时识别显式问句和无问号任务指令
- [ ] 是否拆分全部大题、小问和共同题干

## 覆盖完整性

- [ ] 是否覆盖正文中的全部直接任务
- [ ] 是否覆盖观察与思考、大家谈谈、一起探究、做一做等全部栏目任务
- [ ] 是否登记全部例题设问及教材已有解答
- [ ] 是否覆盖练习全部题目
- [ ] 是否覆盖习题各组全部题目
- [ ] 是否覆盖数学活动、回顾与反思、项目式学习中的明确任务
- [ ] 任务清单数量是否与参考解答条目数量一致

## 单题结构

- [ ] 每题是否包含 `source_id`、`source_type`、`question_id`、教材位置和教材顺序
- [ ] 每题是否包含任务类型和认知层级
- [ ] 每题是否使用中文标记答案来源
- [ ] 教材已有解答是否标记 `答案来源: 教材原文`
- [ ] AI推导是否标记 `答案来源: AI参考推导`
- [ ] 原题是否忠实引用且未被改写
- [ ] 图片题是否绑定正确教材图片

## 解答质量

- [ ] 计算题是否包含关键步骤、结果和必要检验
- [ ] 证明题是否包含完整推理链及每步依据
- [ ] 作图题是否包含步骤、依据和检验
- [ ] 操作探究题是否包含操作、现象、结论和理由
- [ ] 开放题是否包含参考要点和合理答案边界
- [ ] 调查实践题是否避免虚构真实数据
- [ ] 主解法是否处于教材当前认知边界内
- [ ] 数学概念、公式、计算、推理和符号是否全部正确
- [ ] 数学表达式是否全部使用 Markdown 可渲染 LaTeX：行内公式使用 `$...$`，多步推导使用 `$$...$$` 或 `aligned` 环境，是否不存在裸文本公式

## 顺序与异常

- [ ] 解答顺序是否与教材顺序完全一致
- [ ] 是否未按认知层级重新分组题目
- [ ] 正常题目是否未写审核状态或生成状态
- [ ] 仅题目残缺、图片不可用或条件无法确认时是否标记暂停
- [ ] 暂停题是否写明具体异常原因且没有猜测作答

## 最终outputs

- [ ] 文件名和存放目录是否符合skills定义
- [ ] YAML 元数据是否包含 `content_type: textbook_solution`
- [ ] 文件是否包含教材任务清单、参考解答和覆盖统计
- [ ] 是否未声称产物为出版社标准答案
