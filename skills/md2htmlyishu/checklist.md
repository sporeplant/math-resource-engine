# md2htmlyishu 检查清单

生成 HTML 后，逐项检查：

## 基础项

- [ ] 每页 `.slide` 宽度 1920px，高度 1080px
- [ ] 页面内容无溢出（无横向滚动条，底部无截断）
- [ ] 缩放功能正常（`fitSlides()` 正确执行）
- [ ] 内边距合理（左右 100px，上下 70px）
- [ ] LaTeX 公式正确渲染（无 `$` 源码残留，数学区间内无 HTML 标签）
- [ ] 多个编号数学小题使用 `.question-row` 横向排列，末两页作业清单除外
- [ ] 多图使用 `.img-row` 横向排列，两个容器均具备 `flex-wrap: wrap`
- [ ] 图片正常显示，无破图
- [ ] 图片路径正确（CDN URL 或相对路径）

## 行距项

- [ ] 普通正文行距为 2.0 倍
- [ ] 行间 LaTeX 公式（`display="true"`）行距为 2.4 倍，上下留白充足

## 封面项

- [ ] 封面背景为 `knowledge/images/bg/` 中的随机油画
- [ ] 封面仅显示课题与课时（`###` 标题行），无副标题、作者、日期等额外文字
- [ ] 封面无版权标注（无 `.cover-credit` 元素）

## 教学目标页

- [ ] 页面内容上下左右四向居中
- [ ] 正文列表序号左对齐，无错位
- [ ] 标题居中显示

## 多图与多题排列

- [ ] 多张连续图片默认横向排列
- [ ] 横向空间不足时自动纵向换行，无硬溢出
- [ ] 图片使用 `.img-item` 自适应布局，非固定 `.img-col`

## 风格项

- [ ] 背景色为 `#FDF6E3`（奶油黄）
- [ ] 标题 64px 粗体，底部有荧光黄涂抹效果
- [ ] 正文 40px，行高 2.0
- [ ] LaTeX 公式 44px，独立公式上下留白充足
- [ ] 连续公式推导步骤使用 `.formula-line`，段落间距为 36px
- [ ] 无卡片、圆角、阴影、渐变背景
- [ ] 无多余装饰元素

## 荧光笔标记

- [ ] 关键词使用 `.hl` 类，荧光黄半透明
- [ ] 荧光笔效果为底部涂抹，非全字高亮
- [ ] 无嵌套标记（如"样本容量"不被拆成"样本"+"容量"）

## 手写体

- [ ] 标题使用楷体/手写体，font-weight 900
- [ ] 正文使用楷体/手写体，font-weight 400
- [ ] 蓝笔批注使用 `.hw-blue`，颜色 `#1565C0`
- [ ] 红笔结论使用 `.hw-red`，颜色 `#C62828`，font-weight 700

## 红笔结论框

- [ ] 结论框 border 3px solid `#C62828`
- [ ] 结论框无背景色或极浅背景
- [ ] 结论文字为红色粗体
- [ ] 仅出现在真正的结论/口诀行，不误标正文步骤

## 表格

- [ ] 表格边框 2.5px solid `#1a1a2e`
- [ ] 表格文字 36px
- [ ] 表头加粗，无斑马纹
- [ ] 表格居中显示

## 蓝笔思路

- [ ] 蓝笔仅出现在"思路""引导"等明确提示处
- [ ] 不过度标记操作步骤

## 来源标注与页码

- [ ] 右下角来源标注，红色圆点前缀
- [ ] 底部居中页码 `— N —`
- [ ] 位置不遮挡正文内容

## 导航与交互

- [ ] 左下角无全屏按钮
- [ ] 右下角无导航圆点
- [ ] 键盘翻页正常（空格、方向键、PageUp/PageDown、Home、End）
- [ ] 单击页面翻下一页，右键返回上一页
- [ ] 黑屏功能正常（长按 PageDown 进入，长按任意翻页键退出）

## 整体感觉

- [ ] 像老师在黑板上手写板书
- [ ] 不像设计出来的 PPT 模板
- [ ] 信息层级靠字体、颜色、位置区分，不靠装饰元素
