# 检查清单

## 基础检查项

- [ ] 活动顺序是否严格遵循教材原文文本的先后顺序（练习和习题除外）
- [ ] 是否已读取并验证对应课时教材参考解答
- [ ] 是否已读取并验证对应课时练习册题库、答案和逐题索引
- [ ] 每个活动是否标注了 `教材对应位置`
- [ ] 活动是否指向学习目标
- [ ] 活动起点是否适配学生现有水平
- [ ] 台阶设置是否合理（小台阶推进，问题链交织）
- [ ] 是否有明确的分层设计
- [ ] 每个活动是否有反馈节点
- [ ] 是否避免了伪探究
- [ ] 活动产出是否可评价
- [ ] 是否仅使用了教材、练习册、练习本这三种教学材料
- [ ] 无"学习单""检测卡""投影展示""彩色笔"等额外物料

## 质量检查项

- [ ] 基础层活动类型是否匹配（观察/辨认/模仿/复述）
- [ ] 中间层活动类型是否匹配（解释/归纳/表达关系）
- [ ] 拓展层活动类型是否匹配（分析/探究/论证/评价）
- [ ] 同一主题下三层活动是否有内在关联
- [ ] 课堂练习/检测题目来源是否符合规则：教材练习栏目全部题目、习题A组选1题、习题B组选1题，总数 ≥ 4
- [ ] 课堂练习/检测环节安排是否合理（穿插反馈或集中限时）
- [ ] 每道课堂练习/检测题目是否标注了预估完成时间
- [ ] 每个活动是否标记 `time_priority: core | flex | backup`
- [ ] 每个活动是否按指令、学生作答、互动、反馈、转换分解 `time_budget`
- [ ] 时间是否使用0.5分钟精度，分项和是否等于活动标题时间
- [ ] 核心活动按班级画像修正后的预计耗时是否不超过40分钟
- [ ] 弹性和备用任务是否可以在超时时明确后移
- [ ] 未被选用的教材习题是否已标记为课下作业候选
- [ ] 教材题的预期回答、成功标准、反馈要点和答案是否与教材参考解答中同一 `question_id` 一致
- [ ] 所有被引用的教材 `question_id` 是否均存在于教材参考解答
- [ ] 所有被引用的练习册 `question_id` 是否均存在于 `knowledge/workbook-index/`
- [ ] 练习册题的预期回答、成功标准、反馈要点和答案是否按索引映射到答案文件
- [ ] 多道题共用同一题源时是否使用 `question_ids` 列表，且未重复题源字段或逗号拼接编号
- [ ] 是否避免了三层活动混在一起无区分
- [ ] 是否避免了用简单问答替代数学活动
- [ ] 每个 `ASK_*` 是否按“提问→分级备用提示→教师参考预期”排列，且提示只在学生停滞或答偏时使用
