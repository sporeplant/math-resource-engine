# 作业设计skills 检查清单

- [ ] 是否分层布置（基础层/中间层/拓展层）
- [ ] 基础层必做 ≤ 3 题
- [ ] 中间层必做 ≤ 2 题
- [ ] 拓展层选做 ≤ 1 题
- [ ] 每道题携带完整题源字段（source_id、source_type、question_id）
- [ ] 题目来源来自教材原文或练习册题库，未自行生成
- [ ] 教材题已按 question_id 与教材参考解答核对
- [ ] 练习册题已按 question_id 与 `knowledge/workbook-index/` 和答案文件核对
- [ ] 无 LLM 生成新题
- [ ] 作业量符合课时剩余时间
- [ ] 数学表达式是否使用 LaTeX 格式（`$...$ `），无裸文本公式

### 前台呈现检查

- [ ] 前台作业是否为 **必做**（2-3行）+ **选做**（1行），合计 4 行以内
- [ ] 前台仅含可定位信息（教材/练习册名称 + 栏目 + 题号 + 小题号），不出现题型说明、预计用时
- [ ] 前台不出现 `question_id`、`source_id`、`WB-*` 等后台字段
- [ ] 前台每条任务能在后台 `## homework` 中找到对应分层条目，无新增题目
- [ ] 分层明细引导语（`> 分层明细及预计用时见折叠区。`）仅在后文确有对应 `<details>` 区域时保留
