# 作业设计skills 检查清单

- [ ] 是否分层布置（基础层/中间层/拓展层）
- [ ] 基础层必做 ≤ 3 题
- [ ] 中间层必做 ≤ 2 题
- [ ] 拓展层选做 ≤ 1 题
- [ ] 每道题携带完整题源字段（source_id、source_type、question_id；练习册题另带 answer_id）
- [ ] 题目来源符合优先顺序（教材原文 → 练习册索引 → 练习册题库）
- [ ] 练习册补充题是否通过练习册索引按 `tier` 字段分层选取
- [ ] 开放答案题（`is_open_answer: true`）是否仅用于拓展层选做
- [ ] 教材题已按 question_id 与教材参考解答核对
- [ ] 无 LLM 生成新题
- [ ] 作业量符合课时剩余时间
