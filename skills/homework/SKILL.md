# 作业设计skills

## 1. 定位

负责根据学习目标、评价任务和知识分析结果，生成分层课后作业。

## 2. 输入

- 学习目标（来自 学习目标skills）
- 评价任务（来自 评价设计skills）
- 知识分析结果（来自 知识分析skills）
- 对应课时的教材练习与习题
- 练习册题库（来自 `knowledge/workbooks/`）
- 练习册参考答案（来自 `knowledge/workbook-answers/`）
- 练习册逐题索引（来自 `knowledge/workbook-index/`）
- 对应课时教材参考解答
- 教材-练习册资源盘点（来自 `resource_audit`）

## 3. outputs要求

分层作业：
- 基础层必做：约20分钟
- 中间层必做：约25分钟
- 拓展层选做：增加1道题
- 每层必须标注预计完成时间

## 4. 选题规则

- 每道题必须携带题源字段（`source_id`、`source_type`、`question_id`）
- 必须先依据 `orchestrator/resource-scheduling.md` 完成资源盘点，再从候选题中选作业题
- 从教材原文和练习册题库共同选题：
  - 练习栏目未被课堂练习选用的剩余题目
  - 习题A组未被课堂练习选用的题目
  - 习题B组未被课堂练习选用的题目
  - 练习册中与本课目标匹配、课堂未使用且答案可映射的题目
- 标注预估完成时间
- 不默认布置整页练习册；练习册未布置题必须写入 `deferred_exercises`，标注后移到习题课、讲评课、周练、单元整理或暂不处理的理由

## 5. 教材参考解答核对

- 教材题的预期回答、成功标准、反馈要点和答案必须按 `question_id` 与教材参考解答核对
- 不得自行改写数学结论或答案边界
- 教材参考解答的 `答案来源` 为 `AI参考推导` 或 `AI参考推导补充` 时，不得表述为教材原文

## 5a. 练习册答案核对

- 练习册题必须按 `question_id` 从 `knowledge/workbook-index/` 定位题目和答案位置
- 答案文件中为“略”“答案不唯一”等开放题边界时，作业说明不得改成唯一答案
- 缺少练习册题库、答案或索引时，不得布置该练习册题

## 6. 禁止事项

- 禁止使用 LLM 生成新题
- 禁止无题源字段的题目进入作业
- 禁止不做分层直接布置统一作业
- 禁止跳过教材参考解答核对直接出题
- 禁止跳过练习册逐题索引和答案核对直接布置练习册题
