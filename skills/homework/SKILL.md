# 作业设计skills

## 1. 定位

负责根据学习目标、评价任务和知识分析结果，生成分层课后作业。

## 2. 输入

- 学习目标（来自 学习目标skills）
- 评价任务（来自 评价设计skills）
- 知识分析结果（来自 知识分析skills）
- 对应课时的教材练习与习题
- 练习册题库（来自 `knowledge/workbooks/`）
- 对应课时教材参考解答

## 3. outputs要求

分层作业：
- 基础层必做 ≤ 3 题
- 中间层必做 ≤ 2 题
- 拓展层选做 ≤ 1 题

## 4. 选题规则

- 每道题必须携带题源字段（`source_id`、`source_type`、`question_id`）
- 优先从教材原文选题：
  - 练习栏目未被课堂练习选用的剩余题目
  - 习题A组未被课堂练习选用的题目
  - 习题B组未被课堂练习选用的题目
- 不足时从练习册题库补充
- 标注预估完成时间

## 5. 教材参考解答核对

- 教材题的预期回答、成功标准、反馈要点和答案必须按 `question_id` 与教材参考解答核对
- 不得自行改写数学结论或答案边界
- 教材参考解答的 `答案来源` 为 `AI参考推导` 或 `AI参考推导补充` 时，不得表述为教材原文

## 6. 禁止事项

- 禁止使用 LLM 生成新题
- 禁止无题源字段的题目进入作业
- 禁止不做分层直接布置统一作业
- 禁止跳过教材参考解答核对直接出题
