# 二、总结与反思

本章学习了常量和变量、函数及其表示、画函数图象的一般步骤等内容。在本章的学习过程中，利用建立函数模型的方法解决了变化过程中的相关问题，突出体现了函数思想以及数形结合思想方法的应用，有利于发展抽象能力，培养应用意识。 

# 1. 常量和变量.

在一个变化过程中，数值保持不变的量是常量，可以取不同数值的量是变量. 

# 2. 函数.

在函数概念中, 特别强调: 自变量的取值范围; 当自变量取定一个数值时, 对应的函数值唯一确定. 

# 3. 函数的表示.

可以用数值表、图象和表达式来表示两个变量之间的函数关系，它们分别表现出具体、形象直观和便于抽象应用的特点. 

# 4. 画函数图象的一般步骤:

(1) 列表; (2) 描点; (3) 连线. 

# 三、注意事项

在研究函数的问题时，确定自变量的取值范围应注意以下两点： 

(1) 自变量的取值要使函数表达式有意义; 

(2) 自变量的取值要符合实际问题。
