# 第21章知识图谱

```yaml
## 本章知识体系结构图

```
                                     四边形
                                   /        \
                                  /          \
                           平行四边形          梯形
                           /    |    \        /    \
                          /     |     \      /      \
                      矩形    菱形   (一般) 等腰梯形  直角梯形
                        \     /
                         \   /
                         正方形
```

## 对角线性质递进关系

```
一般四边形      平行四边形        矩形            菱形           正方形
无特殊性质  →   互相平分    →  平分且相等    →  垂直平分    →  垂直平分且相等
                                                                    ↑
                                                     矩形(相等) + 菱形(垂直)
```

## 判定方法分类线索

```
判定起点有两条路径：
  路径A：以四边形为起点 → 直接判定为特殊四边形
  路径B：以平行四边形为起点 → 加一个条件升级为更特殊的四边形

例（矩形）：
  路径A：三个角是直角的四边形 → 矩形
  路径B：平行四边形 + 一个直角（或对角线相等） → 矩形

例（菱形）：
  路径A：四条边相等的四边形 → 菱形
  路径B：平行四边形 + 一组邻边相等（或对角线垂直） → 菱形
```

## 易混点标注

```
1. 一组对边平行 + 另一组对边相等 ≠ 平行四边形
   （正确：一组对边平行且相等 或 两组对边分别相等）

2. 对角线相等 + 不一定 → 矩形
   （必须先保证是平行四边形，再用对角线相等推矩形）

3. 对角线垂直 + 不一定 → 菱形
   （必须先保证是平行四边形，再用对角线垂直推菱形）

4. 对角线垂直且相等 + 不一定 → 正方形
   （缺少"互相平分"条件时，反例：两条线段垂直相等但任意相交）

5. 梯形的判定强调"只有一组对边平行"
   （若不加"只有"，则平行四边形也被归为梯形，与定义矛盾）
```
