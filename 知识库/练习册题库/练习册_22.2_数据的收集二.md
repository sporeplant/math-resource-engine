# 22.2 数据的收集(二)


# 知识点拨

抽样调查只考察了总体中的一部分个体，调查结果不如普查准确。为了得到较为准确的结果，调查的个体不能太少。

# 数学思考

3. 某校计划调查全校学生的视力情况,现有各班报送的全体学生的视力情况材料.学校从中选取了其中一个班的数据进行整理,以反映全校情况. 

(1)该校采用的这种抽样调查方式是否合适？为什么？ 

(2)上述方法与从各班中随机抽取5名同学的视力情况进行调查相比，哪种调查方式得到的结果更可靠？ 

4. 某市教育主管部门为了解全市各校八年级学生的学习情况，让每个学校选出年级排名为前20名的学生参加学习竞赛。这样作抽样调查是否合适？怎样选取样本比较有代表性？
