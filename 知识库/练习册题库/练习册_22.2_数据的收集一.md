# 22.2 数据的收集(一)


# 知识点拨

1. 普查是指对全体对象进行调查. 

2. 总体是指要考察对象的全体，个体是指组成总体的每一个对象. 

3. 从总体中抽取部分个体进行调查的方式叫作抽样调查。能保证总体中每个个体有相同的机会被抽到的抽样调查称为简单随机抽样。 

4. 从总体中抽取并进行调查的部分个体叫作总体的一个样本。样本中包含个体的数目叫作样本容量。

# 2. 填空题.

(1)某市今年有7万名考生参加中考,为了解这7万名考生的数学成绩,该市教研室进行了一次摸底考试,从中抽取了1500名考生的数学成绩进行分析. 在这个问题中,总体是_________,个体是_____. 

(2)为了解小区内 236 户家庭对小区公共区域环境是否满意，小明利用周末调查了其中的 50 户家庭，有 46 户表示满意。在这一调查中，样本容量为____。 

(3)下列调查中, 适合采用抽样调查的是____。(填序号) 

①调查某市中学生对 2025 年中国航天日主题 “海上生明月，九天揽星河” 的了解 情况；②检测某批节能灯的使用寿命；③某体育运动学校通过调查选出花样滑冰成绩最好的学生参加全国比赛. 

(4)为了解一本300页的书稿大约有多少字，可以从中随机选择一页，数一数该页的字数。下列说法中，正确的是____。（填序号） 

①300页书稿的字数是总体；②每页书稿是个体；③从书稿中选定的那页的字数是总体的一个样本；④300是样本容量.
