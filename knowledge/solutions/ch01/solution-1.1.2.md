**原题**：某水库一监测点将水深为 $5 \, m$ 处的水面设定为警戒水位，规定超过警戒水位的部分记为正，低于警戒水位的部分记为负。(1) "$+1$ m" 表示什么意义？此时水库监测点的实际水深是多少米？

**参考解答**：

"$+1$ m" 表示高于警戒水位 $1$ m（即超过警戒水位的部分为正数 $+1$）。

此时水库监测点的实际水深为 $5 + 1 = 6$ m。

```yaml

```

**原题**：根据有理数的意义，我们知道有理数可作如下分类：

$$
\text {有理数} \left\{ \begin{array}{l l} \text {整数} \\ \text {分数} \end{array} \right.
$$

你能进一步将整数和分数分类吗？有理数还有其他分类方法吗？请把你的想法与同学交流一下。

**参考解答**：

**（一）将整数和分数进一步分类：**

$$
\text{有理数}
\begin{cases}
\text{整数}
\begin{cases}
\text{正整数（如 } 1, 2, 3, \ldots）\\
0\\
\text{负整数（如 } -1, -2, -3, \ldots）
\end{cases}\\[8pt]
\text{分数}
\begin{cases}
\text{正分数（如 } \dfrac{1}{2}, \dfrac{1}{3}, 3\dfrac{1}{2}, \ldots）\\
\text{负分数（如 } -\dfrac{1}{4}, -\dfrac{22}{7}, -8\dfrac{3}{4}, \ldots）
\end{cases}
\end{cases}
$$

**（二）有理数的另一种分类方法——按符号（正负性）分类：**

$$
\text{有理数}
\begin{cases}
\text{正有理数}
\begin{cases}
\text{正整数}\\
\text{正分数}
\end{cases}\\[8pt]
0\\[8pt]
\text{负有理数}
\begin{cases}
\text{负整数}\\
\text{负分数}
\end{cases}
\end{cases}
$$

**合理答案边界**：学生能从"整分"或"正负"两个维度之一完成分类即符合要求。两种分类方法对应于不同的分类标准（按是否为分数分类，或按符号分类），是同一集合的不同划分方式。


**原题**：通过查阅资料、交流研讨，了解把无限循环小数 $1.\dot{2}\dot{7}$ 化为分数的方法。

**参考解答**：

本题为探究题，要求学生通过查阅资料或交流获取方法。以下给出一种典型解法供参考：

设 $x = 1.272727\ldots$（即 $1.\dot{2}\dot{7}$）。

则 $100x = 127.272727\ldots$

两式相减：

$$
\begin{aligned}
100x - x &= 127.272727\ldots - 1.272727\ldots \\
99x &= 126 \\
x &= \frac{126}{99} = \frac{14}{11}
\end{aligned}
$$

因此 $1.\dot{2}\dot{7} = \dfrac{14}{11}$。

**方法要点**：将无限循环小数化为分数的一般方法是：设该小数为 $x$，两边同乘 $10^{n}$（$n$ 为循环节长度）后相减消去循环部分，解方程即得。

**合理答案边界**：学生能通过自主查阅获得正确方法并验证即可，不要求独立推导。若学生未使用方程法而使用其他合理方法（如无穷等比数列求和）且结果正确，同样符合要求。

### 习题C组第6题第(2)问

```yaml
question_id: "1.1.2-习题C-6-2"
source_id: "教材原文_1.1.2_正数和负数"
source_type: textbook
教材位置: "习题C组第6题第(2)问"
教材顺序: 16
任务类型: 应用题
认知层级: 拓展层
答案来源: AI参考推导
```

**原题**：(2) 另一个零件的标注为 $\Phi \_\_^{+0.4}_{-0.4}$，该零件直径的标准尺寸有些模糊。已知该零件的七个合格产品的直径分别为 $73.1 \mathrm{~mm}$，$72.7 \mathrm{~mm}$，$72.8 \mathrm{~mm}$，$73.2 \mathrm{~mm}$，$72.9 \mathrm{~mm}$，$73.3 \mathrm{~mm}$，$72.6 \mathrm{~mm}$，则该零件直径的标准尺寸是多少毫米？（注：标准尺寸为整毫米数）

**参考解答**：

标注 $\Phi \_\_^{+0.4}_{-0.4}$ 表示上偏差为 $+0.4$ mm，下偏差为 $-0.4$ mm。设标准尺寸为 $d$（整毫米数），则合格产品的直径 $D$ 应满足：

$$d - 0.4 \leq D \leq d + 0.4$$

即合格直径在 $[d - 0.4, d + 0.4]$ 范围内。

七个合格直径：$73.1$，$72.7$，$72.8$，$73.2$，$72.9$，$73.3$，$72.6$。

最大值为 $73.3$ mm，最小值为 $72.6$ mm。

由 $d - 0.4 \leq 72.6$ 得 $d \leq 73.0$；
由 $73.3 \leq d + 0.4$ 得 $d \geq 72.9$。

因此 $72.9 \leq d \leq 73.0$，又 $d$ 为整毫米数，所以 $d = 73$ mm。

验证：$d = 73$ 时，合格范围为 $[72.6, 73.4]$，七个直径全部在此范围内。✓

答：该零件直径的标准尺寸是 $73$ mm。
