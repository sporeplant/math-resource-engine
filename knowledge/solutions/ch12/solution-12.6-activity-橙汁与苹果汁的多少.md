**原题**：有甲、乙两个杯子，分别盛了体积相同的橙汁与苹果汁。将这两种果汁进行部分混合，把甲杯中的苹果汁与橙汁的体积比记为 $m$，乙杯中的橙汁与苹果汁的体积比记为 $n$。通过操作和计算，比较 $m,n$ 的大小。

活动一：将甲杯中的橙汁全部倒入乙杯，并搅拌均匀，再将乙杯中混合果汁的一半倒入甲杯，比较m,n的大小。

**参考解答**：

设甲、乙两杯最初各盛 $V\ \mathrm{mL}$ 果汁。

初始状态：
- 甲杯：$V\ \mathrm{mL}$ 橙汁，$0\ \mathrm{mL}$ 苹果汁
- 乙杯：$0\ \mathrm{mL}$ 橙汁，$V\ \mathrm{mL}$ 苹果汁

操作步骤：

1. 将甲杯中的橙汁全部倒入乙杯。
   - 甲杯：$0\ \mathrm{mL}$ 橙汁，$0\ \mathrm{mL}$ 苹果汁（空杯）
   - 乙杯：$V\ \mathrm{mL}$ 橙汁，$V\ \mathrm{mL}$ 苹果汁（总量 $2V\ \mathrm{mL}$）

2. 将乙杯中混合果汁的一半倒入甲杯。
   乙杯混合果汁中橙汁与苹果汁各占一半，倒入的 $V\ \mathrm{mL}$ 中
   含 $\dfrac{V}{2}\ \mathrm{mL}$ 橙汁和 $\dfrac{V}{2}\ \mathrm{mL}$ 苹果汁。
   - 甲杯：$\dfrac{V}{2}\ \mathrm{mL}$ 橙汁，$\dfrac{V}{2}\ \mathrm{mL}$ 苹果汁
   - 乙杯：$\dfrac{V}{2}\ \mathrm{mL}$ 橙汁，$\dfrac{V}{2}\ \mathrm{mL}$ 苹果汁

计算：
- $m = \dfrac{\text{甲杯苹果汁}}{\text{甲杯橙汁}} = \dfrac{V/2}{V/2} = 1$
- $n = \dfrac{\text{乙杯橙汁}}{\text{乙杯苹果汁}} = \dfrac{V/2}{V/2} = 1$

结论：$m = n = 1$。

**原题**：甲、乙两个杯子中原来的橙汁与苹果汁均为 $a\ \mathrm{mL}$。准备一个小勺，一勺果汁的体积为 $b\ \mathrm{mL}$，且 $b$ 小于 $a$。
第一次操作：先从甲杯中取出一勺橙汁倒入乙杯，并搅拌均匀，再从乙杯中取出一勺混合果汁倒入甲杯。
第二次操作：在第一次操作的基础上，先从甲杯中取出一勺混合果汁倒入乙杯，并搅拌均匀，再从乙杯中取出一勺混合果汁倒入甲杯。
根据上述操作，填写下表。

**参考解答**：

初始状态：
- 甲杯：$a\ \mathrm{mL}$ 橙汁，$0\ \mathrm{mL}$ 苹果汁
- 乙杯：$0\ \mathrm{mL}$ 橙汁，$a\ \mathrm{mL}$ 苹果汁
- $m_0 = \dfrac{0}{a} = 0$，$n_0 = \dfrac{0}{a} = 0$，$m_0 = n_0$

**第一次操作：**

Step 1：从甲杯取一勺橙汁（$b\ \mathrm{mL}$）倒入乙杯。
- 甲杯：$(a-b)\ \mathrm{mL}$ 橙汁，$0\ \mathrm{mL}$ 苹果汁
- 乙杯：$b\ \mathrm{mL}$ 橙汁，$a\ \mathrm{mL}$ 苹果汁（总量 $(a+b)\ \mathrm{mL}$）

Step 2：从乙杯取一勺混合果汁（$b\ \mathrm{mL}$）倒入甲杯。
乙杯混合果汁中，橙汁比例 $\dfrac{b}{a+b}$，苹果汁比例 $\dfrac{a}{a+b}$。
取出 $b\ \mathrm{mL}$ 中含 $\dfrac{b^2}{a+b}\ \mathrm{mL}$ 橙汁，$\dfrac{ab}{a+b}\ \mathrm{mL}$ 苹果汁。

- 甲杯：
  橙汁：$(a-b) + \dfrac{b^2}{a+b} = \dfrac{a^2}{a+b}$（mL）
  苹果汁：$0 + \dfrac{ab}{a+b} = \dfrac{ab}{a+b}$（mL）

- 乙杯：
  橙汁：$b - \dfrac{b^2}{a+b} = \dfrac{ab}{a+b}$（mL）
  苹果汁：$a - \dfrac{ab}{a+b} = \dfrac{a^2}{a+b}$（mL）

第一次操作后：
$m_1 = \dfrac{ab/(a+b)}{a^2/(a+b)} = \dfrac{b}{a}$，
$n_1 = \dfrac{ab/(a+b)}{a^2/(a+b)} = \dfrac{b}{a}$，
$m_1 = n_1 = \dfrac{b}{a}$。

**第二次操作：**

第二次操作开始前：
- 甲杯：$\dfrac{a^2}{a+b}\ \mathrm{mL}$ 橙汁，$\dfrac{ab}{a+b}\ \mathrm{mL}$ 苹果汁（总量 $a\ \mathrm{mL}$）
- 乙杯：$\dfrac{ab}{a+b}\ \mathrm{mL}$ 橙汁，$\dfrac{a^2}{a+b}\ \mathrm{mL}$ 苹果汁（总量 $a\ \mathrm{mL}$）

Step 1：从甲杯取一勺混合果汁（$b\ \mathrm{mL}$）倒入乙杯。
甲杯混合果汁中，橙汁比例 $\dfrac{a}{a+b}$，苹果汁比例 $\dfrac{b}{a+b}$。
取出 $b\ \mathrm{mL}$ 中含 $\dfrac{ab}{a+b}\ \mathrm{mL}$ 橙汁，$\dfrac{b^2}{a+b}\ \mathrm{mL}$ 苹果汁。

- 甲杯：
  橙汁：$\dfrac{a^2}{a+b} - \dfrac{ab}{a+b} = \dfrac{a(a-b)}{a+b}$（mL）
  苹果汁：$\dfrac{ab}{a+b} - \dfrac{b^2}{a+b} = \dfrac{b(a-b)}{a+b}$（mL）
  总量：$\dfrac{(a+b)(a-b)}{a+b} = a-b$（mL）

- 乙杯：
  橙汁：$\dfrac{ab}{a+b} + \dfrac{ab}{a+b} = \dfrac{2ab}{a+b}$（mL）
  苹果汁：$\dfrac{a^2}{a+b} + \dfrac{b^2}{a+b} = \dfrac{a^2+b^2}{a+b}$（mL）
  总量：$\dfrac{2ab + a^2 + b^2}{a+b} = \dfrac{(a+b)^2}{a+b} = a+b$（mL）

Step 2：从乙杯取一勺混合果汁（$b\ \mathrm{mL}$）倒入甲杯。
乙杯混合果汁中，橙汁比例 $\dfrac{2ab}{(a+b)^2}$，苹果汁比例 $\dfrac{a^2+b^2}{(a+b)^2}$。
取出 $b\ \mathrm{mL}$ 中含 $\dfrac{2ab^2}{(a+b)^2}\ \mathrm{mL}$ 橙汁，$\dfrac{b(a^2+b^2)}{(a+b)^2}\ \mathrm{mL}$ 苹果汁。

- 甲杯：
  橙汁：
  $$
  \dfrac{a(a-b)}{a+b} + \dfrac{2ab^2}{(a+b)^2}
  = \dfrac{a(a-b)(a+b) + 2ab^2}{(a+b)^2}
  = \dfrac{a(a^2+b^2)}{(a+b)^2}
  $$
  （$\mathrm{mL}$）
  苹果汁：
  $$
  \dfrac{b(a-b)}{a+b} + \dfrac{b(a^2+b^2)}{(a+b)^2}
  = \dfrac{b(a-b)(a+b) + b(a^2+b^2)}{(a+b)^2}
  = \dfrac{2a^2b}{(a+b)^2}
  $$
  （$\mathrm{mL}$）

- 乙杯：
  橙汁：
  $$
  \dfrac{2ab}{a+b} - \dfrac{2ab^2}{(a+b)^2}
  = \dfrac{2ab(a+b) - 2ab^2}{(a+b)^2}
  = \dfrac{2a^2b}{(a+b)^2}
  $$
  （$\mathrm{mL}$）
  苹果汁：
  $$
  \dfrac{a^2+b^2}{a+b} - \dfrac{b(a^2+b^2)}{(a+b)^2}
  = \dfrac{(a^2+b^2)(a+b) - b(a^2+b^2)}{(a+b)^2}
  = \dfrac{a(a^2+b^2)}{(a+b)^2}
  $$
  （$\mathrm{mL}$）

第二次操作后：
$m_2 = \dfrac{2a^2b/(a+b)^2}{a(a^2+b^2)/(a+b)^2} = \dfrac{2ab}{a^2+b^2}$
$n_2 = \dfrac{2a^2b/(a+b)^2}{a(a^2+b^2)/(a+b)^2} = \dfrac{2ab}{a^2+b^2}$
$m_2 = n_2 = \dfrac{2ab}{a^2+b^2}$。

**表格结果：**

| 行 | 混合前的体积（橙汁/苹果汁） | 第一次操作后的体积（橙汁/苹果汁） | 第二次操作后的体积（橙汁/苹果汁） |
|---|---|---|---|
| 甲杯 | $a$ / $0$ | $\dfrac{a^2}{a+b}$ / $\dfrac{ab}{a+b}$ | $\dfrac{a(a^2+b^2)}{(a+b)^2}$ / $\dfrac{2a^2b}{(a+b)^2}$ |
| 乙杯 | $0$ / $a$ | $\dfrac{ab}{a+b}$ / $\dfrac{a^2}{a+b}$ | $\dfrac{2a^2b}{(a+b)^2}$ / $\dfrac{a(a^2+b^2)}{(a+b)^2}$ |
| $m,n$ 的大小关系 | $m=n=0$ | $m=n=\dfrac{b}{a}$ | $m=n=\dfrac{2ab}{a^2+b^2}$ |

**原题**：观察表中数据，你有什么猜想？说明你的猜想的正确性。

**参考解答**：

**猜想**：在每次完整的操作之后，恒有 $m = n$，即甲杯中苹果汁与橙汁的体积比始终等于乙杯中橙汁与苹果汁的体积比。

**正确性说明**：

从计算结果观察：
- 混合前：$m = 0$，$n = 0$，$m = n$。
- 第一次操作后：$m = \dfrac{b}{a}$，$n = \dfrac{b}{a}$，$m = n$。
- 第二次操作后：$m = \dfrac{2ab}{a^2+b^2}$，$n = \dfrac{2ab}{a^2+b^2}$，$m = n$。

一般性证明：

设一次完整操作之后，甲杯中有橙汁 $O\ \mathrm{mL}$、苹果汁 $A\ \mathrm{mL}$，乙杯中有橙汁 $(a-O)\ \mathrm{mL}$、苹果汁 $(a-A)\ \mathrm{mL}$（因总量守恒）。则
$$
m = \frac{A}{O},\qquad n = \frac{a-O}{a-A}.
$$

要证 $m = n$，即证
$$
\frac{A}{O} = \frac{a-O}{a-A}.
$$

交叉相乘得
$$
A(a-A) = O(a-O),
$$
整理得
$$
aA - A^2 = aO - O^2,
$$
即
$$
a(A-O) = (A-O)(A+O).
$$

若 $A \neq O$，两边同除以 $A-O$ 得
$$
a = A + O,
$$
这正是一次完整操作后甲杯总量恒为 $a\ \mathrm{mL}$ 的条件，成立。

若 $A = O$，则 $m = n = 1$，等式也成立。

因此，无论进行多少次操作，每次完整操作后恒有 $m = n$。猜想成立。

**结论**：$m = n$ 是每次完整操作后的恒等式。
