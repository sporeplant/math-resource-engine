**原题**：当 $d \neq 0$, $b + c \neq 0$ 时，分式 $\dfrac{ab+ac}{bd+cd}$ 能不能化简？如果能，那么化简的依据是什么，化简的结果又是什么？

**参考解答**：分式 $\dfrac{ab+ac}{bd+cd}$ 可以化简。

化简过程：
- 分子分解因式：$ab+ac = a(b+c)$
- 分母分解因式：$bd+cd = d(b+c)$
- 原式 $= \dfrac{a(b+c)}{d(b+c)} = \dfrac{a}{d}$

化简的依据是分式的基本性质——分子和分母同除以不为零的整式 $(b+c)$，分式的值不变。

化简的结果为 $\dfrac{a}{d}$。

### 做一做

```yaml

```

**原题**：当 $p = 12$，$q = -8$ 时，请分别用直接代入求值和化简后代入求值两种方法求分式 $\dfrac{p^2-pq}{p^2-2pq+q^2}$ 的值，并比较哪种方法较简单。

**参考解答**：

**方法一（直接代入求值）：**
- 分子：$p^2 - pq = 12^2 - 12 \times (-8) = 144 + 96 = 240$
- 分母：$p^2 - 2pq + q^2 = 12^2 - 2 \times 12 \times (-8) + (-8)^2 = 144 + 192 + 64 = 400$
- 原式 $= \dfrac{240}{400} = \dfrac{3}{5}$

**方法二（化简后代入求值）：**
- 原式 $= \dfrac{p(p-q)}{(p-q)^2} = \dfrac{p}{p-q}$（由 $p \neq q$，可约去 $p-q$）
- 代入 $p = 12$，$q = -8$：$\dfrac{12}{12-(-8)} = \dfrac{12}{20} = \dfrac{3}{5}$

**比较**：化简后代入求值的方法较简单（计算步骤更少，数值更小）。

**原题**：已知 $x$, $y$ 满足 $|x+1| + (y-2)^2 = 0$。求 $\dfrac{x^2-2xy+y^2}{(x-y)^3}$ 的值。

**参考解答**：
- 由非负性：$|x+1| \geq 0$，$(y-2)^2 \geq 0$，且和为 $0$，得：
  - $x+1 = 0 \Rightarrow x = -1$
  - $y-2 = 0 \Rightarrow y = 2$
- 化简分式：$\dfrac{x^2-2xy+y^2}{(x-y)^3} = \dfrac{(x-y)^2}{(x-y)^3} = \dfrac{1}{x-y}$（$x \neq y$）
- 代入 $x = -1$, $y = 2$：$\dfrac{1}{-1-2} = -\dfrac{1}{3}$
