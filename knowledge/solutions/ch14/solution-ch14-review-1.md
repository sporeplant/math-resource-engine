**原题**：求下列各数的算术平方根和平方根：（1）36；（2）0.64；（3）$\frac{25}{16}$；（4）7；（5）$\frac{9}{2}$；（6）$\left(-\frac{2}{3}\right)^2$。

**解答**：
| | 算术平方根 | 平方根 |
|---|---|---|
| 36 | 6 | $\pm 6$ |
| 0.64 | 0.8 | $\pm 0.8$ |
| $\frac{25}{16}$ | $\frac{5}{4}$ | $\pm\frac{5}{4}$ |
| 7 | $\sqrt{7}$ | $\pm\sqrt{7}$ |
| $\frac{9}{2}$ | $\frac{3}{\sqrt{2}}=\frac{3\sqrt{2}}{2}$ | $\pm\frac{3\sqrt{2}}{2}$ |
| $\left(-\frac{2}{3}\right)^2$ | $\frac{2}{3}$ | $\pm\frac{2}{3}$ |


**原题**：圆柱体积 $54\pi \mathrm{m}^3$，底面圆直径与高相等。求底面半径。

**解答**：设底面半径为 $r$，则高 $h=2r$。$\pi r^2\times2r=54\pi$，$2r^3=54$，$r^3=27$，$r=3\mathrm{m}$。
