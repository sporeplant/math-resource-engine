**原题**：判断下列各式的计算结果是否正确：
（1）$\sqrt{7} + \sqrt{3} = \sqrt{10}$；（2）$\frac{\sqrt{8}}{\sqrt{18}} = \frac{2}{3}$；
（3）$2\sqrt{3} \times 3\sqrt{2} = 6\sqrt{6}$；（4）$\sqrt{3} - \sqrt{2} = 1$。

**解答**：
- （1）**错误**。$\sqrt{7} + \sqrt{3}$ 不能合并。$\sqrt{7} + \sqrt{3} \approx 2.646 + 1.732 = 4.378$，而 $\sqrt{10} \approx 3.162$。
- （2）**正确**。$\frac{\sqrt{8}}{\sqrt{18}} = \sqrt{\frac{8}{18}} = \sqrt{\frac{4}{9}} = \frac{2}{3}$。
- （3）**正确**。$2\sqrt{3} \times 3\sqrt{2} = 6\sqrt{6}$。
- （4）**错误**。$\sqrt{3} - \sqrt{2} \approx 1.732 - 1.414 = 0.318 \neq 1$。

**原题**：计算下列各式：
（1）$\sqrt{48} + 2\sqrt{3} - \sqrt{75}$；
（2）$\frac{1}{2}\sqrt{3} + \sqrt{\frac{3}{4}} - 6\sqrt{\frac{1}{3}}$；
（3）$2\sqrt{8} - 5\sqrt{32} - 3\sqrt{5}$；
（4）$\sqrt{40} - 2\sqrt{\frac{1}{10}} + \sqrt{90}$。

**解答**：
- （1）$\sqrt{48} + 2\sqrt{3} - \sqrt{75} = 4\sqrt{3} + 2\sqrt{3} - 5\sqrt{3} = \sqrt{3}$。
- （2）$\frac{1}{2}\sqrt{3} + \sqrt{\frac{3}{4}} - 6\sqrt{\frac{1}{3}} = \frac{1}{2}\sqrt{3} + \frac{\sqrt{3}}{2} - 6 \times \frac{\sqrt{3}}{3} = \sqrt{3} - 2\sqrt{3} = -\sqrt{3}$。

- （4）$\sqrt{40} - 2\sqrt{\frac{1}{10}} + \sqrt{90} = 2\sqrt{10} - 2 \times \frac{\sqrt{10}}{10} + 3\sqrt{10} = 2\sqrt{10} - \frac{\sqrt{10}}{5} + 3\sqrt{10} = \frac{10\sqrt{10}}{5} - \frac{\sqrt{10}}{5} + \frac{15\sqrt{10}}{5} = \frac{24\sqrt{10}}{5}$。

**原题**：计算下列各式：
（1）$\sqrt{84} \times (-\sqrt{21})$；（2）$2\sqrt{18} \div \sqrt{32}$；
（3）$\sqrt{5} \div \sqrt{15} \times \sqrt{30}$；（4）$\sqrt{72} \div 3\sqrt{\frac{1}{2}} \times \sqrt{12}$；
（5）$(3\sqrt{5} + 2\sqrt{3})(3\sqrt{5} - 2\sqrt{3})$；（6）$(\sqrt{48} + \sqrt{27} - \sqrt{12}) \div \sqrt{6}$。

**解答**：
- （1）$\sqrt{84} \times (-\sqrt{21}) = -\sqrt{84 \times 21} = -\sqrt{1764} = -42$。
- （2）$2\sqrt{18} \div \sqrt{32} = 2\sqrt{\frac{18}{32}} = 2\sqrt{\frac{9}{16}} = 2 \times \frac{3}{4} = \frac{3}{2}$。
- （3）$\sqrt{5} \div \sqrt{15} \times \sqrt{30} = \sqrt{\frac{5}{15} \times 30} = \sqrt{10}$。
- （4）$\sqrt{72} \div 3\sqrt{\frac{1}{2}} \times \sqrt{12} = \frac{6\sqrt{2}}{3 \times \frac{1}{\sqrt{2}}} \times 2\sqrt{3} = \frac{6\sqrt{2} \times \sqrt{2}}{3} \times 2\sqrt{3} = 4 \times 2\sqrt{3} = 8\sqrt{3}$。
- （5）$(3\sqrt{5} + 2\sqrt{3})(3\sqrt{5} - 2\sqrt{3}) = (3\sqrt{5})^2 - (2\sqrt{3})^2 = 45 - 12 = 33$。
- （6）$(\sqrt{48} + \sqrt{27} - \sqrt{12}) \div \sqrt{6} = (4\sqrt{3} + 3\sqrt{3} - 2\sqrt{3}) \div \sqrt{6} = 5\sqrt{3} \div \sqrt{6} = 5\sqrt{\frac{3}{6}} = 5\sqrt{\frac{1}{2}} = \frac{5\sqrt{2}}{2}$。

```yaml
认知层级: 拓展层

```

**原题**：计算下列各式：
（1）$\left(\sqrt{\frac{3}{2}} - \sqrt{\frac{1}{2}}\right)\left(\sqrt{8} - \sqrt{\frac{2}{3}}\right)$；
（2）$\left(\sqrt{1\frac{2}{3}} - \sqrt{1\frac{1}{4}}\right)^2$；
（3）$(\sqrt{50} + \sqrt{32}) \div \left(\sqrt{2} - \sqrt{\frac{1}{8}}\right)$；
（4）$(\sqrt{63} - \sqrt{28}) \div \left(\sqrt{\frac{7}{4}} + \sqrt{7}\right)$。

**解答**：
- （1）$\left(\sqrt{\frac{3}{2}} - \sqrt{\frac{1}{2}}\right)\left(\sqrt{8} - \sqrt{\frac{2}{3}}\right)$
$= \left(\frac{\sqrt{6}}{2} - \frac{\sqrt{2}}{2}\right)\left(2\sqrt{2} - \frac{\sqrt{6}}{3}\right)$
$= \frac{\sqrt{6} - \sqrt{2}}{2} \times \frac{6\sqrt{2} - \sqrt{6}}{3}$
$= \frac{(\sqrt{6} - \sqrt{2})(6\sqrt{2} - \sqrt{6})}{6}$
$= \frac{\sqrt{6} \times 6\sqrt{2} - \sqrt{6} \times \sqrt{6} - \sqrt{2} \times 6\sqrt{2} + \sqrt{2} \times \sqrt{6}}{6}$
$= \frac{6\sqrt{12} - 6 - 12 + \sqrt{12}}{6} = \frac{7\sqrt{12} - 18}{6} = \frac{14\sqrt{3} - 18}{6} = \frac{7\sqrt{3} - 9}{3}$。

- （2）$\left(\sqrt{1\frac{2}{3}} - \sqrt{1\frac{1}{4}}\right)^2$
$= \left(\sqrt{\frac{5}{3}} - \sqrt{\frac{5}{4}}\right)^2 = \left(\frac{\sqrt{15}}{3} - \frac{\sqrt{5}}{2}\right)^2$
$= \left(\frac{\sqrt{15}}{3}\right)^2 - 2 \times \frac{\sqrt{15}}{3} \times \frac{\sqrt{5}}{2} + \left(\frac{\sqrt{5}}{2}\right)^2$
$= \frac{15}{9} - \frac{2\sqrt{75}}{6} + \frac{5}{4} = \frac{5}{3} - \frac{2 \times 5\sqrt{3}}{6} + \frac{5}{4}$
$= \frac{5}{3} - \frac{5\sqrt{3}}{3} + \frac{5}{4} = \frac{20}{12} - \frac{20\sqrt{3}}{12} + \frac{15}{12} = \frac{35 - 20\sqrt{3}}{12}$。

- （3）$(\sqrt{50} + \sqrt{32}) \div \left(\sqrt{2} - \sqrt{\frac{1}{8}}\right)$
$= (5\sqrt{2} + 4\sqrt{2}) \div \left(\sqrt{2} - \frac{1}{2\sqrt{2}}\right)$
$= 9\sqrt{2} \div \left(\sqrt{2} - \frac{\sqrt{2}}{4}\right) = 9\sqrt{2} \div \frac{3\sqrt{2}}{4} = 9\sqrt{2} \times \frac{4}{3\sqrt{2}} = 12$。

- （4）$(\sqrt{63} - \sqrt{28}) \div \left(\sqrt{\frac{7}{4}} + \sqrt{7}\right)$
$= (3\sqrt{7} - 2\sqrt{7}) \div \left(\frac{\sqrt{7}}{2} + \sqrt{7}\right)$
$= \sqrt{7} \div \frac{3\sqrt{7}}{2} = \sqrt{7} \times \frac{2}{3\sqrt{7}} = \frac{2}{3}$。

**原题**：计算下列各式：
（1）$\frac{4}{\sqrt{5} - 1}$；（2）$\frac{5}{\sqrt{13}} - \frac{6}{\sqrt{13} - 3}$。

**解答**：
- （1）$\frac{4}{\sqrt{5} - 1} = \frac{4(\sqrt{5} + 1)}{(\sqrt{5} - 1)(\sqrt{5} + 1)} = \frac{4(\sqrt{5} + 1)}{5 - 1} = \sqrt{5} + 1$。

- （2）$\frac{5}{\sqrt{13}} - \frac{6}{\sqrt{13} - 3}$
$= \frac{5\sqrt{13}}{13} - \frac{6(\sqrt{13} + 3)}{(\sqrt{13} - 3)(\sqrt{13} + 3)}$
$= \frac{5\sqrt{13}}{13} - \frac{6(\sqrt{13} + 3)}{13 - 9}$
$= \frac{5\sqrt{13}}{13} - \frac{6(\sqrt{13} + 3)}{4}$
$= \frac{5\sqrt{13}}{13} - \frac{3(\sqrt{13} + 3)}{2}$
$= \frac{10\sqrt{13}}{26} - \frac{39(\sqrt{13} + 3)}{26}$
$= \frac{10\sqrt{13} - 39\sqrt{13} - 117}{26} = \frac{-29\sqrt{13} - 117}{26}$。

**原题**：当 $x = \sqrt{5} - 1$ 时，求代数式 $x^2 + 5x - 6$ 的值。

**解答**：
$x^2 + 5x - 6 = (\sqrt{5} - 1)^2 + 5(\sqrt{5} - 1) - 6$
$= (5 - 2\sqrt{5} + 1) + 5\sqrt{5} - 5 - 6$
$= 6 - 2\sqrt{5} + 5\sqrt{5} - 11$
$= 3\sqrt{5} - 5$。

### 复习题C组第8题

```yaml

答案来源: AI参考推导
```

**原题**：已知 $n$ 是正整数，$\sqrt{432n}$ 也是正整数，那么满足条件的 $n$ 的最小值是多少？

**解答**：
将 432 分解质因数：$432 = 2^4 \times 3^3$。

要使 $\sqrt{432n}$ 为正整数，$432n$ 必须是完全平方数，即每个质因数的指数都是偶数。

当前：$2^4$（指数 4 已是偶数），$3^3$（指数 3 是奇数）。

需要再乘一个 3 使 $3^3 \times 3 = 3^4$（指数 4 为偶数）。

所以 $n$ 至少包含因子 3，即 $n$ 的最小值为 $3$。

验证：$432 \times 3 = 1296 = 36^2$，$\sqrt{432 \times 3} = 36$，是正整数。

### 复习题C组第9题

```yaml

source_id: "教材原文_ch15_回顾与反思"

认知层级: 拓展层
答案来源: AI参考推导
```

**原题**：当 $x = \frac{1}{2}(\sqrt{11} - 3)$，$y = \frac{1}{2}(\sqrt{11} + 3)$ 时，求代数式 $\frac{x}{y} + \frac{y}{x}$ 的值。

**解答**：
$\frac{x}{y} + \frac{y}{x} = \frac{x^2 + y^2}{xy} = \frac{(x + y)^2 - 2xy}{xy}$。

先求 $x + y$ 和 $xy$：
- $x + y = \frac{1}{2}(\sqrt{11} - 3) + \frac{1}{2}(\sqrt{11} + 3) = \frac{1}{2} \times 2\sqrt{11} = \sqrt{11}$。
- $xy = \frac{1}{2}(\sqrt{11} - 3) \times \frac{1}{2}(\sqrt{11} + 3) = \frac{1}{4}[(\sqrt{11})^2 - 3^2] = \frac{1}{4}(11 - 9) = \frac{1}{4} \times 2 = \frac{1}{2}$。

所以 $\frac{x}{y} + \frac{y}{x} = \frac{(x + y)^2 - 2xy}{xy} = \frac{(\sqrt{11})^2 - 2 \times \frac{1}{2}}{\frac{1}{2}} = \frac{11 - 1}{\frac{1}{2}} = 10 \times 2 = 20$。
