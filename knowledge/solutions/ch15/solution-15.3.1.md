```yaml
任务类型: 判断题
认知层级: 中间层
答案来源: AI参考推导
```

**原题**：判断下列各式的计算结果是否正确：
（1）$\sqrt{3} + \sqrt{2} = \sqrt{5}$；（2）$\sqrt{5 - 2} = \sqrt{5} - \sqrt{2}$；
（3）$\sqrt{4} + \sqrt{25} = \sqrt{4 + 25}$；（4）$\frac{\sqrt{36} + \sqrt{12}}{2} = 3 + \sqrt{3}$。

**解答**：
- （1）**错误**。$\sqrt{3} + \sqrt{2}$ 不能合并，$\sqrt{3} + \sqrt{2} \neq \sqrt{5}$。$\sqrt{3} + \sqrt{2} \approx 1.732 + 1.414 = 3.146$，而 $\sqrt{5} \approx 2.236$，两者不等。
- （2）**错误**。$\sqrt{5 - 2} = \sqrt{3} \neq \sqrt{5} - \sqrt{2}$（$\approx 2.236 - 1.414 = 0.822$）。
- （3）**错误**。$\sqrt{4} + \sqrt{25} = 2 + 5 = 7$，而 $\sqrt{4 + 25} = \sqrt{29} \neq 7$。
- （4）**正确**。$\frac{\sqrt{36} + \sqrt{12}}{2} = \frac{6 + 2\sqrt{3}}{2} = 3 + \sqrt{3}$。

（3）$\sqrt{5} + \frac{1}{4}\sqrt{80} - \sqrt{245}$；（4）$12\sqrt{\frac{1}{6}} + \sqrt{24} - \sqrt{150}$。

**原题**：计算下列各式：
（1）$\frac{1}{2}(\sqrt{32} + \sqrt{64}) - \frac{1}{5}(\sqrt{50} - \sqrt{75})$；
（2）$\sqrt{0.2} - \frac{1}{2}\sqrt{20} - \left(\sqrt{\frac{5}{4}} - \sqrt{0.45} + \sqrt{90}\right)$。

**解答**：
- （1）
$\frac{1}{2}(\sqrt{32} + \sqrt{64}) - \frac{1}{5}(\sqrt{50} - \sqrt{75})$
$= \frac{1}{2}(4\sqrt{2} + 8) - \frac{1}{5}(5\sqrt{2} - 5\sqrt{3})$
$= 2\sqrt{2} + 4 - \sqrt{2} + \sqrt{3}$
$= \sqrt{2} + \sqrt{3} + 4$。

- （2）
$\sqrt{0.2} - \frac{1}{2}\sqrt{20} - \left(\sqrt{\frac{5}{4}} - \sqrt{0.45} + \sqrt{90}\right)$
$= \sqrt{\frac{1}{5}} - \frac{1}{2} \times 2\sqrt{5} - \left(\frac{\sqrt{5}}{2} - \sqrt{\frac{45}{100}} + 3\sqrt{10}\right)$
$= \frac{\sqrt{5}}{5} - \sqrt{5} - \frac{\sqrt{5}}{2} + \frac{3\sqrt{5}}{10} - 3\sqrt{10}$
$= \frac{2\sqrt{5}}{10} - \frac{10\sqrt{5}}{10} - \frac{5\sqrt{5}}{10} + \frac{3\sqrt{5}}{10} - 3\sqrt{10}$
$= \frac{-10\sqrt{5}}{10} - 3\sqrt{10}$
$= -\sqrt{5} - 3\sqrt{10}$。
