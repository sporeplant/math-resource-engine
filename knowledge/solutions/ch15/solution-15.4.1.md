**原题**：计算 $\sqrt{3} \times (\sqrt{6} + \sqrt{10})$。

**解答**：
$\sqrt{3} \times (\sqrt{6} + \sqrt{10}) = \sqrt{3} \times \sqrt{6} + \sqrt{3} \times \sqrt{10} = \sqrt{18} + \sqrt{30} = 3\sqrt{2} + \sqrt{30}$。

用到了**乘法分配律**。

### 习题B组第4题


**原题**：计算下列各式：
（1）$\frac{4}{\sqrt{2}} - \frac{1}{2\sqrt{2} + 1}$；（2）$\frac{1}{\sqrt{8} - 1} + \frac{1}{\sqrt{12} + 1}$。

**解答**：
- （1）$\frac{4}{\sqrt{2}} - \frac{1}{2\sqrt{2} + 1} = 2\sqrt{2} - \frac{2\sqrt{2} - 1}{(2\sqrt{2} + 1)(2\sqrt{2} - 1)} = 2\sqrt{2} - \frac{2\sqrt{2} - 1}{8 - 1} = 2\sqrt{2} - \frac{2\sqrt{2} - 1}{7}$
$= \frac{14\sqrt{2}}{7} - \frac{2\sqrt{2} - 1}{7} = \frac{12\sqrt{2} + 1}{7}$。

- （2）$\frac{1}{\sqrt{8} - 1} + \frac{1}{\sqrt{12} + 1} = \frac{1}{2\sqrt{2} - 1} + \frac{1}{2\sqrt{3} + 1}$
$= \frac{2\sqrt{2} + 1}{(2\sqrt{2} - 1)(2\sqrt{2} + 1)} + \frac{2\sqrt{3} - 1}{(2\sqrt{3} + 1)(2\sqrt{3} - 1)}$
$= \frac{2\sqrt{2} + 1}{8 - 1} + \frac{2\sqrt{3} - 1}{12 - 1}$
$= \frac{2\sqrt{2} + 1}{7} + \frac{2\sqrt{3} - 1}{11}$
$= \frac{11(2\sqrt{2} + 1) + 7(2\sqrt{3} - 1)}{77}$
$= \frac{22\sqrt{2} + 11 + 14\sqrt{3} - 7}{77}$
$= \frac{22\sqrt{2} + 14\sqrt{3} + 4}{77}$。
