**原题**：小红和小丽分别将9000字和7500字的两篇文稿录入计算机，所用时间相同。已知两人每分钟录入字数的和是220。试列出方程，并求方程的解。

**参考解答**：

设小红平均每分钟录入 $x$ 字，则小丽平均每分钟录入 $(220 - x)$ 字。

根据时间相等，列方程：

$$\frac{9000}{x} = \frac{7500}{220 - x}$$

去分母，得

$$9000(220 - x) = 7500x$$

$$1\,980\,000 - 9000x = 7500x$$

$$1\,980\,000 = 16\,500x$$

$$x = 120$$

经检验，$x = 120$ 是原分式方程的解（$x \neq 0$ 且 $220 - x \neq 0$）。

**原题**：某工程队承建一所实验小学。在施工过程中，由于引进了先进设备，工作效率提高了20%，因此提前1个月完工。那么，这个工程队原计划用几个月的时间建成这所实验小学？

**参考解答**：

设原计划用 $x$ 个月完成。

原计划工作效率为 $\frac{1}{x}$，实际工作效率为 $(1 + 20\%) \times \frac{1}{x} = \frac{1.2}{x}$，实际工作时间为 $(x - 1)$ 个月。

列方程：

$$\frac{1.2}{x} \cdot (x - 1) = 1$$

去分母，得

$$1.2(x - 1) = x$$

$$1.2x - 1.2 = x$$

$$0.2x = 1.2$$

$$x = 6$$

经检验，$x = 6$ 是原分式方程的解，且符合实际意义。

答：这个工程队原计划用6个月的时间建成这所实验小学。

**原题**：请说说用分式方程解决实际问题的一般步骤。它与用一元一次方程以及二元一次方程组解决实际问题的一般步骤有哪些异同？

**参考解答**：

用分式方程解决实际问题的一般步骤：
1. 审题，找出题目中的等量关系；
2. 设未知数；
3. 根据等量关系，列出分式方程；
4. 解分式方程；
5. 检验（既要检验是否为原方程的增根，又要检验是否符合实际意义）；
6. 写出答案。

与一元一次方程、二元一次方程组解决实际问题的异同：
- 相同点：都要经历审题、设未知数、找等量关系、列方程、解方程、检验、作答等步骤。
- 不同点：分式方程解完后必须验根（排除使分母为0的增根），即对解进行双重检验（是否为增根、是否符合实际意义）；整式方程一般只检验是否符合实际意义。

**原题**：在"阳光体育一小时"活动中，小明和小亮参加跳绳比赛。在某段相同时间内，小明跳了180次，小亮跳了210次。已知小明每分钟比小亮少跳20次，则小明和小亮平均每分钟各跳多少次？

**参考解答**：

设小明平均每分钟跳 $x$ 次，则小亮平均每分钟跳 $(x + 20)$ 次。

根据时间相等，列方程：

$$\frac{180}{x} = \frac{210}{x + 20}$$

去分母，得

$$180(x + 20) = 210x$$

$$180x + 3600 = 210x$$

$$3600 = 30x$$

$$x = 120$$

经检验，$x = 120$ 是原分式方程的解（$x \neq 0$ 且 $x + 20 \neq 0$）。

小明每分钟跳120次，小亮每分钟跳 $120 + 20 = 140$ 次。

答：小明平均每分钟跳120次，小亮平均每分钟跳140次。

**原题**：某项工作，甲、乙两人合做3天后，剩下的工作由乙单独来做，用1天即可完成。已知乙单独完成这项工作所需天数是甲单独完成这项工作所需天数的2倍，那么甲、乙单独完成这项工作各需多少天？

**参考解答**：

设甲单独完成需要 $x$ 天，则乙单独完成需要 $2x$ 天。

甲的工作效率为 $\frac{1}{x}$，乙的工作效率为 $\frac{1}{2x}$。

前3天甲乙合做完成的工作量为 $3\left(\frac{1}{x} + \frac{1}{2x}\right) = \frac{9}{2x}$；

最后1天乙单独完成的工作量为 $\frac{1}{2x}$。

列方程：

$$\frac{9}{2x} + \frac{1}{2x} = 1$$

$$\frac{10}{2x} = 1$$

$$\frac{5}{x} = 1$$

$$x = 5$$

经检验，$x = 5$ 是原分式方程的解（$x \neq 0$）。

乙单独完成需要 $2 \times 5 = 10$（天）。

答：甲单独完成需要5天，乙单独完成需要10天。

**原题**：某校八年级(1)班和(2)班的同学，在双休日参加修整花卉的社会实践活动。已知(1)班比(2)班每小时多修整2盆花，(1)班修整66盆花所用的时间与(2)班修整60盆花所用的时间相等，那么(1)班和(2)班的同学平均每小时各修整多少盆花？

**参考解答**：

设(2)班平均每小时修整 $x$ 盆花，则(1)班平均每小时修整 $(x + 2)$ 盆花。

根据时间相等，列方程：

$$\frac{66}{x + 2} = \frac{60}{x}$$

去分母，得

$$66x = 60(x + 2)$$

$$66x = 60x + 120$$

$$6x = 120$$

$$x = 20$$

经检验，$x = 20$ 是原分式方程的解（$x \neq 0$ 且 $x + 2 \neq 0$）。

(1)班每小时修整 $20 + 2 = 22$（盆）。

答：(1)班平均每小时修整22盆花，(2)班平均每小时修整20盆花。

**原题**：一艘轮船在静水中航行的速度是21km/h，顺水航行80km后返回，返回时用同样的时间只航行了60km。求水流的速度。

**参考解答**：

设水流的速度为 $x\ \mathrm{km/h}$。

顺水速度为 $(21 + x)\ \mathrm{km/h}$，顺水航行 $80\ \mathrm{km}$ 所用时间为 $\dfrac{80}{21 + x}\ \mathrm{h}$；

逆水速度为 $(21 - x)\ \mathrm{km/h}$，逆水航行 $60\ \mathrm{km}$ 所用时间为 $\dfrac{60}{21 - x}\ \mathrm{h}$。

根据时间相等，列方程：

$$\frac{80}{21 + x} = \frac{60}{21 - x}$$

去分母，得

$$80(21 - x) = 60(21 + x)$$

$$1680 - 80x = 1260 + 60x$$

$$420 = 140x$$

$$x = 3$$

经检验，$x = 3$ 是原分式方程的解（$21 + x \neq 0$ 且 $21 - x \neq 0$）。

答：水流的速度为 $3\ \mathrm{km/h}$。

**原题**：一项工程原计划由52人在一定时间内完成，但从开工之日起就采用了提高工作效率50%的新技术，这样，改用40人去工作，结果还比原计划提前6天完成。那么，采用新技术完成这项工程用时多少天？

**参考解答**：

设原计划完成这项工程用时 $t$ 天，则采用新技术实际用时 $(t - 6)$ 天。

将每人每天完成的工作量看作1个单位。

原计划总工作量为 $52t$；

采用新技术后，每人工作效率提高50%，每人每天完成 $1.5$ 个单位，实际总工作量为 $40 \times 1.5 \times (t - 6) = 60(t - 6)$。

根据总工作量相等，列方程：

$$52t = 60(t - 6)$$

$$52t = 60t - 360$$

$$8t = 360$$

$$t = 45$$

实际用时 $t - 6 = 45 - 6 = 39$（天）。

经检验，$t = 45$ 是原分式方程的解（方程中不含分母，验根确保实际天数合理），且符合实际意义。

答：采用新技术完成这项工程用时39天。

**原题**：A,B两地之间的路程为200km，甲、乙两辆汽车同时从A地开往B地，两车速度的比是4:5，结果乙车比甲车早30min到达B地。求甲车的速度。

**参考解答**：

设甲车速度为 $4x\ \mathrm{km/h}$，则乙车速度为 $5x\ \mathrm{km/h}$。

甲车所用时间为 $\dfrac{200}{4x} = \dfrac{50}{x}\ \mathrm{h}$；

乙车所用时间为 $\dfrac{200}{5x} = \dfrac{40}{x}\ \mathrm{h}$。

乙车比甲车早 $30\ \mathrm{min}$（$\dfrac{1}{2}\ \mathrm{h}$），列方程：

$$\frac{50}{x} - \frac{40}{x} = \frac{1}{2}$$

$$\frac{10}{x} = \frac{1}{2}$$

$$x = 20$$

经检验，$x = 20$ 是原分式方程的解（$x \neq 0$）。

甲车速度为 $4 \times 20 = 80$（km/h）。

答：甲车的速度为 $80\ \mathrm{km/h}$。

**原题**：今年父亲的年龄是儿子年龄的3倍，5年后父亲和儿子年龄的比是22:9。求父亲和儿子今年的年龄。

**参考解答**：

设儿子今年年龄为 $x$ 岁，则父亲今年年龄为 $3x$ 岁。

5年后父亲年龄为 $(3x + 5)$ 岁，儿子年龄为 $(x + 5)$ 岁。

根据比例关系，列方程：

$$\frac{3x + 5}{x + 5} = \frac{22}{9}$$

去分母，得

$$9(3x + 5) = 22(x + 5)$$

$$27x + 45 = 22x + 110$$

$$5x = 65$$

$$x = 13$$

经检验，$x = 13$ 是原分式方程的解（$x + 5 \neq 0$），且符合实际意义。

父亲今年年龄为 $3 \times 13 = 39$（岁）。

答：父亲今年39岁，儿子今年13岁。
