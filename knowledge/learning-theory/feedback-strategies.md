# 课堂反馈策略

定义 MRE 所有活动设计中使用的快反馈机制。每个活动必须嵌入至少一个反馈节点；反馈必须即时、过程导向、分层差异化。

---

## 核心原则

反馈必须**高频、即时、过程导向**。禁止"只检查最终答案对不对"。

---

## 各层反馈

### 基础层反馈

**核心**：肯定参与和努力。

| 可观察行为 | 反馈回应 |
|---------------------|-------------------|
| 学生尝试观察/辨认/说出 | "很好——你注意到了 X。还能指出 Y 吗？" |
| 学生复述定义时有小错误 | "接近了。我们一起核对一下原文。大声读一遍。" |
| 学生正确模仿了例题 | "步骤都对。现在试一下把这个数字改掉，重新做一遍。" |
| 学生犯了事实性错误 | 不要说"错了"。改为："我们再看看图。你在这里看到了什么？" |
| 学生沉默不语 | 给一个选择："是 A 还是 B？猜一下没关系。" |

**关键规则**：基础层学生答错绝不惩罚。这一层的目标是参与，不是准确率。只要开口了，反馈必须先肯定开口这个行为本身。

### 中间层反馈

**核心**：指出推理的完整性和准确性。

| 可观察行为 | 反馈回应 |
|---------------------|-------------------|
| 学生解释规律正确但不完整 | "你关于 X 的推理是对的。Y 呢？你的解释能覆盖那种情况吗？" |
| 学生从不足的案例中归纳 | "你展示了案例 1 和案例 2 成立。试试案例 3 会怎样？" |
| 学生区分概念但忽略了细微差别 | "A 和 B 在 X 点上确实不同，你说对了。还有别的不同吗？" |
| 学生表达的推理链有缺口 | "第一步到第二步没问题。但第二步到第三步是怎么过去的？" |

**关键规则**：不要替学生补全推理。指出缺口，让他自己填。

### 拓展层反馈

**核心**：推向方法优化、推广和批判。

| 可观察行为 | 反馈回应 |
|---------------------|-------------------|
| 学生正确分析了一种解法 | "分析得不错。有没有更快的办法？你会怎么改？" |
| 学生比较了两种方法 | "你偏向方法 A。在什么条件下方法 B 反而更好？" |
| 学生构造了变式 | "有趣的变式。原证明还成立吗？哪一步会出问题？" |
| 学生推广了结论 | "你推广到了 N 的情形。能证明它对所有 N 都成立吗？" |

**关键规则**：拓展层反馈不应只确认正确性——应推动学生进入下一个抽象层次或批判层次。

## 按活动类型的反馈

### 观察活动
- 反馈节点：学生辨认出特征之后。
- 引导语："你具体注意到了什么？在图上的哪个位置？"
- 避免："对。"（太模糊——什么对了？）
- 更好："对，你注意到了两条对角线在它们的中点相交。"

### 动手操作活动
- ⚠️ **已知障碍**：个体动手操作（折纸、剪切、拼装）在本班会消耗计划时间的 2–3 倍。物料不能保证随时可用。完成率低。
- **推荐默认方式**：仅教师演示——教师在前方操作，全体学生观看。耗时 1/3，产出干净的结果。
- **如果使用个体操作**：限定为零额外物料动作（手指描课本图、直尺和铅笔画图、折单张笔记本纸）。课前自己试做一次，计时的结果乘以 2.5 作为学生所需时间。
- **反馈节点**：操作（或教师演示）之后，数学结论之前。
- **引导语**："我折了之后 / 你画了之后，什么变了？什么没变？"
- **避免**：告诉学生他们应该观察到什么。
- **更好**："描述你现在看到的。和操作之前比有什么不同？"
- **回退方案**：如果个体操作产生混乱或超出计划时间 3 分钟以上，立即叫停所有学生。切换到教师演示："所有人把纸放下，看前面。"

### 归纳活动
- 反馈节点：学生提出推广规律之后。
- 引导语："你的规则对这个新案例也成立吗？"（给出一个边界案例）
- 避免："这个规则是对的。"（过早闭合）
- 更好："用这三个例子检验你的规则。成立吗？"

### 表达活动
- 反馈节点：学生描述数学关系之后。
- 引导语："能说得更精确一点吗？用哪个数学术语更合适？"
- 避免：替学生用完美数学语言重新表述。
- 更好："你说'它们对得上'。什么对得上？长度？角度？还是两者？"

### 探究活动
- 反馈节点：在预设检查点，不要只在结束之后。
- 引导语："到目前为止你弄清楚了什么？卡在什么地方？"
- 避免："解法是这样的。"（截断探究过程）
- 更好："你试了哪些办法？哪个方向看起来最有希望？为什么？"

### 论证活动
- 反馈节点：每一步逻辑之后，不要只在结论处。
- 引导语："你声称 X 能从 Y 推出。哪条定理或定义支持这一步？"
- 避免："证明正确。"（即便正确——追问每一步为什么成立）
- 更好："再走一遍第三步。那个替换为什么有效？"

## 反馈传递规则

### 规则 1：教师巡视优先；同伴反馈为补充

**默认方式**：反馈来自教师巡视，在独立书写期间走动阅读学生笔记本。
- 每次独立书写阶段教师都走动巡视，每分钟读 5–6 本。
- 如果同一错误在 ≥ 3 本笔记本中出现，暂停全班，在黑板匿名纠正。
- 个别错误在学生桌旁轻声处理：一句话指导，然后继续走动。

**同伴反馈选项**（选择性使用，需要准备）：
- ⚠️ **已知障碍**："互换笔记本检查"在本班往往导致茫然对视和社交不适。学生不知道该看什么，或不愿批评同伴。
- **准备要求**：(1) 互换前在黑板上写出 2–3 条检查清单。(2) 用一个示例答案示范检查过程。(3) 保持机械性——检查有无标注步骤、有无理由，不检查推理质量。
- **回退方案**：如果学生困惑，跳过互换。教师读出清单，学生自查自己的笔记本。

### 规则 2：具体优于笼统
- ❌ "很好。"
- ✅ "你正确识别了已知条件给出了相等的边，这让我们可以先证明它是菱形。"

### 规则 3：过程优于结果
- ❌ "答案错了。"
- ✅ "你的前两步没问题。我们重新看看第三步——你在这里用了哪条定理？"

### 规则 4：学生先自纠
检测到错误时：
1. 指出有错误的具体步骤（不指出错误是什么）。
2. 问："你注意到这一步有什么问题吗？"
3. 仅当学生一次引导后仍不能自纠时，给出一个支架提示。
4. 如仍卡住，给出带解释的纠正。

## 反馈节点放置

MRE 输出中的每个活动必须包含 `反馈节点` 字段，明确指出：

1. **在第几步**进行反馈。
2. **收集什么证据**（可观察行为，不是推断）。
3. **反馈如何传递**（教师巡视 / 全班手势信号 / 教师主导问答 / 对照黑板自查 / 同伴互查——仅在黑板上准备了检查清单时使用）。

示例（默认——教师巡视）：
```
反馈节点：第 2 步后（独立书写证明）
→ 教师走动，读 5–6 本笔记本，检查"两步走"结构
→ 如 ≥ 80% 符合，继续。如 < 80%，暂停并在黑板用新例题重新讲解。
```

示例（同伴互查——仅在做好准备时使用）：
```
反馈节点：第 2 步后（独立证明）
→ 黑板检查清单："(1) 两步都标注了？(2) 每一步都有理由？"
→ 学生互换笔记本，只检查这 2 项——最多 1 分钟
→ 教师点名 2 位学生汇报："你的同伴两步都有了吗？"
→ 回退方案：如有困惑，教师读出清单，学生自查
```
