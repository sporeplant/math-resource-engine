# 矩形的性质定理

矩形的四个角都是直角.
矩形的两条对角线相等. 

例 1 如图 21.5-3, 矩形 $ABCD$ 的两条对角线相交于点 $O$ , $\angle AOD = 120^{\circ}$ , $AB = 4 \mathrm{~cm}$ . 求矩形对角线的长. 

解：∵ 四边形 ABCD 是矩形， 
$\therefore A C = B D, A O = C O, B O = D O.$$\therefore A O = C O = B O = D O.$$\because \angle A O D = 1 2 0 ^ {\circ},$$\therefore \angle A O B = 6 0 ^ {\circ}.$
$\therefore \triangle AOB$ 是等边三角形. 
$\therefore A O = B O = A B = 4 \mathrm{cm}.$$\therefore A C = A O + C O = A O + B O = 8 (\mathrm{cm}),$
即矩形 ABCD 对角线的长为 8 cm.
