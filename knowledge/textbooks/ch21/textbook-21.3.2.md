# 平行四边形的判定（第二课时）

# 第二课时开始

小亮和小芳分别按下列方法得到了各自的四边形。

小亮的做法

用 4 根木条搭成如图所示的四边形，其中，\(AB = CD\)，\(AC = BD\)。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/b3ed09d3f503ad0faa4bae2220569a7f187596bb342191146db5985f647e5f15.jpg)

小芳的做法

画两条相交于点 O 的直线，截取 \(OA = OC\)，\(OB = OD\)；连接 \(AB, BC, CD, DA\)，得到四边形 \(ABCD\)。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/5b63cd9396c7566d83b555efb760c1d37197b0a67fea647d24b510acedda7512.jpg)
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/9b724104fff60a52330378be643e176830bd45bb98796bd15d9cbf49d6c66786.jpg)

他们得到的四边形是不是平行四边形？请你进行判断，并尝试说明理由。

我们发现：两组对边分别相等的四边形是平行四边形，两条对角线互相平分的四边形是平行四边形。

现在，我们先来证明两组对边分别相等的四边形是平行四边形。

已知：如图 21.3-5，在四边形 \(ABCD\) 中，\(AB = CD\)，\(AD = CB\)。

求证：四边形 \(ABCD\) 是平行四边形。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/6830cfb564b55c79cd37fb9add47ffde6226eb9d312aba778901f64ca5a02d6d.jpg)
图21.3-5
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/eedfeaf1895e6c2c5098db4a4ec1558bdaaca68cf03a7f66594b3ede5bc8be63.jpg)
图21.3-6

证明：如图21.3-6，连接 \(BD\)。

在 \(\triangle ABD\) 和 \(\triangle CDB\) 中，

∵ \(AB = CD\)，\(AD = CB\)，\(BD = DB\)，

\(\therefore \triangle ABD \cong \triangle CDB\)。

\(\therefore \angle ABD = \angle CDB\)，\(\angle ADB = \angle CBD\)。

∴ \(AB \parallel CD\)，\(AD \parallel CB\)。

∴ 四边形 \(ABCD\) 是平行四边形。

求证：两条对角线互相平分的四边形是平行四边形。

# 平行四边形的判定定理

两组对边分别相等的四边形是平行四边形。

两条对角线互相平分的四边形是平行四边形。

例 3 已知：如图 21.3-7，\(\square ABCD\) 的两条对角线 \(AC, BD\) 相交于点 \(O\)，\(E, F\) 分别为 \(OA, OC\) 的中点。

求证：四边形 \(EBFD\) 是平行四边形。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/b779bc47a8239b4a0b4f9233ff0f478e407702b65389649291210776be76f454.jpg)
图21.3-7

证明：∵ 四边形 \(ABCD\) 是平行四边形，
\(\therefore OA = OC\)，\(OB = OD\)。
∵ \(E, F\) 分别为 \(OA, OC\) 的中点，
\(\therefore OE = OF\)。
∴ 四边形 \(EBFD\) 是平行四边形。

# 大家谈谈

在例 3 的已知条件中，如果 \(E, F\) 不再为 \(OA, OC\) 的中点，请你谈谈：

（1）点 \(E, F\) 分别在 \(OA, OC\) 上，怎样确定点 \(E\) 和点 \(F\) 的位置，可使得四边形 \(EBFD\) 是平行四边形？

（2）点 \(E, F\) 分别在 \(OA, OC\) 的延长线上，怎样确定点 \(E\) 和点 \(F\) 的位置，可使得四边形 \(EBFD\) 是平行四边形？
