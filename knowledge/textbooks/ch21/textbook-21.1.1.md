# 21.1 多边形

四边形、五边形等多边形是比三角形复杂一些的图形。这些图形与三角形相比，有哪些新的性质呢？

# 大家谈谈

如图21.1-1，观察这些图形，谈谈它们在组成方式上有什么共同特征？
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/878510c438f68de99ce5af822dca1a337e979eabcb05b530ed635fa07c390279.jpg)
图21.1-1

这些图形与我们学过的三角形一样，都是在平面上由不在同一条直线上的线段首尾顺次相接所组成的。

形如图 21.1-1 中的图形那样，在平面上，由不在同一条直线上的线段首尾顺次相接（且不能相交）组成的图形，叫作多边形（polygon）。一般把边数为 \(n\) 的多边形叫作 \(n\) 边形（\(n\) 为正整数，且 \(n \geq 3\)）。

如图 21.1-2，组成多边形的各条线段叫作多边形的边，多边形相邻两边组成的角叫作多边形的内角，多边形一边的延长线与相邻的另一边所组成的角叫作多边形的外角，多边形每一个内角的顶点叫作多边形的顶点，连接多边形不相邻两个顶点的线段叫作多边形的对角线。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/e33b966c37fe7a186d59746ca5e99445bd3c9d5712d62b2a5cc0d0a1ee5c221d.jpg)
图21.1-2

一个多边形如果总在它的任意一条边所在直线的同一侧，那么这个多边形叫作凸多边形。目前我们只研究凸多边形。

如图21.1-2所示的五边形，我们记作五边形ABCDE。用类似的方法可以表示其他多边形。

# 四边形的内角和等于 \(360^{\circ}\)。

现在，我们来证明这个结论。

已知：如图21.1-4所示的四边形ABCD。

求证：\(\angle A + \angle B + \angle C + \angle D = 360^{\circ}\)。

证明：如图21.1-5，连接AC。

∵ 在 \(\triangle ABC\) 中，\(\angle BAC + \angle B + \angle BCA = 180^{\circ}\)，

在 \(\triangle ACD\) 中，\(\angle DAC+\angle D+\angle DCA=180^{\circ}\)，
\(\therefore (\angle BAC + \angle DAC) + \angle B + \angle D + (\angle BCA + \angle DCA) = 360^{\circ}\)。
\(\therefore \angle BAD + \angle B + \angle BCD + \angle D = 360^{\circ}\)。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/a38aaa8dfee7a74c729cc7027cbd95cd4e2c5ce49842acc48bf9295770baa04e.jpg)

# 四边形的外角和等于 \(360^{\circ}\)。

例如图21.1-7，已知∠EAD是四边形ABCD的一个外角，且∠EAD=∠C。那么，∠B与∠D互补吗？为什么？

解：\(\angle B\) 与 \(\angle D\) 互补。理由如下：

\[
\begin{array}{l} \because \angle EAD + \angle DAB = 180^{\circ}, \\ \angle EAD = \angle C, \\ \therefore \angle C + \angle DAB = 180^{\circ}. \end{array}
\]

又∵ 四边形 ABCD 的内角和等于 \(360^{\circ}\)，
\(\therefore \angle B + \angle D = 180^{\circ}\)，
即 \(\angle B\) 与 \(\angle D\) 互补。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/b78d88d5ced6e1f8c542871d39584680d402aca3736805834323a8ee1ea28259.jpg)
图21.1-7
