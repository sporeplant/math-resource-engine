# 平行四边形的性质（第二课时）

# 第二课时开始

由前面的探究过程，我们还发现平行四边形的对角线互相平分。

现在，我们来证明这个结论。

已知：如图21.2-7，在\(\square ABCD\)中，对角线AC，BD相交于点O。

求证：\(OA = OC\)，\(OB = OD\)。
![](../images/213dec621ecae1fd7df2cc33fbcae609ee898560c7018931bc3ac88dd82495b4.jpg)
图21.2-7

证明：在 \(\triangle AOB\) 和 \(\triangle COD\) 中，

∵ 四边形 ABCD 是平行四边形，
\(\therefore AB \parallel CD\)，\(AB = CD\)。
\(\therefore \angle BAO = \angle DCO\)。
又∵ \(\angle AOB = \angle COD\)，
\(\therefore \triangle AOB \cong \triangle COD\)。
\(\therefore OA = OC\)，\(OB = OD\)。

例 2 如图 21.2-8，O 为 \(\square ABCD\) 两条对角线的交点，\(AC = 24 \mathrm{mm}\)，\(BD = 38 \mathrm{mm}\)，\(BC = 28 \mathrm{mm}\)。求 \(\triangle OAD\) 的周长。

解：∵ 四边形 ABCD 是平行四边形，

\(\therefore AD = BC\)，\(OA = \frac{1}{2}AC\)，
\(OD = \frac{1}{2}BD\)。
∵ \(BC = 28 \mathrm{mm}\)，\(AC = 24 \mathrm{mm}\)，
\(BD = 38 \mathrm{mm}\)，
\(\therefore AD = 28 \mathrm{mm}\)，\(OA = 12 \mathrm{mm}\)，\(OD = 19 \mathrm{mm}\)。
\(\therefore \triangle OAD\) 的周长 = \(AD + OA + OD = 28 + 12 + 19 = 59(\mathrm{mm})\)。
![](../images/2b1aff91759a8ef7e5af50a5da136772d1dedb2b8dc9b2afedc057b496e748cb.jpg)
图21.2-8

例 3 已知：如图 21.2-9，在 \(\square ABCD\) 中，对角线 \(AC, BD\) 相交于点 \(O\)，直线 \(EF\) 过点 \(O\)，交 \(AD\) 于点 \(E\)，交 \(BC\) 于点 \(F\)。

求证：\(OE = OF\)，\(AE = CF\)，\(DE = BF\)。

证明：∵ 四边形 ABCD 是平行四边形，

\[
\begin{array}{l}
\therefore OA = OC\)，\(AD = BC\)，\(AD \parallel BC\)。 \\
\therefore \angle EAO = \angle FCO\)。
\end{array}
\]

又∵ \(\angle AOE = \angle COF\)，
\(\therefore \triangle AOE \cong \triangle COF\)。
\(\therefore OE = OF\)，\(AE = CF\)。
\(\therefore AD - AE = BC - CF\)。
\(\therefore DE = BF\)。
![](../images/fe8db0b43a9af1f89b3dec91db9cef5d0a16ea919a63440ea53787c60d645d1b.jpg)
图21.2-9
