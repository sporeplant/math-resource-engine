通过探究，可发现：

平行四边形是中心对称图形，它的对称中心是两条对角线的交点。

同时，我们还发现平行四边形的对边相等，对角相等，对角线互相平分。

我们先来证明平行四边形的对边相等，对角相等。

已知：如图21.2-3，四边形ABCD是平行四边形。

求证：（1）\(AD = CB\)，\(AB = CD\)。

（2）\(\angle BAD = \angle DCB\)，\(\angle ABC = \angle CDA\)。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/d588884d6c451c0e76f93fa06eabeab77f97e43ebb3c29805cd1b5cd0a164e45.jpg)
图21.2-3

证明：（1）如图21.2-4，连接 \(BD\)。在 \(\triangle ABD\) 和 \(\triangle CDB\) 中，

∵ 四边形 ABCD 是平行四边形，

∴ \(AD\parallel CB\)，\(AB\parallel CD\)。

∴ \(\angle ADB = \angle CBD\)，\(\angle ABD = \angle CDB\)。

又∵ \(BD = DB\)，

∴ \(\triangle ABD \cong \triangle CDB\)。

∴ \(AD = CB\)，\(AB = CD\)。

（2）∵ \(\triangle ABD \cong \triangle CDB\)，

∴ \(\angle BAD = \angle DCB\)。

∵ \(\angle ABD = \angle CDB\)，\(\angle ADB = \angle CBD\)，

∴ \(\angle ABD + \angle CBD = \angle CDB + \angle ADB\)，

即 \(\angle ABC = \angle CDA\)。

∴ \(\angle BAD = \angle DCB\)，\(\angle ABC = \angle CDA\)。
![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/0d544004c3284613cff826fca25974e6791f46784e6c845858e970ef12600b29.jpg)
图21.2-4

# 平行四边形的性质定理

平行四边形的对边相等。

平行四边形的对角相等。

例 1 如图 21.2-6，在 \(\square ABCD\) 中，\(\angle B + \angle D = 260^{\circ}\)。求 \(\angle A\)，\(\angle C\) 的度数。

解：∵ 四边形 ABCD 是平行四边形，

∴ \(\angle B = \angle D\)，\(\angle A = \angle C\)，\(AD \parallel CB\)。
又∵ \(\angle B + \angle D = 260^{\circ}\)，
∴ \(\angle B = \angle D = \frac{260^{\circ}}{2} = 130^{\circ}\)。
∵ \(AD \parallel CB\)，
∴ \(\angle A = 180^{\circ} - \angle B = 180^{\circ} - 130^{\circ} = 50^{\circ}\)。
∴ \(\angle C = \angle A = 50^{\circ}\)。
