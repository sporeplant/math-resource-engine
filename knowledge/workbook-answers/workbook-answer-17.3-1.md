# 17.3 勾股定理(一)

1. (1)B (2)D (3)A (4)C (5)D
2. (1)25 (2)49 (3)8-2 $\sqrt{6}$ (4)7 (5)5或 $\sqrt{7}$

3. (1) $2 \sqrt{5}$ (2) 16

4. 48000元

5. 解：由题意可知 $\angle ECD + \angle ACB = 90^{\circ},$ $\angle ACB + \angle CAB = 90^{\circ},$ $\therefore \angle ECD = \angle CAB.$ 在△CDE和△ABC中， $\left\{ \begin{array}{l} \angle EDC = \angle CBA, \\ \angle ECD = \angle CAB, \\ EC = CA, \end{array} \right.$ $\therefore \triangle CDE \cong \triangle ABC(AAS).$ $\therefore AB = CD.$ $\therefore AB^{2} + DE^{2} = CD^{2} + DE^{2} = CE^{2} = 3.$ 同理，得 $FG^{2} + LK^{2} = HL^{2} = 1.$ $\therefore S_{1} + S_{2} + S_{3} + S_{4} = HL^{2} + CE^{2} = 1 + 3 = 4.$
