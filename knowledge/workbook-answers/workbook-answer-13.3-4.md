# 13.3 全等三角形的判定(四)

1. (1)C (2)C (3)B (4)C

2. (1)答案不唯一，如 $\angle C = \angle E$ (2) $55^{\circ}$ (3)9 (4)20 (5) $125^{\circ}$ (6) $DE = DF$ ，SAS(答案不唯一）

3. 证明：(1) $\because \angle B + \angle C = 180^{\circ}, \angle B = 90^{\circ}$ , $\therefore \angle C = \angle B = 90^{\circ}$ . 又 $\because \angle BAD = \angle CAD, AD = AD$ , $\therefore \triangle BAD \cong \triangle CAD(AAS)$ . $\therefore DB = DC$ . (2)如图，过点 $D$ 作 $DE \perp AB$ 于点 $E$ , 作 $DF \perp AC$ , 交 $AC$ 的延长线于点 $F$ .

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/0dbc7514d2d2029fe3d16c59bda13bba29273f91d83be822e218143c523fe9b0.jpg)

$\therefore \angle AED = \angle AFD = 90^{\circ}.$ $\because AD$ 平分 $\angle BAC$ $\therefore \angle EAD = \angle FAD,$ 在△AED和△AFD中， $\left\{ \begin{array}{l}\angle AED = \angle AFD,\\ \angle EAD = \angle FAD,\\ AD = AD, \end{array} \right.$ $\therefore \triangle AED\cong \triangle AFD(AAS).$ $\therefore DE = DF.$ $\because \angle ABD + \angle ACD = 180^{\circ},$ $\angle ACD + \angle FCD = 180^{\circ},$ $\therefore \angle ABD = \angle FCD.$

在 $\triangle EBD$ 和 $\triangle FCD$ 中， $\left\{ \begin{array}{l} \angle BED = \angle CFD, \\ \angle EBD = \angle FCD, \\ DE = DF, \end{array} \right.$ $\therefore \triangle EBD \cong \triangle FCD$ (AAS). $\therefore DB = DC.$

4. 证明： $\because \angle BCE + \angle ECF = \angle B'CF + \angle ECF = 90^\circ,$ $\therefore \angle BCE = \angle B'CF.$ 在 $\triangle BCE$ 和 $\triangle B'CF$ 中， $\left\{ \begin{array}{l} \angle B = \angle B', \\ BC = B'C, \\ \angle BCE = \angle B'CF, \end{array} \right.$ $\therefore \triangle BCE \cong \triangle B'CF (\mathrm{ASA})$

5. 证明：∵DF⊥AB，DE⊥AC，
∴∠BFD=∠CED=90°.
在△BDF 和△CDE 中， $\left\{\begin{array}{l}DF=DE,\\\angle BFD=\angle CED,\\BF=CE,\end{array}\right.$ ∴△BDF≌△CDE(SAS).
∴∠B=∠C.

6. 解：(1) $AB = AP, AB \perp AP.$ (2) $BQ = AP, BQ \perp AP.$ 证明：如图，延长 $BQ$ 交 $AP$ 于点 $G$ .

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/43e0c87e8e6f1ea26799e2701c7383e314db4ad5ba63250a0bde461c4548bc3c.jpg)

在 $\triangle BCQ$ 和 $\triangle ACP$ 中， $\left\{\begin{aligned}BC&=AC,\\ \angle BCQ&=\angle ACP,\\ CQ&=CP,\end{aligned}\right.$ $\therefore \triangle BCQ \cong \triangle ACP(SAS).$ $\therefore BQ = AP, \angle CBQ = \angle CAP.$

$\because \angle ACB = 90^{\circ},$ $\therefore \angle CBQ + \angle BQC = 90^{\circ}.$ $\because \angle BQC = \angle AQG,$ $\therefore \angle AQG + \angle GAQ = 90^{\circ}.$ $\therefore \angle AGQ = 180^{\circ} - 90^{\circ} = 90^{\circ}.$ $\therefore BQ\bot AP.$ (3)成立.
证明：在△BCQ和△ACP中， $\left\{ \begin{array}{l}BC = AC,\\ \angle BCQ = \angle ACP,\\ CQ = CP, \end{array} \right.$ $\therefore \triangle BCQ\cong \triangle ACP(SAS).$ $\therefore BQ = AP.$ 如图，延长QB交AP于点N，
