$\because BF \perp AC, F$ 为线段 $AD$ 的中点， $\therefore BF$ 垂直平分 $AD$ . $\therefore AB = BD$ . $\because DE$ 是 $BC$ 的垂直平分线， $\therefore BD = CD$ .

$\therefore AB = CD.$ (2) $FC = 4\mathrm{cm}$

6. 解：(1) ∵ OM，ON 分别是 AB，AC 的垂直平分线，
∴ AM = BM，AN = CN。
∵ △AMN 的周长为 6，
∴ AM + MN + AN = 6。
∴ BM + MN + CN = 6，即 BC = 6。
(2) ∵ AM = BM，AN = CN，
∴ ∠BAM = ∠B = 30°，∠CAN = ∠C = 25°。
∵ ∠B + ∠BAC + ∠C = 180°，且 ∠BAC = ∠BAM + ∠MAN + ∠CAN，
∴ ∠B + ∠BAM + ∠MAN + ∠CAN + ∠C = 180°，即 ∠B + ∠B + ∠MAN + ∠C + ∠C = 180°。
∴ ∠MAN = 180° - 2∠B - 2∠C = 180° - 60° - 50° = 70°。
