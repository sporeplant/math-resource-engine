(1) 证明: $\because AD$ 是 $\triangle ABC$ 的角平分线, $DE$ , $DF$ 分别是 $\triangle ABD$ 和 $\triangle ACD$ 的高, $\therefore DE = DF$ , $\angle EAD = \angle FAD$ . 在 $\triangle AED$ 和 $\triangle AFD$ 中, $\left\{ \begin{array}{l} \angle DEA = \angle DFA, \\ \angle EAD = \angle FAD, \\ AD = AD, \end{array} \right.$

$\therefore \triangle AED \cong \triangle AFD$ (AAS). $\therefore AE = AF.$ $\therefore AD$ 为线段 $EF$ 的垂直平分线，即 $AD$ 垂直平分 $EF$ .
(2) 解： $\because DE = DF,$ $\therefore S_{\triangle ABC} = S_{\triangle ABD} + S_{\triangle ACD} = \frac{1}{2} AB \cdot DE + \frac{1}{2} AC \cdot DF = \frac{1}{2} DE \cdot (AB + AC) = 28.$ 又 $\because AB + AC = 8 + 6 = 14,$ $\therefore DE = 4.$

由(2)的证明可得 $FM = FN$ $\because \angle NBF + \angle BNF + \angle NFB = 180^{\circ},$ $\angle FBM + \angle BMF + \angle MFB = 180^{\circ},$ $\therefore \angle NBF + \angle BNF + \angle NFB + \angle FBM+$ $\angle BMF + \angle MFB = 360^{\circ},$ 即 $\angle ABC + \angle BNF + \angle BMF + \angle MFN =$ $360^{\circ}.$ $\therefore \angle MFN = 360^{\circ} - \angle ABC - \angle BMF-$ $\angle BNF = 120^{\circ}.$ $\because \angle CFA = 180^{\circ} - (\angle FAC + \angle FCA) =$ $180^{\circ} - \frac{1}{2} (\angle BAC + \angle ACB) = 180^{\circ}-$ $\frac{1}{2} (180^{\circ} - \angle ABC) = 180^{\circ} - \frac{1}{2} (180^{\circ}-$ $60^{\circ}) = 120^{\circ},$ $\therefore \angle DFE = \angle CFA = \angle MFN = 120^{\circ}.$ 又 $\because \angle MFN = \angle DFM + \angle DFN,$ $\angle DFE = \angle DFN + \angle NFE,$ $\therefore \angle DFM = \angle NFE.$ 在△DMF和△ENF中，
