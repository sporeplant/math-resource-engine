# 12.5 分式方程的应用(一)

1. (1)B (2)C (3)C (4)B (5)B

2. (1) $\frac{120}{x} + 3 = \frac{180}{x}$ (2) $\frac{1}{5} + \frac{1}{x} = \frac{1}{3}$ (3) 120 (4) 10 (5) $\frac{x + 1 + 8}{x - 1} = \frac{5}{3}$

3. 解：(1)设每个小组有 $x$ 名同学。根据题意，得 $\frac{360}{3x} - \frac{360}{4x} = 3$ 。解得 $x = 10$ 。经检验， $x = 10$ 是原分式方程的解。∴每个小组有10名同学。(2)①设原计划每天改造地下管网的长度为 $x \mathrm{~m}$ ，则实际施工时，每天改造地下管网的长度为 $(1 + 20\%)\mathrm{x} \mathrm{~m}$ 。根据题意，得 $\frac{3600}{x} - \frac{3600}{(1 + 20\%)x} = 10$ 。解得 $x = 60$ 。经检验， $x = 60$ 是原分式方程的解。 $(1 + 20\%)x = 72(\mathrm{~m})$ 。∴实际施工时，每天改造地下管网的长度是 $72 \mathrm{~m}$ 。②设施工20天后每天改造地下管网的长度还要增加 $a \mathrm{~m}$ 。根据题意，得 $(40 - 20)(72 + a) \geqslant 3600 - 72 \times 20$ 。解得 $a \geqslant 36$ 。∴施工20天后每天改造地下管网的长度至少还要增加 $36 \mathrm{~m}$ 。

4. 略
