# 17.3 勾股定理(二)

2. (1)12 (2) $(x - 6.8)^{2} + x^{2} = 10^{2}$ (3)17 (4) $\frac{21}{8}$

3. 解：(1)3.2
(2)∵∠MPN=90°,
∴∠APM+∠BPN=90°.
∵∠APM+∠AMP=90°,
∴∠AMP=∠BPN.
在△AMP和△BPN中, $\left\{\begin{array}{l}\angle AMP=\angle BPN,\\\angle MAP=\angle PBN,\\MP=PN,\end{array}\right.$ ∴△AMP≌△BPN(AAS).
∴AM=PB=2.4m.

$\therefore AP = \sqrt{PM^2 - AM^2} = 0.7(\mathrm{m})$ $\therefore$ 乙房间的宽度 $AB = AP + PB = 0.7+$ $2.4 = 3.1(\mathrm{m})$ (3）① $\angle MPN = 180^{\circ} - \angle APM-$ $\angle BPN = 60^{\circ}$ ②如图，过点 $N$ 作 $MA$ 的垂线，垂足为 $D$ ，连接NM.

设 AB = x，则 DN = AB = x.

∵ $\angle MPN = 60^{\circ}$ ，MP = PN，

∴ $\triangle PNM$ 为等边三角形.

∴ $\angle MNP = 60^{\circ}$ ，MP = MN.

∵ ND // AB，

∴ $\angle DNP = 45^{\circ}$ .

∴ $\angle DNM = 60^{\circ} - 45^{\circ} = 15^{\circ}$ .

∵ $\angle APM = 75^{\circ}$ ,

∴ $\angle AMP = 15^{\circ}$ .

∴ $\angle DNM = \angle AMP$ .

在 $\triangle AMP$ 和 $\triangle DNM$ 中， $\left\{\begin{aligned}\angle AMP &= \angle DNM, \\ \angle MAP &= \angle NDM, \\ MP &= NM,\end{aligned}\right.$ ∴ $\triangle AMP \cong \triangle DNM (AAS)$ .

∴ AM = DN.

∴ AB = DN = AM = 2.8 m，即丙房间的宽度 AB 为 2.8 m.

4. $5 \mathrm{~m}$

5. 解：如图，作 $EJ \perp A'C$ 于点 $J$ ， $FH \perp A'C$ 于点 $H$ ，与卡车车身下底交于点 $G$ ， $FI \perp EJ$ 于点 $I$ .

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/cf1a0e3dc6b06a4ed052ea73d963c26c69e5ee340deac96d3756283c6cc2eae7.jpg)

$\because AC \perp A'C, GB \perp AC,$ $\therefore FH // EJ // AB, FI // BG // A'C.$ $\therefore IJ = FH, GH = BC, \angle EIF = \angle BGF = \angle CHG = \angle A'CB = 90^\circ, \angle FBG = \angle BA'C.$ 设 $A'B = AB = x$ m. $\because A'C^2 + BC^2 = A'B^2$ ，且 $AC = 4$ m, $A'C = 2$ m, $\therefore 2^2 + (4 - x)^2 = x^2.$ 解得 $x = 2.5$ 。 $\therefore A'B = 2.5$ m, $GH = BC = 4 - 2.5 = 1.5$ (m). $\because$ 四边形 $A'DEF$ 是长方形， $ED = 5$ m, $EF = 2.5$ m, $\therefore A'F = ED = 5$ m. $\therefore EF = BF = A'B = 2.5$ m.
在 $\triangle BGF$ 和 $\triangle A'CB$ 中， $\left\{ \begin{array}{l} \angle BGF = \angle A'CB, \\ BF = A'B, \\ \angle FBG = \angle BA'C, \end{array} \right.$ $\therefore \triangle BGF \cong \triangle A'CB (\text{ASA}).$ $\therefore FG = BC = 1.5$ m, $BG = A'C = 2$ m. $\therefore IJ = FH = FG + GH = 1.5 + 1.5 = 3$ (m). $\because \angle BFE = 90^\circ, \angle IFH = 180^\circ - \angle CHG = 90^\circ,$ $\therefore \angle EFI = 90^\circ - \angle BFI = \angle BFG.$ 在 $\triangle EIF$ 和 $\triangle BGF$ 中， $\left\{ \begin{array}{l} \angle EIF = \angle BGF, \\ \angle EFI = \angle BFG, \\ EF = BF, \end{array} \right.$

$\therefore \triangle EIF \cong \triangle BGF(AAS).$ $\therefore EI = BG = 2 \, m.$ $\therefore EJ = EI + IJ = 2 + 3 = 5(m).$ ∴木箱上部顶点 E 到地面 $A^{\prime}C$ 的距离为 5 m.
