## 知识点03 一次函数与不等式

1. 一次函数与不等式：

①若一次函数 $y = k x + b ( k \neq 0 )$ 的图像经过点 $\left( m , ~ n \right)$ ，则不等式 $k x + b > n$ 的解集取点 $\left( m , ~ n \right)$ 上方所在图像所对应的自变量范围；不等式 $k x + b < n$ 的解集取点 $\left( m , ~ n \right)$ 下方所在图像所对应的自变量范围。

②若一次函数 $y = k _ { 1 } x + b _ { 1 } \big ( k _ { 1 } \neq 0 \big )$ 的图像与一次函数 $y = k _ { 2 } x + b _ { 2 } \big ( k _ { 2 } \neq 0 \big )$ 的图像的交点坐标为$( m , ~ n )$ ，则不等式 $k _ { 1 } x + b _ { 1 } > k _ { 2 } x + b _ { 2 }$ 的解集取函数 $y = k _ { 1 } x + b _ { 1 } \big ( k _ { 1 } \neq 0 \big )$ 的图像在 $y = k _ { 2 } x + b _ { 2 } \big ( k _ { 2 } \neq 0 \big )$ 图像上方的部分所对应的自变量的范围；不等式 $k _ { 1 } x + b _ { 1 } < k _ { 2 } x + b _ { 2 }$ 的解集取函数 $y = k _ { 1 } x + b _ { 1 } \big ( k _ { 1 } \neq 0 \big )$ 的图像在 $y = k _ { 2 } x + b _ { 2 } \big ( k _ { 2 } \neq 0 \big )$ 图像下方的部分所对应的自变量的范围。这两部分都是以两个函数的交点为分界点存在。
