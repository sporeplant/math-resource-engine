## 知识点01 平行四边形的判定

## 1. 平行四边形的判定：

如图：判定四边形 ABCD 是平行四边形：

![](../images/1424f5d5cdca4b549e0bc167f95d45e8b2c942c135f3fd90662c3f27acbb32ea.jpg)

①利用边判定：

I：利用一组对边判定：一组对边 平行且相等 的四边形是平行四边形。

符号语言：若 \(A B \bot \stackrel { \parallel } = C D \ntrianglerighteq { \downarrow } A D \mathrel { \mathop { \cong } } \quad B C\)

则四边形 ABCD 是平行四边形

II：利用两组对边判定：两组对边分别 平行 或分别 相等 的四边形是平行四边形。

符号语言：若 \(A B \_{ } \quad / \quad C D, \ A D \_{ } \quad / \quad B C \boxdot { \Sigma } { A B } \ = \quad C D, \ A D \_ = \quad B C\)

则四边形 ABCD 是平行四边形

②利用角判定：

两组对角分别 相等 的四边形是平行四边形。

符号语言：若 \(\angle A B C \_ = \angle A D C, \angle B A D \_ = \angle B C D\)

则四边形 ABCD 是平行四边形

③利用对角线判定：

对角线 相互平分 的四边形是平行四边形。

符号语言：若 \(O A \_ = \_ O C, \ O B \_ = \_ O D\)

则四边形 ABCD 是平行四边形

## 【即学即练1】

1．在四边形 ABCD 中，对角线 AC 与 BD 交于点 O，下列各组条件，其中不能判定四边形 ABCD 是平行四边形的是（ ）

A． \(O A = O C, O B = O D\)  
B． \(O A = O C, A B \parallel C D\)  
C． \(A B = C D, O A = O C\)

D． \(\angle A D B = \angle C B D, \angle B A D = \angle B C D\)

【解答】解： \(A_{\circ} \because O A = O C, O B = O D,\) ，

∴四边形 ABCD 是平行四边形．故能判定这个四边形是平行四边形；

B、 \(\because O A = O C, A B \parallel C D,\) ，

∴四边形 ABCD 是平行四边形．故能判定这个四边形是平行四边形；

C、 \(\scriptstyle A B = C D, O A = O C,\)

∴四边形 ABCD 不是平行四边形．故不能判定这个四边形是平行四边形；

D、 \(\angle A D B = \angle C B D, \angle B A D = \angle B C D,\)

∴四边形 ABCD 是平行四边形，故能判定这个四边形是平行四边形

故选：C

## 【即学即练2】

2．如图，在四边形 ABCD 中，\(AE \bot BD\) 于点 E，\(CF \bot BD\) 于点 F，且 AB＝CD，BF＝DE，求证：四边形 ABCD是平行四边形

![](../images/98d27b922d48345c44c261e700bbd50d5e149572a03896d5817e46a87cc77e1e.jpg)

【解答】证明：∵\(AE \bot BD\) 于 E，\(CF \bot BD\) 于 F，


$\(\therefore \angle A E D = \angle C F B = 90^{\circ}, $\)



$\(\because B F = D E, $\)



$\(\therefore B F - E F = D E - E F, $\)


即 \(B E = D F,\)

在 \(\mathrm { R t } \triangle A D E\) 和 \(\mathrm { R t } \triangle C B F\) 中，


$\(\left\{ \begin{array}{l} A B = C D \\ B E = D F \end{array}, \right. $\)



$\(\therefore \mathrm{Rt} \triangle A D E \cong \mathrm{Rt} \triangle C B F (H L), $\)



$\(\therefore \angle A B D = \angle C B D, $\)



$\(\therefore A B \parallel C D, $\)



$\(\because A B = C D, $\)


∴四边形 ABCD 是平行四边形

## 【即学即练3】

3．如图，四边形 ABCD 对角线交于点 O，且 O 为 AC 中点，AE＝CF，\(DF \parallel BE\)，求证：四边形 ABCD 是平行四边形

![](../images/ac389fcfcade5ece6fd8b07a73976fb7ca6236b29d27a9a439087d0b4985d538.jpg)

【解答】证明： \(\because O\) 为 AC 中点，


$\(\therefore O A = O C, $\)



$\(\because A E = C F, $\)



$\(\therefore O E = O F, $\)



$\(\because D F \parallel B E, $\)



$\(\therefore \angle E = \angle F, $\)


在\(\triangle BOE\) 和\(\triangle DOF\) 中， \(\left\{ \begin{array} { l l } { \angle E = \angle \mathtt { F } } \\ { 0 . \mathtt { E } = 0 . \mathtt { F } } \\ { \angle \mathtt { B O E } = \angle \mathtt { D O F } } \end{array} \right.\)


$\(\therefore \triangle B O E \cong \triangle D O F (A S A), $\)



$\(\therefore O B = O D, $\)


又 \(\because O A = O C .\)

∴四边形 ABCD 是平行四边形

## 知识点02 三角形的中位线

1. 三角形中位线的定义：

连接三角形任意两边的 中点 得到的线段叫做三角形的中位线。

2. 三角形的中位线定理：

三角形的中位线 平行于 第三边，且等于第三边的 一半 。

几何语言：∵点 D、E 分别是 AB、AC 的中点


$\(\therefore D E \parallel B C, D E = \frac {1}{2} B C $\)

## 题型 01 熟悉平行四边形的判定条件

【典例 1】下列条件中，能判定四边形是平行四边形的是（ ）

A．一组对边相等，另一组对边平行  
B．一组对边平行，一组对角互补  
C．一组对角相等，一组邻角互补  
D．一组对角互补，另一组对角相等

【解答】解：A、一组对边相等，另一组对边平行，也有可能是等腰梯形

B、一组对边平行，一组对角互补，也有可能是等腰梯形  
C、一组对角相等，一组邻角互补可得到两组对角分别相等，所以是平行四边形  
D、一组对角互补，另一组对角相等，可能是含两个直角的一般四边形

故选：C

【变式 1】在四边形 ABCD 中，对角线 AC 与 BD 相交于 O 点，给出五组条件：

（1） \(A B = D C, A D \parallel B C ;\) ；  
（2） \(\scriptstyle A B = C D, \ A B \parallel C D ;\)  
（3） \(A B \parallel C D, \ A D \parallel B C ;\)  
（4） \(O A = O C, O B = O D ;\)  
（5） \(A B = C D, \ A D = B C .\)

能判定此四边形是平行四边形的有（ ）组．

A．1

B．2

C．3

D．4

【解答】解：（1）由 \(^{ } A B = D C, ~ A D \parallel B C^{, , }\) 可知，四边形 ABCD 的一组对边平行，另一组对边相等，据此不能判定该四边形是平行四边形，故本选项不符合题意；

（2）由 \({ }^{ } A B = C D, ~ A B \parallel C D^{\prime \prime}\) 可知，四边形 ABCD 的一组对边平行且相等，据此能判定该四边形是平行四边形，故本选项符合题意；  
（3）由 \(^{\prime \prime} A B \parallel D C, ~ A D \parallel B C^{\prime \prime}\) 可知，四边形 ABCD 的两组对边互相平行，则该四边形是平行四边形，故本选项符合题意；  
（4）由 \({ }^{ } A O = C O, \ B O = D O^{, }\) 可知，四边形 ABCD 的两条对角线互相平分，则该四边形是平行四边形，故本选项符合题意；  
（5）由 \({ }^{ } { }^{4} B = D C, A D = B C^{, }\) 可知，四边形 ABCD 的两组对边相等，则该四边形是平行四边形，故本选项符合题意；

故选：D．

![](../images/7206c1cd9e8ee968e182cb9b4d8d9be1723b3be2e7679db0048b63a27c5083d9.jpg)

【变式 2】如图，四边形 ABCD 的对角线相交于点 O，下列条件不能判定四边形 ABCD 是平行四边形的是（ ）

![](../images/bed7ed9760afc74e6f53712c7c55f8dbc7273d6059aa1a45c98e2b589f1b6dfe.jpg)

\(O A = \frac { 1 } { 2 } A C, O B = \frac { 1 } { 2 } B D\)

B． \(_{A B} = C D, \ A O = O C\)

C． \(A B \parallel C D, \angle D A C = \angle B C A\)

D． \(A B = C D, B C = A D\)

【解答】解： \(A \ 、 \ \because O A = { \frac { 1 } { 2 } } A C, \ O B = { \frac { 1 } { 2 } } B D\) ，


$\(\therefore O A = O C, \quad O B = O D, $\)


∴四边形 ABCD 是平行四边形，故选项 A 不符合题意；

B、 \(\scriptstyle A B = C D, A O = O C,\) ，当 \(\angle B A C = \angle D C A\) 时，四边形 ABCD 不是平行四边形，故选项 B 符合题意；


$\(C 、 \because \angle D A C = \angle B C A, $\)



$\(\therefore A D \parallel B C, $\)



$\(\because A B \parallel C D, $\)


∴四边形 ABCD 是平行四边形，故选项 C 不符合题意；


$\(D 、 \because A B = C D, B C = A D, $\)


∴四边形 ABCD 是平行四边形，故选项 D 不符合题意；

故选：B．

## 题型 02 添加平行四边形的判定条件

【典例 1】在四边形 ABCD 中，\(AB \parallel DC\)，要使四边形 ABCD 成为平行四边形，还需添加的条件是（ ）

A． \(\angle A + \angle C = 180^{\circ}\)

B． \(\angle B + \angle D = 180^{\circ}\)

C． \(\angle A + \angle D = 180^{\circ}\)

D． \(\angle A + \angle B = 180^{\circ}\)

【解答】解：选项 A，B 中的两对角是对角关系，不能推出 \(A D \parallel B C\) ，

选项 C 只能推出 \(A B \parallel D C,\) ，

选项 D 中两角是同旁内角，


$\(\because \angle A + \angle B = 180^{\circ}, $\)



$\(\therefore A D \parallel B C, $\)


又 \(\because A B \parallel D C\) ，

∴四边形 ABCD 为平行四边形，

故选：D．

【变式 1】如图，在四边形 ABCD 中，\(AB \parallel CD\)，要使四边形 ABCD 是平行四边形，下列可添加的条件不正确的是（ ）

A． \(A D = B C\)

B．AB＝CD

C．\(AD \parallel BC\)

D． \(\angle A = \angle C\)

【解答】解：A、当 \(A B \parallel C D, \ A D = B C\) 时，四边形 ABCD 可能为等腰梯形，所以不能证明四边形 ABCD为平行四边形；

B、 \(A B \parallel C D, \ A B = D C\) ， 组对边分别平行且相等，可证明四边形 ABCD 为平行四边形；

C、 \(A B \parallel C D, \ A D \parallel B C\) ，两组对边分别平行，可证明四边形 ABCD 为平行四边形；

D、 \(\because A B \parallel C D\) ，


$\(\therefore \angle A + \angle D = 180^{\circ}, $\)



$\(\because \angle A = \angle C, $\)



$\(\therefore \angle C + \angle D = 180^{\circ}, $\)



$\(\therefore A D \parallel B C, $\)


∴四边形 ABCD 为平行四边形；

故选：A．

【变式 2】如图，在四边形 ABCD 中，\(AB \parallel CD\)，若添加一个条件，使四边形 ABCD 为平行四边形，则下列正确的是（ ）

A． \(A D = B C\)

B． \(\angle A B D = \angle B D C\)

C．AB＝AD

【解答】解：A、由 \(A B \parallel C D, \ A D = B C,\) ，不能判定四边形 ABCD 为平行四边形，故选项 A 不符合题意；

B、 \(\because A B \parallel C D\) ，


$\(\therefore \angle A B D = \angle B D C, $\)


∴不能判定四边形 ABCD 为平行四边形，故选项 B 不符合题意；

C、由 \(A B \parallel C D, \ A B = A D\) ，不能判定四边形 ABCD 为平行四边形，故选项 C 不符合题意；

D、 \(\because A B \parallel C D\) ，


$\(\therefore \angle A B C + \angle C = 180^{\circ}, $\)



$\(\because \angle A = \angle C, $\)



$\(\therefore \angle A B C + \angle A = 180^{\circ}, $\)



$\(\therefore A D \parallel B C, $\)


又 \(\because A B \parallel C D,\) ，

∴四边形 ABCD 是平行四边形，故选项 D 符合题意；

故选：D．

【变式 3】如图，在四边形 ABCD 中，\(AB \parallel CD\)，添加下列一个条件后，一定能判定四边形 ABCD 是平行四边形的是（ ）

\(\because A B \parallel C D\) ，


$\(\therefore \angle A + \angle D = 180^{\circ}, $\)



$\(\because \angle B = \angle D, $\)



$\(\therefore \angle A + \angle B = 180^{\circ}, $\)



$\(\therefore A D \parallel B C, $\)


∴四边形 ABCD 是平行四边形，

故选：C

## 题型 03 三角形的中位线

【典例 1】如图，DE 是 \(\triangle A B C\) 的中位线，若 \(B C = 8\) ，则 DE 的长是（ ）

![](../images/ebf65b1847863dc141c3718cacf65f7caec1768481c3a8302df5802cb80f83a0.jpg)

A．3

B．4

C．5

D．6

【解答】解： \(\because D E\) 是 \(\triangle A B C\) 的中位线， \(B C = 8\) ，


$\(\therefore \mathrm{DE} = \frac {1}{2} \mathrm{BC} = \frac {1}{2} \times 8 = 4. $\)


故选：B．

【变式 1】如图，DE 垂直平分 \(\triangle A B C\) 的边 AB，交 CB 的延长线于点 D，交 AB 于点 E，F 是 AC 的中点，连接 \(A D, \ E F .\) ．若 \(A D = 5, C D = 9\) ，则 EF 的长为（ ）

![](../images/d7cce649a02610fbfde4a3312a82699528f5c829082a0ac2db4c2ce26303c08d.jpg)

A．3

B．2.5

C．2

D．1.5

【解答】解： \(\because D E\) 垂直平分 \(\triangle A B C\) 的边 \(A B, \ A D = 5\) ，


$\(\therefore A D = D B = 5, \quad A E = E B, $\)


∴点 E 是 AB 的点，

\(\because F\) 是 AC 的中点，

\(\therefore E F\) 是 \(\triangle A B C\) 的中位线，


$\(\therefore \mathrm{EF} = \frac {1}{2} \mathrm{BC}. $\)



$\(\because C D = 9, \quad D B = 5, $\)



$\(\therefore B C = C D - B D = 9 - 5 = 4, $\)



$\(\therefore \mathrm{EF} = \frac {1}{2} \mathrm{BC} = \frac {1}{2} \times 4 = 2. $\)


故选：C

【变式 2】如图，在 \(\mathrm { R t } \triangle A B C\) 中， \(\angle C = 90^{\circ}\) °，AC＝6，BC＝8，点 N 是 BC 边上一点，点 M 为 AB 边上的动点，点 D、E 分别为 CN，MN 的中点，则 DE 的最小值是（ ）

![](../images/63a3473fd82823a65284ddd2977094f34488cfe82e497d7e23e42a07ad44ed2c.jpg)

A．2

B． \(\frac { 12 } { 5 }\)

C．3

D． \(\frac { 24 } { 5 }\)

【解答】解：连接 CM，当 \(CM \bot AB\) 时，CM 的值最小（垂线段最短），此时 DE 有最小值，

![](../images/8d0c9a5acd5743538ec1c06ff3d0fa43c1ba7dc2ce421868e53a4333acc6e70e.jpg)

理由是： \(\because \angle C = 90^{\circ}\) ， \(A C = 6\) ， \(B C = 8\) ，


$\(\therefore A B = \sqrt {A C^{2} + B C^{2}} = \sqrt {6^{2} + 8^{2}} = 10, $\)



$\(\therefore \frac {1}{2} A C \cdot B C = \frac {1}{2} A B \cdot C M, $\)



$\(\therefore \frac {1}{2} \times 6 \times 8 = \frac {1}{2} \times 10 \times C M, $\)



$\(\therefore C M = \frac {24}{5}, $\)


∵点 D、E 分别为 CN，MN 的中点，


$\(\therefore D E = \frac {1}{2} C M = \frac {1}{2} \times \frac {24}{5} = \frac {12}{5}, $\)


即 DE 的最小值是 \(\frac { 12 } { 5 }\)

故选：B．

【变式 3】如图，DE 是\(\triangle ABC\) 的中位线，\(\angle ACB\) 的角平分线交 DE 于点 F，若 \(A C = 6, B C = 14\) ，则 DF 的长为 4 ．

![](../images/384ffff77681bfce1b642f0193272d966559760418ee004c1e0c660dfe50570a.jpg)

【解答】解：∵DE 是\(\triangle ABC\) 的中位线， \(B C = 14, A C = 6\) ，


$\(\therefore D E = \frac {1}{2} B C = \frac {1}{2} \times 14 = 7, A E = C E = \frac {1}{2} A C = \frac {1}{2} \times 6 = 3, D E \parallel B C, $\)



$\(\therefore \angle C F E = \angle B C F, $\)


∵CF 平分 \(\angle A C B\) ，


$\(\therefore \angle B C F = \angle E C F, $\)



$\(\therefore \angle E C F = \angle C F E, $\)



$\(\therefore E F = C E = 3, $\)



$\(\therefore D F = D E - E F = 7 - 3 = 4, $\)


故答案为：4

【变式 4】如图，在四边形 ABCD 中，\(AB \parallel CD\)，E，F 分别是 AC，BD 的中点，已知 AB＝12，CD＝6，则EF＝ 3 ．

![](../images/ae90eef4e1be177c6e54e4fb261736ad38e648e5693d46ce9e9ddbd22ad8abc8.jpg)

【解答】解：连接 CF 并延长交 AB 于 G，


$\(\because A B \parallel C D, $\)



$\(\therefore \angle F D C = \angle F B G, $\)


在\(\triangle FDC\) 和\(\triangle FBG\) 中，


$\(\left\{ \begin{array}{l} \angle F D C = \angle F B G \\ F D = F B \\ \angle D F C = \angle B F G \end{array}, \right. $\)



$\(\therefore \triangle F D C \cong \triangle F B G (A S A) $\)



$\(\therefore B G = D C = 6, \quad C F = F G, $\)



$\(\therefore A G = A B - B G = 12 - 6 = 6, $\)



$\(\because C E = E A, \quad C F = F G, $\)



$\(\therefore E F = \frac {1}{2} A G = 3, $\)


故答案为：3

## 题型04 平行四边形的判定证明

1．下列条件中，能判定四边形是平行四边形的是（ ）

A．对角线互相平分  
B．对角线互相垂直  
C．对角线相等

D．对角线互相垂直且相等

【解答】解：A、对角线互相平分的四边形是平行四边形．正确

B、对角线互相垂直的四边形不一定是平行四边形．错误  
C、对角线相等的四边形不一定是平行四边形．错误  
D、对角线互相垂直且相等的四边形不一定是平行四边形．错误

故选：A．

2．下列说法正确的是（ ）

A．一组对边平行，另一组对边相等的四边形是平行四边形  
B．平行四边形的对角互补  
C．有两组对角相等的四边形是平行四边形  
D．平行四边形的对角线平分每一组对角

【解答】解：A．∵一组对边平行，另一组对边相等的四边形是等腰梯形或平行四边形，

∴A 错误，不符合题意；

B．∵两组对角分别相等的四边形是平行四边形，

∴B 错误，不符合题意；

C．∵两组对角分别相等的四边形是平行四边形，

∴C 正确，符合题意；

D．∵菱形对角线平分每一组对角，平行四边形的对角线不平分每一组对角，

∴D 错误，不符合题意；
