## 知识点01 矩形的定义及其性质

1. 矩形的定义：

有一个角是 直角 的平行四边形是矩形。

2. 矩形的性质：

①矩形是特殊的平行四边形，具有平行四边形的一切性质。

特殊性质：

②边的特殊性：邻边 相互垂直  
③角的特殊性：四个角都是 直角（或 \(90^\circ\)）  
④对角线的特殊性：对角线 相等 。即对角线 相互平分且相等 。

即： \(A C \_ = B D, \ O A \_ = \_ O B \_ = \_ O C \_ = \_ O D \ /\)

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/abf3f04c609cf64fa6609b403e3098b9994ff523a1379871125fdfe520369b39.jpg)

由此可得：\(\triangle OAB\)，\(\triangle OBC\)，\(\triangle OCD\)，\(\triangle OAD\) 均是 等腰三角形

⑤面积：等于任意一组 邻边 的乘积。

⑥对称性：既是中心对称图形，也是轴对称图形。

## 知识点03 矩形的判定

1. 直接判定：

有三个角（四个角）是 直角 的四边形是矩形。

符号语言： \(\because \angle A B C = \angle B C D = \angle C D A = \angle A D C\)

∴四边形 ABCD 是矩形

2. 利用平行四边形判定：

①有一个角是直角的平行四边形是矩形。

符号语言：∵在▱ABCD 中， \(\angle A B C = 90^{\circ}\)

∴四边形 ABCD 是矩形

②对角线相等的平行四边形是矩形。

符号语言：∵在▱ABCD 中，AD=BC

∴四边形 ABCD 是矩形

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/4787fe47f71c7d0f52bcc49d837c1261f8cb76f707b06ea76a034f74dade035d.jpg)

【解答】解：过点 P 作 GH 分别交 AD、BC 于点 G、H，

由矩形性质可知，


$\(S_{\triangle A D C} = S_{\triangle A B C}, \quad S_{\triangle P F C} = S_{\triangle P H C}, \quad S_{\triangle A G P} = S_{\triangle A E P}, $\)



$\(\therefore S_{\triangle A D C} - S_{\triangle P F C} - S_{\triangle A G P} = S_{\triangle A B C} - S_{\triangle P H C} - S_{\triangle A E P}, $\)


即 \(\begin{array} { r } { S _ { \parallel \dot { 2 } \dot { 2 } \bar { t } \xi, G P F D } = S _ { \parallel \dot { 2 } \dot { 2 } \bar { t } \xi, E P H B }, } \end{array}\) ，

\(\therefore { \frac { 1 } { 2 } } S _ { \perp { \parallel } i \neq { \parallel } \mathscr { U } G P F D } = { \frac { 1 } { 2 } } S _ { \perp { \parallel } i \neq { \parallel } \mathscr { U } E P H B }\) ，即 \(S_{\triangle D P F} = S_{\triangle P E B}\)


$\(\because G P = A E = 2, \quad P F = 9, $\)



$\(\therefore S_{\triangle D P F} = \frac {1}{2} \times 2 \times 9 = 9 = S_{\triangle P E B}. $\)


即图中阴影面积为 \(S_{\triangle D P F} + S_{\triangle P E B} = 9 + 9 = 18\)

故答案为：18．

20．在矩形 ABCD 中，AB＝3，BC＝4，E，F 是对角线 AC 上的两个动点，分别从 A，C 同时出发相向而行，速度均为 1cm/s，运动时间为 t 秒，0\(\le\)t\(\le\)5

（2）若 G，H 分别是 AB，DC 中点，求证：四边形 EGFH 是平行四边形．

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/e2b2bcc8253c709919a4011e560b5ef5eab53f2c705d70ee5f0c74d9d4582ef6.jpg)

【解答】（1）解：∵四边形 ABCD 是矩形，


$\(\therefore \angle B = 90^{\circ}, $\)



$\(\therefore A C = \sqrt {A B^{2} + B C^{2}} = \sqrt {3^{2} + 4^{2}} = 5, $\)


由题意得： \(\scriptstyle A E = C F = t\) ，

∴EF 相遇前为： \(E F = A C \ { - } A E \ { - } C F = 5 \ { - } \ 2 t ;\) ；

故答案为：5﹣2t 或 \(2 t - 5 ;\) ；

（2）证明：∵四边形 ABCD 是矩形，


$\(\therefore A B = C D, A B \parallel C D, A D \parallel B C, \angle B = 90^{\circ}, $\)

$\(\therefore A G = B G, \quad C H = D H, $\)



$\(\therefore A G = C H, $\)



$\(\because A E = C F, $\)



$\(\therefore A F = C E, $\)

$\(\left\{ \begin{array}{l} \mathrm{AG=CH} \\ \angle \mathrm{GAF} = \angle \mathrm{HCE}, \\ \mathrm{AF=CE} \end{array} \right. $\)



$\(\therefore \triangle A F G \cong \triangle C E H (S A S), $\)



$\(\therefore G F = H E, $\)
