## 类型一：平行四边形中的折叠问题

1．如图，在平行四边形 \(A B C D\) 中，将 \(\triangle A B C\) 沿着 AC 所在的直线折叠得到 \(\triangle A B^{\prime} C, B^{\prime}\) ′C 交 AD 于点 E，连接 \(B^{\prime} D _ { \mathrm { : } }\) ，若 \(\angle B = 60^{\circ}\) ， \(\angle A C B = 45^{\circ}\) °， \(A C = \sqrt { 6 }\) ，则 \(B^{\prime} \ D\) 的长是（ ）

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/f0a247c5baa5af009bd0ae59ac2a720de4d30ef4a0e64b777df382db6ddc8f36.jpg)

A．1

B． \(\sqrt { 2 }\)

C． \(\sqrt { 3 }\)

D． \(\frac { \sqrt { 6 } } { 2 }\)

【解答】解：∵四边形 ABCD 是平行四边形，


$\(\therefore A D \parallel B C, A B \parallel C D, \angle A D C = 60^{\circ}, $\)



$\(\therefore \angle C A E = \angle A C B = 45^{\circ}, $\)


∵将 \(\triangle A B C\) 沿 AC 翻折至 \(\triangle A B^{\prime} C,\) ，


$\(\therefore \angle A C B^{\prime} = \angle A C B = 45^{\circ}, \angle A B^{\prime} C = \angle B = 60^{\circ}, $\)



$\(\therefore \angle A E C = 180^{\circ} - \angle C A E - \angle A C B^{\prime} = 90^{\circ}, $\)



$\(\therefore A E = C E = \frac {\sqrt {2}}{2} A C = \sqrt {3}, $\)



$\(\because \angle A E C = 90^{\circ}, \angle A B^{\prime} C = 60^{\circ}, \angle A D C = 60^{\circ}, $\)



$\(\therefore \angle B^{\prime} A D = 30^{\circ}, \angle D C E = 30^{\circ}, $\)



$\(\therefore B^{\prime} E = D E = 1, $\)



$\(\therefore B^{\prime} D = \sqrt {B^{\prime} E^{2} + D E^{2}} = \sqrt {2}. $\)


故选：B

2．如图，将▱ ABCD 折叠，使点 D、C 分别落在点 F、E 处（点 F、E 都在 AB 所在的直线上），折痕为 MN，若 \(\angle A M F = 50^{\circ}\) °，则 \(\angle A = 65^{\circ}\)

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/8d308126d18998d59cad779d0575d20a7bcf4cbfecd80f2cd499c7d9d3e1e118.jpg)

【解答】解：∵四边形 ABCD 是平行四边形，


$\(\therefore A B \parallel C D, $\)


根据折叠的性质可得： \(M N \parallel A E, \angle F M N = \angle D M N,\)


$\(\therefore A B \parallel C D \parallel M N, $\)



$\(\therefore \angle D M N = \angle F M N = \angle A, $\)



$\(\because \angle A M F = 50^{\circ}, $\)



$\(\therefore \angle D M F = 180^{\circ} - \angle A M F = 130^{\circ}, $\)



$\(\therefore \angle F M N = \angle D M N = \angle A = 65^{\circ}, $\)


故答案为：65．

3．如图，将▱ ABCD 沿对角线 AC 折叠，使点 B 落在 \(B^{\prime}\) ′处，若 \(\angle 1 = \angle 2 = 44^{\circ}\) °，则\(\angle B\) 为（ ）

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/473766db2ed05f1464dfe3c4311c9fb1285a0fd4f4a8482e3bc7f60d1b3dce7f.jpg)

A． \(66^{\circ}\)

B． \(104^{\circ}\)

C． \(114^{\circ}\)

D． \(124^{\circ}\)

【解答】解：∵四边形 ABCD 是平行四边形，


$\(\therefore A B \parallel C D, $\)



$\(\therefore \angle A C D = \angle B A C, $\)


由折叠的性质得： \(\angle B A C = \angle B^{\prime} A C\) ，


$\(\therefore \angle B A C = \angle A C D = \angle B^{\prime} A C = \frac {1}{2} \angle 1 = 22^{\circ}, $\)



$\(\therefore \angle B = 180^{\circ} - \angle 2 - \angle B A C = 180^{\circ} - 44^{\circ} - 22^{\circ} = 114^{\circ}; $\)


故选：C

4．如图，在▱ ABCD 中，E 为边 CD 上一点，将 \(\triangle A D E\) 沿 AE 折叠至 \(\triangle A D^{\prime}\) ′E 处， \(A D^{\prime}\) ′与 CE 交于点 F．若\(\angle B = 52^{\circ}\) ， \(\angle D A E = 20^{\circ}\) ，则 \(\angle F E D^{\prime}\) 的大小为 \(36^\circ\)

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/74e7f95992dc55e0053723ff0e8f57159326c21151e2ed214845afdc1777be6b.jpg)

【解答】解：∵四边形 ABCD 是平行四边形，


$\(\therefore \angle D = \angle B = 52^{\circ}, $\)


由折叠的性质得： \(\angle D^{\prime} \ = \angle D = 52^{\circ}, \angle E A D^{\prime} \ = \angle D A E = 20^{\circ}\)


$\(\therefore \angle A E F = \angle D + \angle D A E = 52^{\circ} + 20^{\circ} = 72^{\circ}, \angle A E D^{\prime} = 180^{\circ} - \angle E A D^{\prime} - \angle D^{\prime} = 108^{\circ}, $\)



$\(\therefore \angle F E D^{\prime} = 108^{\circ} - 72^{\circ} = 36^{\circ}; $\)


故答案为： \(36^{\circ}\)

5．如图，P 是平行四边形纸片 ABCD 的 BC 边上一点，以过点 P 的直线为折痕折叠纸片，使点 C，D 落在纸片所在平面上 \(C^{\prime}, D^{\prime}\) 处，折痕与 AD 边交于点 M；再以过点 P 的直线为折痕折叠纸片，使点 B 恰好落在 \(C^{\prime} \mathrm { ~ \ } P\) 边上 \(B^{\prime}\) 处，折痕与 AB 边交于点 N．若 \(\angle M P C = 74^{\circ}\) ，则 \(\angle N P B^{\prime} = \underline { { { 16 } } }^{\circ}\)

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/049898301fbd87751edd11863f533e469218cea7d7611f311696252b322b0696.jpg)

【解答】解：

∵点 C，D 落在纸片所在平面 \(\perp C^{\prime}, D^{\prime}\) 处，折痕与 AD 边交于点 M，


$\(\therefore \angle M P C^{\prime} = \angle M P C = 74^{\circ}, $\)

$\(\because \angle B P N = \angle B^{\prime} P N, $\)

## 类型二：矩形中的折叠问题

6．如图，矩形 ABCD 沿对角线 BD 折叠，已知长 \(B C = 8 c m\) ，宽 AB＝6cm，那么折叠后重合部分的面积是（ ）

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/fb25a95c41092452f44682e00e0ed0ae39d76433747b2856575f34eb5934eb2e.jpg)

A． \(48 c m^{2}\)

B． \(24 c m^{2}\)

C． \(18 . 75 c m^{2}\)

D． \(18 c m^{2}\)

【解答】解：∵四边形 ABCD 是矩形，


$\(\therefore A D \parallel C B, $\)



$\(\therefore \angle A D B = \angle D B C, $\)



$\(\because \angle C^{\prime} B D = \angle D B C $\)



$\(\therefore \angle A D B = \angle E B D, $\)



$\(\therefore D E = B E, $\)



$\(\therefore C^{\prime} E = 8 - D E, $\)



$\(\because C^{\prime} D = A B = 6, $\)



$\(\therefore 6^{2} + (8 - D E)^{2} = D E^{2}, $\)



$\(\therefore D E = \frac {25}{4}, $\)



$\(\therefore S_{\triangle B D E} = D E \times C D \div 2 = 18. 75 c m^{2}. $\)


故选：C

7．如图，长方形纸片 ABCD，E 为 CD 边上一点，将纸片沿 BE 折叠，点 C 落在点 C'处，将纸片沿 AE 折叠，点 D 落在点 D'处，且 D'恰好在线段 BE 上．若 \(\angle A E C = \alpha { \mathrm { : } }\) ，则 \(\angle C E B =\) （ ）

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/6e00fb97caba3f083a9d05eb7f91acec2e4a4194a45e47453adffa67c85e487a.jpg)

A． \(60^{\circ} - \frac { 2 } { 3 } a\)

B． \(60^{\circ} - \frac { 1 } { 3 } a\)

C． \(60^{\circ} + \frac { 2 } { 3 } a\)

\(60^{\circ} + \frac { 1 } { 3 } a\)

【解答】解：由折叠的性质得： \(\angle A E D = \angle A E D^{\prime}, \angle C E B = \angle C E B\) ，


$\(\because \angle A E D^{\prime} = 180^{\circ} - \angle C E B - \angle A E D, \angle A E D^{\prime} = \angle A E C^{\prime} + \angle C^{\prime} E B = \alpha + \angle C^{\prime} E B, $\)



$\(\therefore \angle A E D^{\prime} = 180^{\circ} - \angle C E B - \angle A E D^{\prime}, $\)



$\(\therefore 2 \angle A E D^{\prime} = 180^{\circ} - \angle C E B, $\)



$\(\therefore 2 (\alpha + \angle C E B) = 180^{\circ} - \angle C E B, $\)



$\(\therefore 3 \angle C E B = 180^{\circ} - 2 \alpha, $\)



$\(\therefore \angle C E B = 60^{\circ} - \frac {2}{3} \alpha, $\)


故选：A．

## 类型三：菱形中的折叠问题

【解答】解：连接 BD，如图所示：

∵四边形 ABCD 为菱形，


$\(\therefore A B = A D, $\)



$\(\because \angle A = 60^{\circ}, $\)


\(\therefore \triangle A B D\) 为等边三角形， \(\angle A D C = 120^{\circ}, \angle C = 60^{\circ}\)

## 类型四：正方形中的折叠问题

14．如图，在正方形 ABCD 中，点 E，F 分别在边 AB，CD 上， \(\angle E F C = 120^{\circ}\) °，若将四边形 EBCF 沿 EF折叠，点 B 恰好落在 AD 边上，则 \(\angle A E B^{\prime}\) 为（ ）

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/7020a6f6d37324a914e3074ed34bf6fd4be0d3a8b6a7f2cfb7bd31e60f1cab58.jpg)

A． \(70^{\circ}\)

B． \(65^{\circ}\)

C． \(30^{\circ}\)

D． \(60^{\circ}\)

【解答】解：∵四边形 ABCD 是正方形，


$\(\therefore A B \parallel C D, $\)



$\(\therefore \angle B E F + \angle E F C = 180^{\circ}, $\)



$\(\because \angle E F C = 120^{\circ}, $\)



$\(\therefore \angle B E F = 180^{\circ} - \angle E F C = 60^{\circ}, $\)


∵将四边形 EBCF 沿 EF 折叠，点 B 恰好落在 AD 边上，


$\(\therefore \angle B E F = \angle F E B^{\prime} = 60^{\circ}, $\)



$\(\therefore \angle A E B^{\prime} = 180^{\circ} - \angle B E F - \angle F E B^{\prime} = 60^{\circ}, $\)


故选：D．

15．如图，把正方形纸片 ABCD 沿对边中点所在的直线对折后展开，折痕为 MN，再过点 B 折叠纸片，使点 A 落在 MN 上的点 F 处，折痕为 BE．若 FN＝3，则正方形纸片的边长为 \(\underline { { 2 \sqrt { 3 } } } .\)

【解答】解：设正方形纸片的边长为 x，则 \(B F = A B = x, B N = { \frac { 1 } { 2 } } B C = { \frac { 1 } { 2 } } x\) ，

∴Rt\(\triangle BFN\) 中，NF＝ \(\therefore \mathrm { R t } \triangle B F N\) \({ \cal N } F = \sqrt { { \tt B F }^{2} - { \tt B N }^{2} } = \frac { 1 } { 2 } \sqrt { 3 } x = 3\)

17．如图，正方形纸片 ABCD 的边长为 12，E 是边 CD 上一点，连接 AE，折叠该纸片，使点 A 落在 AE 上的 G 点，并使折痕经过点 B，折痕 BF 与 AE 交于点 H，点 F 在 AD 上，若 DE＝5，则 AH 的长为 \(- \frac { 60 } { 13 } -\)

![](https://cdn.jsdelivr.net/gh/sporeplant/math-resource-engine@main/knowledge/images/8730451ce3e57159c99e5952397484f01a59cfd39680eb6381722b59a0bbf3bb.jpg)

【解答】解：∵四边形 ABCD 为正方形，


$\(\therefore A B = A D = 12, \angle B A D = \angle D = 90^{\circ}, $\)


由折叠及轴对称的性质可知， \(\triangle A B F { \cong } \triangle G B F\) ，BF 垂直平分 \(A G\) ，


$\(\therefore B F \perp A E, A H = G H, $\)



$\(\therefore \angle B A H + \angle A B H = 90^{\circ}, $\)


又∵ \(\angle F A H + \angle B A H = 90^{\circ}\) 1


$\(\therefore \angle A B H = \angle F A H, $\)



$\(\therefore \triangle A B F \cong \triangle D A E (A S A), $\)



$\(\therefore A F = D E = 5, $\)


在 \(\mathrm { R t } \triangle A B F\) 中，


$\(B F = \sqrt {A B^{2} + A F^{2}} = \sqrt {12^{2} + 5^{2}} = 13, $\)



$\(\because S_{\triangle A B F} = \frac {1}{2} A B \cdot A F = \frac {1}{2} B F \cdot A H, $\)



$\(\therefore 12 \times 5 = 13 A H, $\)
