## 方法一：连接两点构造三角形的中位线

1．如图，在四边形 ABCD 中，点 E、F 分别是边 AB、AD 的中点，BC＝10，CD＝6，EF＝4， \(\angle A F E = 52^{\circ}\) °，则 \(\angle A D C\) 的度数为（ ）

![](../images/e5cd67cb60cb642f3b5e18276494977b3fd1820d5e4f13594f2cfccd1d75bb30.jpg)

A． \(140^{\circ}\)

B． \(142^{\circ}\)

C． \(150^{\circ}\)

D． \(152^{\circ}\)

【解答】解：如图，连接 BD，

∵点 E、F 分别是边 AB、AD 的中点，

\(\therefore E F\) 是 \(\triangle A B D\) 的中位线，


$\(\therefore B D = 2 E F = 2 \times 4 = 8, E F \parallel B D, $\)



$\(\therefore \angle A D B = \angle A F E, $\)



$\(\because \angle A F E = 52^{\circ}, $\)



$\(\therefore \angle A D B = 52^{\circ}, $\)


在 \(\triangle B D C\) 中， \(B D^{2} + C D^{2} = 8^{2} + 6^{2} = 100\) ， \(B C^{2} = 10^{2} = 100\) ，


$\(\therefore B D^{2} + C D^{2} = B C^{2}, $\)



$\(\therefore \angle B D C = 90^{\circ}, $\)



$\(\therefore \angle A D C = \angle A D B + \angle B D C = 52^{\circ} + 90^{\circ} = 142^{\circ}, $\)


故选：B．

![](../images/aee973b2b61f25ac04abbb15d3ba284b1014b3095528572237b9d2f33df5aa71.jpg)

2．如图，在 \(\mathrm { R t } \triangle A B C\) 中， \(\angle C = 90^{\circ}\) °，AC＝6，BC＝8，点 N 是 BC 边上一点，点 M 为 AB 边上的动点，点 D、E 分别为 CN，MN 的中点，则 DE 的最小值是（ ）

![](../images/d1279904a173d9b27fbca203ca62938b67300c4099a426bf7247381885d51b23.jpg)

A．2

B． \(\frac { 12 } { 5 }\)

C．3

D． \(\frac { 24 } { 5 }\)

【解答】解：连接 CM，当 \(C M \bot A B\) 时，CM 的值最小（垂线段最短），此时 DE 有最小值，

![](../images/fd0cda72800c5738c1bd371f83a012fe21acd4ab488135e11f88f910ff958cd9.jpg)

理由是： \(\because \angle C = 90^{\circ}\) ， \(A C = 6\) ， \(B C = 8\) ，


$\(\therefore A B = \sqrt {A C^{2} + B C^{2}} = \sqrt {6^{2} + 8^{2}} = 10, $\)



$\(\therefore \frac {1}{2} A C \cdot B C = \frac {1}{2} A B \cdot C M, $\)



$\(\therefore \frac {1}{2} \times 6 \times 8 = \frac {1}{2} \times 10 \times C M, $\)



$\(\therefore C M = \frac {24}{5}, $\)


∵点 D、E 分别为 CN，MN 的中点，


$\(\therefore D E = \frac {1}{2} C M = \frac {1}{2} \times \frac {24}{5} = \frac {12}{5}, $\)


即 DE 的最小值是 \(\frac { 12 } { 5 }\)

故选：B．

## 5．【三角形中位线定理】

已知：在 \(\triangle A B C\) 中，点 D，E 分别是边 AB，AC 的中点．直接写出 DE 和 BC 的关系；

【应用】

如图，在四边形 ABCD 中，点 E，F 分别是边 AB，AD 的中点，若 \(B C = 5, \ C D = 3, \ E F = 2, \ Z A F E = 45^{\circ}\) °，求 \(\angle A D C\) 的度数；

【拓展】

如图，在四边形 ABCD 中，AC 与 BD 相交于点 E，点 M，N 分别为 AD，BC 的中点，MN 分别交 AC，BD 于点 \(F, G, E F = E G\)

求证： \(B D = A C\)

![](../images/67e0efcd6891eb27340518fef3e665244f69d7c2941ecc8a85101305410380a3.jpg)

（图①）

![](../images/0cb80043128b7deef2bf5e3a0c0e5176207983e074c32d6f1768e8867687c4dd.jpg)

（图②）

![](../images/93a8eab8ed050c3c385748191b9f346464f49f5762093926014157b9322f1ba3.jpg)

（图③）

【解答】解：【三角形中位线定理】 \(D E \parallel B C\) ， \(D E = \frac { 1 } { 2 } B C ;\)

理由：∵点 D，E 分别是边 AB，AC 的中点，

∴DE 是 \(\triangle A B C\) 的中位线，


$\(\therefore D E \parallel B C, D E = \frac {1}{2} B C; $\)


【应用】连接 BD，如图所示，

![](../images/80126ae4b1e325792dbf55ed25d818ae4d5c37a86c95084cd4c78af497b2a68c.jpg)

（图②）

∵E、F 分别是边 AB、AD 的中点，


$\(\therefore E F \parallel B D, B D = 2 E F = 4, $\)



$\(\therefore \angle A D B = \angle A F E = 45^{\circ}, $\)



$\(\because B C = 5, \quad C D = 3, $\)



$\(\therefore B D^{2} + C D^{2} = 25, \quad B C^{2} = 25, $\)



$\(\therefore B D^{2} + C D^{2} = B C^{2}, $\)



$\(\therefore \angle B D C = 90^{\circ}, $\)



$\(\therefore \angle A D C = \angle A D B + \angle B D C = 135^{\circ}; $\)


【拓展】证明：取 DC 的中点 H，连接 MH、NH

![](../images/af6ab4ea2b94b188ecac2672b7f58f5f8346723dce324f94cc72a1ad836a29c7.jpg)

（图③）

∵M、H 分别是 AD、DC 的中点，

\(\therefore M H\) 是 \(\triangle A D C\) 的中位线，

\(\therefore M H \parallel A C\) 且 \(M H = \frac { 1 } { 2 } A C\) （三角形的中位线平行于第三边并且等于第三边的一半），

同理可得 \(N H \parallel B D\) 且 \(N H = \frac { 1 } { 2 } B D\)


$\(\because E F = E G, $\)



$\(\therefore \angle E F G = \angle E G F, $\)



$\(\because M H \parallel A C, N H \parallel B D, $\)



$\(\therefore \angle E F G = \angle H M N, \angle E G F = \angle H N M, $\)



$\(\therefore \angle H M N = \angle H N M, $\)



$\(\therefore M H = N H, $\)



$\(\therefore A C = B D. $\)


## 方法二：利用角平分线，垂直构造三角形的中位线

6．如图， \(\triangle A B C\) 中，M 是 BC 的中点，AD 平分 \(\angle B A C\) ，\(BD \bot AD\) 于点 D，若 AC＝9，DM＝2，则 AB 等于（ ）

![](../images/22e5033bdb8abeded12167b177d0fa324fbe40a9624d766a9a4f06362e636548.jpg)

A．4

B．5

C．6

D．8

【解答】解：如图，延长 BD 与 AC 相交于点 F，

\(\because M\) 为 \(B C\) 中点，

\(\therefore D M\) 是 \(\triangle B C F\) 的中位线，


$\(\therefore D M = \frac {1}{2} C F = 2. $\)



$\(\therefore C F = 4. $\)


\(\because A D\) 平分 \(\angle B A C\) ， \(B D \bot A D\) ，


$\(\therefore A F = A B, \quad B D = D F, $\)



$\(\because A C = 9, $\)



$\(\therefore C F = A C - A F = A C - A B = 9 - A B = 4, $\)



$\(\therefore A B = 5. $\)


故选：B．

![](../images/37a69979e4165b8a59642656193573b278e6c6b7a34c9e8962fba2ebd2f82d28.jpg)

7．如图，DE 垂直平分\(\triangle ABC\) 的边 AB，交 CB 的延长线于点 D，交 AB 于点 E，F 是 AC 的中点，连接 AD、EF．若 \(A D = 5, C D = 9\) ，则 EF 的长为（ ）

![](../images/9911f6ae9969ca7c865ecc36e5480f8f172a40c396ea2b00e0d3e8c0c1a35fc1.jpg)

A．3

B．2.5

C．2

D．1.5

【解答】解： \(\because D E\) 垂直平分\(\triangle ABC\) 的边 AB， \(A D = 5\) ，


$\(\therefore A D = D B = 5, \quad A E = E B, $\)


∴点 E 是 AB 的点，

\(\because F\) 是 AC 的中点，

\(\therefore E F\) 是 \(\triangle A B C\) 的中位线，


$\(\therefore \mathrm{EF} = \frac {1}{2} \mathrm{BC}. $\)



$\(\because C D = 9, \quad D B = 5, $\)



$\(\therefore B C = C D - B D = 9 - 5 = 4, $\)



$\(\therefore \mathrm{EF} = \frac {1}{2} \mathrm{BC} = \frac {1}{2} \times 4 = 2. $\)


故选：C

14．连接三角形两边中点的线段叫做三角形的中位线

（1）请用文字语言叙述三角形的中位线定理：

三角形的中位线 平行 于第三边，并且 等于第三边的一半 ；

（2）证明：三角形中位线定理

已知：如图，DE 是\(\triangle ABC\) 的中位线．

求证： \(\scriptstyle D E = { \frac { 1 } { 2 } } B C, \ D E \parallel B C\)

证明：

![](../images/bbff042707a2e3efebd922beeeeccc98cedc3565021011987a4938de2073f709.jpg)

![](../images/4ab6e351d32102503e299ded1b8dd3970487c5e202dd1078bf5287ed1bcc0e55.jpg)

【解答】解：（1）定理：三角形的中位线平行于第三边并且等于第三边的一半

（2）已知： \(\triangle A B C\) 中，点 D、E 分别是 AB、AC 的中点，

求证： \(D E = \frac { 1 } { 2 } B C, D E { \parallel } B C\) ，

∵点 E 是 AC 的中点，


$\(\therefore A E = C E, $\)

$\(\left\{ \begin{array}{l} \mathrm{AE} = \mathrm{CE} \\ \angle \mathrm{AED} = \angle \mathrm{CEF}, \\ \mathrm{DE} = \mathrm{EF} \end{array} \right. $\)



$\(\therefore \triangle A D E \cong \triangle C E F (S A S), $\)

$\(\therefore A B \parallel C F, $\)
