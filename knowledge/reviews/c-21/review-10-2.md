## 类型一：正方形中的十字架模型

1．如图，在正方形 ABCD 中，点 E 是 DC 边的中点，AE 的垂直平分线分别交 AD，BC 边于点 F，G，垂足为点 H．若 AB＝4，则 GH 的长为 \(- { \frac { 3 { \sqrt { 5 } } } { 2 } } -\)

【解答】解：过点 B 作 \(BN \parallel GF\) 交 AD 于点 N，如图所示：

∵四边形 ABCD 是正方形，AB＝4，


$\(\therefore A D \parallel B C, A D = A B = C D = 4, \angle B A N = \angle D = 90^{\circ}, $\)


∴四边形 BGFN 是平行四边形，


$\(\therefore B N = G F, $\)



$\(\because A E \perp F G, B N \parallel G F, $\)



$\(\therefore B N \perp A E, $\)



$\(\therefore \angle B N A + \angle E A D = 90^{\circ}, $\)



$\(\because \angle A E D + \angle E A D = 90^{\circ}, $\)



$\(\therefore \angle B N A = \angle A E D, $\)


在\(\triangle AED\) 和\(\triangle BNA\) 中，


$\(\left\{ \begin{array}{l} \angle A E D = \angle B N A \\ \angle D = \angle B A N \\ A D = A B \end{array}, \right. $\)



$\(\therefore \triangle A E D \cong \triangle B N A (A A S), $\)



$\(\therefore A E = B N = F G, $\)


∵点 E 是 DC 边的中点，


$\(\therefore D E = \frac {1}{2} C D = 2, $\)



$\(\therefore A E = \sqrt {A D^{2} + D E^{2}} = \sqrt {4^{2} + 2^{2}} = 2 \sqrt {5}, $\)



$\(\therefore F G = 2 \sqrt {5}, $\)


\(\because H\) 是 AE 的中点，


$\(\therefore A H = \frac {1}{2} A E = \sqrt {5}, $\)



$\(\because \angle A H F = \angle D = 90^{\circ}, \angle F A H = \angle E A D, $\)



$\(\therefore \triangle A F H \sim \triangle A E D, $\)


\(\therefore \frac { F H } { D E } = \frac { A H } { A D }\) 即 \(\frac { F H } { 2 } = \frac { \sqrt { 5 } } { 4 }\) ，


$\(\therefore F H = \frac {\sqrt {5}}{2}, $\)



$\(\therefore G H = F G - F H = 2 \sqrt {5} - \frac {\sqrt {5}}{2} = \frac {3 \sqrt {5}}{2}, $\)


故答案为： \(\frac { 3 { \sqrt { 5 } } } { 2 }\)

## 类型二：正方形中的半角 \(( 45^{\circ}\) ）模型
