## 知识点01 菱形的定义与性质

## 1. 分式方程的概念：

有一组邻边 相等 的平行四边形是菱形。

## 2. 菱形的性质：

![](../images/2ee0d85f4b9f6ee7556b4579dba37598ba0310cee2c2c4db1565e782397195b3.jpg)

①菱形是特殊的平行四边形，具有平行四边形所有的性质。

特殊性：

②边的特殊性：四条边都 相等

即： \(A B \_ = B C \_ = C D \_ = A D\)

③对角线的特殊性：对角线相互 垂直 且 平分 每一组对角。

即： \(A C \bot B D\) ，且 \(\angle D A C \_ = \_ \angle B A C \_ = \_ \angle D C A \_ = \_ \angle B C A,\)


$\(\angle A D B = \angle C D B = \angle A B D = \angle C B D 。 $\)


④面积计算：等于对角线乘积的一半。即S ABCD菱形 \(S _ { \stackrel { \ast } { \ast } \pi \beta \sp 6 \mathrm { 3 } A B C D } = \frac { 1 } { 2 } A C \cdot B D\) 。

⑤对称性：既是中心对称图形，也是轴对称图形。

## 【即学即练2】

2．如图，菱形 ABCD 中，AC 交 BD 于 O，\(DE \bot BC\) 于 E，连接 OE，若 \(\angle A B C = 120^{\circ}\) ，则\(\angle OED\)＝（ ）

![](../images/08fcee4b61c75cb6362ee8800d8640683e12bc151d7aea216283a436dfb7d291.jpg)

A． \(20^{\circ}\)

B． \(25^{\circ}\)

C． \(30^{\circ}\)

D． \(35^{\circ}\)

【解答】解：∵四边形 ABCD 是菱形，AC 交 BD 于 O， \(\angle A B C = 120^{\circ}\) ，

∴点 O 为 BD 的中点， \(\angle C B D = \frac { 1 } { 2 } \angle A B C = 60^{\circ}\)

\(\because D E \bot B C,\) ，

\(\therefore \angle D E B = 90^{\circ}\) ，


$\(\therefore O E = O D = \frac {1}{2} B D, \angle O D E = 180^{\circ} - \angle D B E - \angle B E D = 30^{\circ}, $\)



$\(\therefore \angle O E D = \angle O D E = 30^{\circ}, $\)


故选：C

## 知识点02 菱形的判定

1. 直接判定：

四条边都 相等 的四边形是菱形。

符号语言： \(\therefore A B = B C = C D = A D\)

∴四边形 ABCD 是菱形

2. 平行四边形判定：

①邻边相等的平行四边形是菱形。

符号语言：∵在▱ABCD 中， \(\mathsf { A B } = \mathsf { A D }\)

∴四边形 ABCD 是菱形

②对角线相互垂直的平行四边形是菱形。

符号语言：∵在▱ABCD 中， \(A C \bot B D\)

∴四边形 ABCD 是菱形

## 【即学即练2】

5．要使▱ ABCD 成为菱形，则可添加一个条件是（ ）

A． \(A B = A D\)

B． \(A B \bot A D\)

C． \(A D = B C\)

D． \(A C = B D\)

【解答】解：A、由 \(A B = A D\) ，能判定▱ ABCD 为菱形，故选项 A 符合题意；

C、由 \(A D = B C\) ，不能判定▱ ABCD 为菱形，故选项 C 不符合题意；  
D、由 \(A C = B D\) ，能判定▱ ABCD 为矩形，不能判定▱ ABCD 为菱形，故选项 D 不符合题意；

故选：A

## 题型 01 利用菱形的性质求线段或周长

【典例 1】菱形的对角线长分别为 10cm，8cm，则此菱形的周长为（ ）

A．12cm

B． \(\sqrt { 41 } \mathtt { c } \pi\)

\(4 \sqrt { 41 } c m\)

D．24cm

【解答】解：如图，∵菱形的两条对角线长分别为 10cm，8cm，


$\(\therefore A O = 4 c m, B O = 5 c m, $\)


在 Rt\(\triangle AOB\) 中，根据勾股定理， \(A B = \sqrt { A O^{2} + B O^{2} } = \sqrt { 4^{2} + 5^{2} } = \sqrt { 41 } c m\) 所以，周 \(\Vdash = \sqrt { 41 } \times 4 = 4 \sqrt { 41 } ( c m . )\) ）．

故选：C

【变式 1】若菱形的面积为 216，其中一条对角线的长为 24，则该菱形的周长为（

A．36

B．24

C．48

D．60

【解答】解：如图，菱形 ABCD 的面积为 216，AC 与 BD 交于点 O，AC＝24，


$\(\because A C \perp B D, $\)



$\(\therefore \frac {1}{2} A C \cdot B D = \frac {1}{2} \times 24 B D = 216, $\)



$\(\therefore B D = 24, $\)



$\(\because \angle A O B = 90^{\circ}, O A = O C = \frac {1}{2} A C = 12, O B = O D = \frac {1}{2} B D = 9, $\)



$\(\therefore A B = \sqrt {O A^{2} + O B^{2}} = 15, $\)


∴菱形 ABCD 的周 \(\ ! \mathrm { K } = 15 \times 4 = 60\) ，

$\(\therefore 5 B E = 24, $\)



$\(\therefore B E = \frac {24}{5}, $\)

$\(\therefore B D = 2 O B = 12, $\)



$\(\because S _ {\text {菱形} A B C D} = \frac {1}{2} A C \cdot B D = 54, $\)



$\(\therefore A C = 9, $\)



$\(\because A E \perp B C, $\)

## 题型 02 利用菱形的性质求角度

【典例 1】如图，AC，BD 是菱形 ABCD 的对角线，若 \(\angle 1 = 20^{\circ}\) ，则∠2的度数为 \(70^\circ\)

![](../images/0230da468637443a6ef276281f75a7867e130613f56a92b406807ad8de780f23.jpg)

【解答】解：∵AC，BD 是菱形 ABCD 的对角线，


$\(\therefore \angle D A C = \angle 1 = 20^{\circ}, \angle A D B = \angle 2, $\)



$\(\therefore \angle D A B = 40^{\circ}, $\)



$\(\therefore \angle A D C = 140^{\circ}, $\)



$\(\therefore \angle 2 = 70^{\circ}. $\)


故答案为：\(70^\circ\)

【变式 1】如图，在菱形 ABCD 中，AC 交 BD 于点 O，\(DE \bot BC\) 于点 E，连接 OE，若 \(\angle B C D = 40^{\circ}\) ，则\(\angle OED\) 的度数是 \(20^\circ\)

![](../images/062d06a277898e08b0dbd96d3a3cccf88545d1dd50b0bd5a170bade02e8521bf.jpg)

【解答】解：∵四边形 ABCD 是菱形，


$\(\therefore O B = O D, \quad C D = B C, $\)



$\(\therefore \angle C B D = \angle C D B = \frac {1}{2} (180^{\circ} - \angle B C D) = \frac {1}{2} \times (180^{\circ} - 40^{\circ}) = 70^{\circ}, $\)



$\(\because D E \perp B C, $\)



$\(\therefore \angle B E D = 90^{\circ}, $\)



$\(\therefore O E = \frac {1}{2} B D = O B, $\)



$\(\therefore \angle O E B = \angle O B E = 70^{\circ}, $\)



$\(\therefore \angle O E D = 90^{\circ} - \angle O E B = 90^{\circ} - 70^{\circ} = 20^{\circ}, $\)


故答案为： \(20^{\circ}\)

## 题型03 利用菱形的性质求点的坐标

【典例 1】如图：已知点 A 的坐标为 \(( - 2 { \sqrt { 3 } }, 2 )\) ，菱形 ABCD 的对角线交于坐标原点 O，则 C 点的坐标是（ ）

![](../images/e03d2435639deebe096d78e15e24035bdbcc414689d1c854360482e354b07f7e.jpg)

A． \(( - 2 { \sqrt { 3 } }, - 2 )\)

B． \(( 2 { \sqrt { 3 } }, - 2 )\)

C． \(- 2 \sqrt { 3 } )\)

D．（﹣2，﹣2）

【解答】解：∵四边形 ABCD 为菱形，


$\(\therefore O A = O C, \quad O B = O D, $\)


∵点 O 为坐标原点，

∴点 A 和点 C 关于原点对称，点 B 和点 D 关于原点对称，

∵点 A 的坐标为 \(( - 2 { \sqrt { 3 } }, 2 )\) ，

∴C 点坐标为 \(( 2 { \sqrt { 3 } }, \quad - 2 )\) ），

故选：B．

【变式 1】如图，四边形 OABC 是菱形，AC＝12，OB＝16，则顶点 A 坐标是 （10，0）

![](../images/65899a7272fc9ec0f60200d6a9fb5d240d0f6d0749ad83421c67094278871855.jpg)

【解答】解：∵四边形 OABC 是菱形，对角线 OB、AC 交于点 D，


$\(\therefore A C \perp O B, $\)



$\(\therefore \angle A D O = 90^{\circ}, $\)



$\(\because A C = 12, O B = 16, $\)



$\(\therefore A D = C D = \frac {1}{2} A C = 6, O D = B D = \frac {1}{2} O B = 8, $\)



$\(\therefore O A = \sqrt {A D^{2} + O D^{2}} = \sqrt {6^{2} + 8^{2}} = 10, $\)



$\(\therefore A (10, 0), $\)


故答案为：（10，0）

【变式 2】如图，在平面直角坐标系 xOy 中，菱形 ABCD 的顶点 D 在 x 轴上，边 BC 在 y 轴上，若点 A 的坐标为（4，5），则 C 点的坐标为（ ）

![](../images/e23b6aa303278875c45315067abf64da8a1f9639dc796ef771620d38882a7ab4.jpg)

A．（0，﹣2）

B．（0，﹣3）

C．（0，﹣2.5）

D．（﹣2，0）

【解答】解：∵A（4，5），


$\(\therefore O D = 4, A D = 5, $\)


∵四边形 ABCD 是菱形，
