## 知识点03 正多边形

1. 正多边形的概念： 

每条边都 相等 ，每个内角都 相等 的多边形是正多边形。 

2. 正多边形的每个内角计算： 

因为正多边形的内角和为 $\left( n - 2 \right) \cdot 1 8 0 ^ { \circ }$ ，每个内角都相等且有 个内角，所以正多边形的每个内角度数 为： $- { \frac { ( n - 2 ) \cdot 1 8 0 ^ { \circ } } { n } } -$ 

3. 正多边形的每个外角计算： 

正多边形的外角和是 $3 6 0 ^ { \circ }$ ，每个外角也相等，所以正多边形的每个外角度数为 $\cdot { \frac { 3 6 0 ^ { \circ } } { n } }$ 

4. 正多边形的内角与外交关系： 

$$

$$
