# 巧用拆项法

计算： $\frac{1}{a(a + 1)} +\frac{1}{(a + 1)(a + 2)} +\dots +$ $\frac{1}{(a + 9)(a + 10)}.$

分析：本题中有 10 个分式相加，无法通分，而式子中每个分式的分母都可看作两个连续整数的积（设 a 是整数），由此联想到 $\frac{1}{a(a+1)} = \frac{1}{a} - \frac{1}{a+1}$ ，这样可抵消一些项.

解：原式 $= \frac{1}{a} -\frac{1}{a + 1} +\frac{1}{a + 1} -\frac{1}{a + 2} +\cdots +$ $\frac{1}{a + 9} -\frac{1}{a + 10}$ $= \frac{1}{a} -\frac{1}{a + 10}$ $= \frac{a + 10 - a}{a(a + 10)}$ $= \frac{10}{a(a + 10)}.$
