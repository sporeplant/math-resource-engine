# 几何常见错误

初中几何学生典型误解和错误，来源于已有教学设计输出、学生成绩数据（`students/scores.md`）和教材分析。用于活动设计预判学生回答并设计针对性支架。

> **覆盖范围**：第 13 章（全等三角形）、第 21 章（四边形）。随后续章节教学逐步扩展。

---

## 1. 四边形层级混淆

### 错误 1.1："所有矩形都是正方形" / "正方形不是矩形"

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生认为正方形是与矩形隔离的类别，或认为所有矩形都是正方形。 |
| **底层误解** | 子集/超集关系颠倒或否认。学生按外观（"两条长边两条短边"）而非定义属性（四个直角）来定义"矩形"。 |
| **影响分数段** | 基础层（0–49）。依赖视觉原型而非定义的学生。 |
| **预判错误率** | 初次接触约 30%；教学后持续约 15%。 |
| **响应策略** | 1. 画一个正方形。"每个角都是 90° 吗？是。所以它是矩形吗？是。"2. 画一个瘦长的矩形。"每条边都相等吗？不。所以它是正方形吗？不。"3. 结论："所有正方形都是矩形。不是所有矩形都是正方形。"始终使用"正方形是特殊的矩形"这句话。 |

### 错误 1.2：混淆菱形和平行四边形

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生无法说清区别："它们看起来一样"或"菱形就是斜着的平行四边形"。 |
| **底层误解** | 关注视觉方向而非定义属性（四条边相等）。 |
| **影响分数段** | 基础层到低中间层（0–55）。 |
| **预判错误率** | 约 25%。 |
| **响应策略** | 画一个邻边明显不等长的平行四边形。"每条边都相等吗？不。所以它是菱形吗？不。"然后画一个菱形。"它的不同在哪？"引导到："四条边都相等。" |

### 错误 1.3：把性质当判定（或相反）

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | "菱形的对角线垂直，所以对角线垂直的四边形就是菱形。" |
| **底层误解** | 混淆单向蕴含（菱形 → 对角线垂直）和双向等价。这是"逆命题错误"。 |
| **影响分数段** | 所有层次；拓展层学生在陌生情境中也会犯此错误。 |
| **预判错误率** | 综合应用情境中约 30%。 |
| **响应策略** | 画一个对角线垂直但不是菱形的筝形。"对角线垂直。它是菱形吗？不——边不全相等。"明确教授："性质是从已知图形推出结论；判定是从已知条件推出图形。方向相反。" |

## 2. 证明结构错误

### 错误 2.1：正方形判定中跳过"两步走"

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生从部分条件直接写"∴ ABCD 是正方形"，没有先证明它是矩形（或菱形），再证明它还是菱形（或矩形）。 |
| **底层误解** | 把"正方形"当作一步结论，而非两个类别的交集。 |
| **影响分数段** | 所有层次。预判错误率 40%（来源：21.7-analysis-2.md）。 |
| **预判错误率** | 初次约 40%；持续约 20%。 |
| **响应策略** | 1. 每次正方形证明：教师示范"步骤 1：证明它是矩形。步骤 2：证明它还是菱形。步骤 3：因此它是正方形。"2. 黑板用可视化"两步梯子"图。3. 独立练习后教师巡视检查笔记本。如 ≥ 80% 有两步结构，继续。否则用新例题重新示范。4. 可选同伴互查：仅在黑板有检查清单时使用（"✓ 步骤 1 标注了？✓ 步骤 2 标注了？"），最多 1 分钟，如学生困惑则回退到自查。参见 `knowledge/learning-theory/learning-behaviors.md` § 互评中的准备要求和回退方案。 |

### 错误 2.2：每一步缺少理由

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生写出一串等式或全等关系，但不引用哪条定理或定义支持每一步。 |
| **底层误解** | 认为"这很明显"或老师"自然知道"——不理解证明需要显式理由。 |
| **影响分数段** | 基础到中间层（0–65）。 |
| **预判错误率** | 基础层极高（约 80%）；中间层中等（约 30%）。 |
| **响应策略** | 要求学生在每一步后用括号写出理由，如"AB = CD（已知）"或"△ABC ≅ △DEF（SAS）"。每次板书示范都这样做。 |

### 错误 2.3：证明方向反了

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 题目要"证明 ABCD 是平行四边形"，学生证明"如果 ABCD 是平行四边形，那么……"——证明了性质而非判定条件。 |
| **底层误解** | 不区分"已知图形→性质"和"已知条件→图形"。 |
| **影响分数段** | 中间层最受影响（50–69）。 |
| **预判错误率** | 约 20%。 |
| **响应策略** | 开始任何证明前，要求学生口头说出："我需要证明 ___ 是 ___。为此我需要证明 ___。"这在下笔前强制检查方向。 |

## 3. 全等三角形错误（第 13 章，第 21 章前置知识）

### 错误 3.1：把 SSA 当作全等判定

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生声称两个三角形全等，因为"两条边和一个非夹角相等"。 |
| **底层误解** | 从 SAS 推广而不理解为什么角必须是夹角。SSA 不能保证全等（歧义情形）。 |
| **影响分数段** | 所有层次，特别是图形"看起来像"三角形应该全等时。 |
| **预判错误率** | 混合全等问题中约 25%。 |
| **响应策略** | 画两个明显不同的三角形，它们共享 SSA。"相同的两条边，相同的角——但不同的三角形。所以 SSA 不成立。" |

### 错误 3.2：全等语句中对应关系混乱

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 写 △ABC ≅ △DEF，但随后使用 AB = DE 和 BC = DF（对应关系混淆）。 |
| **底层误解** | 不理解全等语句中顶点顺序锁定了对应关系。 |
| **影响分数段** | 基础到低中间层（0–55）。 |
| **预判错误率** | 约 35%。 |
| **响应策略** | 要求学生在使用前显式写出对应关系："A↔D, B↔E, C↔F"。检查对应边匹配：AB↔DE, BC↔EF, AC↔DF。 |

### 错误 3.3：混淆应使用哪个全等判定

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生看到三对相等元素就随意命名判定（如一个是角却说"SSS"，或角不在两边之间却说"SAS"）。 |
| **底层误解** | 背诵判定名称而不检查给定的是什么类型元素（边 vs 角）及它们的相对位置。 |
| **影响分数段** | 基础层严重（约 60%）；中间层（约 20%）。 |
| **预判错误率** | 依问题复杂度约 30–60%。 |
| **响应策略** | 命名判定之前要求："数一数 S 和 A 的个数。对 SAS，A 在两个 S 之间吗？对 ASA，S 在两个 A 之间吗？" |

## 4. 角度计算错误

### 错误 4.1：多步角度链中的算术错误

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 在"180° − 30° 然后 ÷ 2"类问题中，学生计算出错或搞混哪个角是哪个。 |
| **底层误解** | 心算薄弱，且不标注中间结果加剧了问题。 |
| **影响分数段** | 基础到中间层（0–65）。 |
| **预判错误率** | 3 步以上角度问题中约 20–30%。 |
| **响应策略** | 要求学生在图上写出每个中间角度值再继续。"标 ∠1 = ___，然后 ∠2 = ___。"这外化工作记忆。 |

### 错误 4.2：因为"看起来像"就判定一个角是 90° 或 45°

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 学生断言 ∠ABC = 90°，除了图看起来显示直角外没有任何理由。 |
| **底层误解** | 把图当作精确测量而非示意。从直觉几何到演绎几何的过渡未完成。 |
| **影响分数段** | 基础到中间层，特别是在正方形/矩形问题中许多角确实是 90° 时。 |
| **预判错误率** | 约 20%。 |
| **响应策略** | "图是草图，不是测量。你只能在下述情况下声称是 90°：(a) 已知条件给出，(b) 从定理推出，或 (c) 图形被定义为有直角。" |

## 5. 定义与直觉混淆错误

### 错误 5.1：按原型定义而非按属性定义

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | "梯形有一组对边平行，另外一组斜着往里收"——给概念添加了非定义性特征。 |
| **底层误解** | 学生对图形的心理意象包含了附带特征（方向、比例、"样子"），它们不是数学定义的一部分。 |
| **影响分数段** | 所有层次，特别是概念初次形成时。 |
| **预判错误率** | 新概念约 30–40%。 |
| **响应策略** | 呈现在所有无关属性上变化的多种例子：不同方向（旋转 45°）、不同比例（很高、很宽）、不同页面位置。每例都问："这还是 ___ 吗？为什么？" |

### 错误 5.2：定义中遗漏必要条件

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | "正方形是有四条相等边的四边形"——遗漏了直角条件（这定义的是菱形，不是正方形）。 |
| **底层误解** | 学生抓住最显著的特征，忽略了完整的必要条件集。 |
| **影响分数段** | 基础到中间层。 |
| **预判错误率** | 多条件定义（如正方形）约 25%。 |
| **响应策略** | 呈现满足学生不完整定义但明显不是目标图形的反例。"这是一个菱形。它有四条等边。它是正方形吗？不。缺了什么？" |

## 6. 策略选择错误

### 错误 6.1：无法选择判定路径

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 当多条判定路径可用时（如正方形："先证矩形再证菱形"或"先证菱形再证矩形"），学生无法决定用哪条，要么僵住，要么随机选。 |
| **底层误解** | 不在选择策略前分析已知条件。把判定当作背诵的食谱，而非条件化决策。 |
| **影响分数段** | 中间到拓展层（50–100）。预判错误率 35%。 |
| **预判错误率** | 约 35%。 |
| **响应策略** | 教授两步决策启发法："步骤 1：给了什么条件？（边？角？对角线？）步骤 2：哪条路径匹配我的条件？（边条件多 → 先证菱形。角/对角线条件多 → 先证矩形。）" |

### 错误 6.2：综合问题中的知识混乱

| 属性 | 内容 |
|-----------|--------|
| **错误表现** | 在同一问题中混淆矩形、菱形和正方形的性质和判定——对菱形引用矩形性质，或对尚未证明是平行四边形的图形使用正方形判定。 |
| **底层误解** | 四边形关系的心理地图尚未结构化。性质和判定作为孤立事实存储，而非按图形组织。 |
| **影响分数段** | 所有层次。预判错误率 30%。 |
| **预判错误率** | 约 30%。 |
| **响应策略** | 做任何综合问题前，花 1 分钟让学生画出（或参考）四边形层级关系图。"解题之前，先在地图上找到你的位置。" |

## 按学生分层的错误分布

基于 `students/scores.md` 分布（基础层 30 人 / 中间层 11 人 / 拓展层 6 人）：

| 错误类别 | 基础层 (n=30) | 中间层 (n=11) | 拓展层 (n=6) |
|---------------|:------------:|:-------------------:|:--------------:|
| 层级混淆 | 极高 | 中等 | 低 |
| 证明结构 | 极高 | 中高 | 低 |
| 全等三角形 | 高 | 中等 | 低 |
| 角度计算 | 高 | 中等 | 低 |
| 定义与直觉混淆 | 高 | 中等 | 低 |
| 策略选择 | 不适用（未达此层次） | 高 | 中等 |

**设计规则**：任何几何课时，扫描本文档中与该主题相关的错误条目。在活动设计中为每条高频错误预嵌至少一个反例和一个支架提示。
