# 图片引用协议

本协议是图片路径的主控规则。详细 Skill 规则见 `技能/图片资源技能/技能.md`。

---

## 1. 唯一原则

任意 Markdown 文件引用图片时，只允许引用该文件同级目录下的 `images/` 子目录。

标准写法：

```markdown
<img src="./images/xxx.jpg" width="35%">
```

---

## 2. 目录对应关系

| Markdown 文件位置 | 图片目录 | 引用路径 |
|-------------------|----------|----------|
| `知识库/教材原文/*.md` | `知识库/教材原文/images/` | `./images/文件名` |
| `知识库/练习册题库/*.md` | `知识库/练习册题库/images/` | `./images/文件名` |
| `输出/*.md` | `输出/images/` | `./images/文件名` |

---

## 3. `/courseware` 图片处理

`/courseware` 生成课件时，如需使用教材或习题图片：

1. 从源 Markdown 中读取真实图片文件名。
2. 将图片复制或登记到 `输出/images/`。
3. 在课件中使用 `<img src="./images/文件名" width="35%">`。

课件中不得直接引用知识库目录。

---

## 4. 禁止写法

- `![](path)` 或 `![alt](path)`
- 跨目录引用知识库图片
- 引用旧资源目录图片
- `<img src="C:/...">`、`<img src="D:/...">`
- URL 编码后的中文路径

---

## 5. 图片占位符

无法确认图片文件名时，可在草稿中使用占位符：

```text
[图片待确认：请从源教材或习题 Markdown 的 ./images/ 目录核对]
```

最终输出中不得保留占位符。
