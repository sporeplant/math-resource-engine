# 上下文缓存策略

## 允许缓存的知识

### 静态知识（可长期缓存）
- 数学本质定义（函数本质、几何推理本质、方程本质等）
- 教学策略框架（函数教学策略、几何教学策略、统计教学策略等）
- 课程标准内容
- 教材知识结构
- **缓存时长**：knowledge版本更新前长期有效

### 半静态知识（可短期缓存）
- students（班级概况、认知分层等）
- 常见错误模式
- 思维方法框架
- **缓存时长**：7天，或students更新时失效

### 动态知识（不缓存）
- 当前课时上下文
- Skill调用链状态
- Validator校验结果
- 教师当前反馈
- 会话状态
- **处理方式**：每次调用实时获取

## 缓存时长

| 知识类型 | 默认缓存时长 | 失效条件 |
|---------|-------------|---------|
| 数学本质 | 长期 | knowledge版本更新 |
| 教学策略 | 长期 | 策略库版本更新 |
| 课标内容 | 长期 | 课标更新 |
| students | 7天 | 画像更新/新数据输入 |
| 常见错误 | 30天 | 错误库更新 |

## 失效机制

- knowledge版本更新时，相关缓存自动失效
- students更新时，画像缓存立即失效
- 缓存超过最大时长自动清除
- 手动清除：可通过命令清除指定缓存
- 缓存失效后，下次调用时重新加载最新数据

## 缓存容量

- 单缓存条目不超过100KB
- 总缓存容量不超过10MB
- 超出容量按LRU策略淘汰
- 关键知识（数学本质、课标）不参与淘汰
