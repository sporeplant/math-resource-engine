# Validator注册表

系统中所有Validator的登记信息。

---

| 名称 | 审核对象 | 审核规则 | 失败条件 | 回退目标 |
|------|----------|----------|----------|----------|
| 学习目标validators | 分层学习目标 | 行为动词合法、可评价、可观察、无空泛、体现数学活动、符合认知分层、符合课标 | 任意一项不通过 | 学习目标skills |
| 评价validators | 分层评价任务 | 能检测结果、真实证据、可分层、体现思维、避免刷题、支持形成性 | 任意一项不通过 | 评价设计skills |
| 活动validators | 分层教学活动 | 学生中心、真实活动、认知推进、小台阶、快反馈、无伪探究、无灌输 | 任意一项不通过 | 活动设计skills |
| 教学活动时间validators | 完整课时活动 | 双层时间一致、分项预算、班级修正、历史校准、核心任务40分钟可完成 | 核心任务超时、时间字段缺失或预算不一致 | 活动设计skills |
| 教学设计validators | 完整课时方案 | 课时结构、认知推进、小台阶、嵌入评价、数学活动、分层、快反馈、目标一致性 | 任意一项不通过 | 教学设计skills |
| 教学法validators | 完整课时方案 | 低起点、小台阶、快反馈、高参与、无认知跳跃、无超过3分钟纯教师讲解 | 存在认知跳跃或强制失败项 | 教学设计skills |
| 数学validators | 完整课时方案 | 定义准确、公式规范、推理逻辑完整、符号规范、图像一致 | 存在数学定义错误或逻辑断裂 | 教学设计skills |
| 学情适配validators | 完整课时方案 | 符合班级水平、不超纲、支持中间层、支持拓展层、活动类型匹配班级特点 | 存在严重适配问题 | 教学设计skills |
| 提问质量validators | 课堂提问与问题链 | 材料锚点、学生动作、数学指向、评价证据、后续出口、分级备用提示、低质问题拦截 | 存在看图说话、事实复读、封闭确认、提问不对结论负责或平均分不达标 | 提问质量skills |
| 一致性validators | 目标-评价-活动链条 | 目标↔评价一致、评价↔活动一致、活动支撑目标、无无效活动、无不评目标、符合逆向设计 | 任意一项不通过 | 问题来源阶段 |
| 课件validators | 课件 Markdown outputs | A组结构合规（标题页/目标页/例题四页结构/小结三问三答/作业表/分隔符/页数范围）；B组内容合规（emoji≤1/mermaid节点≤6/无连续纯文字/时限标注/评分标注/提问格式） | A组任一项不通过 或 B组违规≥3项 | 课件设计skills |
| 课堂提问调度稿validators | 课堂提问调度稿文件 | 结构合规、板块A质量、板块B质量、数学准确性（零容忍）、协作一致性、页面长度控制 | A组或D组任意一项不通过 或 B组+C组违规≥4项 | 课堂提问调度稿生成skills |
| 复习讲义validators | 复习讲义 Markdown 文件 | outputs目录、YAML、题号连续、题目信息总览、来源均衡、图片路径、禁用表述 | A组任一项不通过，或题号/图片/均衡检查触发硬错误 | 复习课讲义整理skills |
| 教材问题解答validators | 教材问题参考解答 | 题源真实、问题覆盖完整、教材顺序一致、题型适配、数学准确、认知边界合规 | 结构、覆盖、顺序或数学硬检查任一项不通过 | 教材问题解答skills |
