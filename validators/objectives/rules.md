# 学习目标validators

## 验证项

- [ ] 行为动词是否合法（属于对应层级的允许动词列表）
- [ ] 是否可评价（每条目标能设计出对应的评价任务）
- [ ] 是否可观察（目标描述的行为可在课堂中被看到）
- [ ] 是否存在空泛表述（没有"理解""掌握""了解"等模糊词汇）
- [ ] 是否体现数学活动（目标中包含具体的数学操作或思维活动）
- [ ] 是否符合认知分层（三层目标与 认知分层.md 对齐）
- [ ] 是否符合课标（目标与课程标准要求一致）

---

## 验证流程

1. 逐条检查目标中的行为动词
2. 逐条检查目标的可评价性
3. 逐条检查目标的可观察性
4. 检查是否出现禁止词汇
5. 检查三层目标的认知递进关系
6. 对照课标进行一致性校验

---

## 验证结果

- 通过：所有检查项均满足
- 有条件通过：存在可修正的小问题，附修改建议
- 不通过：存在严重问题，需退回重新生成
